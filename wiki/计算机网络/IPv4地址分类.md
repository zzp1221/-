## 核心定义

IPv4地址是32位的二进制标识符，通常以点分十进制记法（Dotted Decimal Notation）表示，如192.168.1.1。传统的IPv4地址采用分类编制方法，根据最高位的比特模式将地址空间分为A、B、C、D、E五类。A类地址（0.0.0.0-127.255.255.255）用前8位作为网络号，第一位固定为0；B类地址（128.0.0.0-191.255.255.255）用前16位作为网络号，前两位固定为10；C类地址（192.0.0.0-223.255.255.255）用前24位作为网络号，前三位固定为110；D类地址（224.0.0.0-239.255.255.255）为组播地址，前四位固定为1110；E类地址（240.0.0.0-255.255.255.255）保留为实验使用。随着互联网的快速发展，无类编址CIDR逐渐取代了传统的分类编址。

## 关键结论

- A类地址网络数少（126个）但每个网络的主机数多（约1677万），分配给大型网络；B类地址适中（16384个网络，每个65534主机）；C类地址网络数多（约209万），但每个网络仅254个主机，适合小型网络
- 私有IP地址范围（RFC 1918）：10.0.0.0/8（A类1个）、172.16.0.0/12（B类16个）、192.168.0.0/16（C类256个）。私有地址不能在公网上路由，需通过NAT访问互联网
- 特殊IPv4地址：0.0.0.0（本网络本主机，DHCP请求源地址）、255.255.255.255（受限广播，仅在本局域网内广播）、127.0.0.0/8（环回/回环地址，127.0.0.1最常用）、169.254.0.0/16（链路本地地址，DHCP失败时自动配置）
- 网络号全0和全1不使用（在分类编址中），主机号全0表示网络本身（如192.168.1.0），主机号全1表示该网络的直接广播地址（如192.168.1.255）
- 每一个IP地址都由网络号和主机号两部分组成，网络号标识主机所在的物理网络，主机号标识该网络中的具体主机。路由器根据网络号转发数据包

## 易错点

1. **网络号与网络地址的区别**：网络地址=网络号+全0主机号。例如192.168.1.0/24是网络地址，192.168.1是网络号（前24位）。在某些语境下两者混用。

2. **A类地址127.0.0.0/8被回环地址占用**：127.0.0.0/8被保留为回环地址，不用于实际网络分配。因此A类地址可用的网络号为1-126（而不是0-127），共126个A类网络。

3. **全1广播地址的范围**：32位全1的255.255.255.255是"受限广播地址"（Limited Broadcast），路由器不转发，仅在发送方所在的本地网络生效。而"直接广播地址"如192.168.1.255（网络号+全1主机号），路由器可以转发。

4. **私有地址在公网被丢弃**：所有公网路由器都应丢弃源地址或目的地址为私有地址的数据包。这不是协议强制的，而是实际网络管理约定（BCP 38入口过滤）。如果发现网络不通且用的是私有地址，首先确认NAT是否配置正确。

## 例题

**例题1**：某主机的IP地址为172.16.5.100，子网掩码为255.255.240.0。判断该主机的IP地址类别、网络地址和该网络能容纳的最大主机数量。

**解答**：172.16.5.100第一个字节172（128-191范围），属于B类地址。子网掩码255.255.240.0=/20，将地址与掩码按位AND：网络地址=172.16.0.0（172.16: 00010000 010, 240: 11110000, 5: 00000101）。主机位数=32-20=12位，最大主机数=2^12-2=4094（减2是去掉网络地址和广播地址）。

**例题2**：某公司有300台主机，请问从以下三个地址块中哪个最适合分配？给出的地址块为：A. 202.196.15.0/24，B. 202.196.16.0/23，C. 202.196.18.0/25。

**解答思路**：/24提供2^8-2=254个可用地址（不够300）；/23提供2^9-2=510个可用地址（足够，且浪费不太多）；/25仅提供126个。因此B最合适。C类地址/23相当于将两个/24合并为一个网络，510个地址刚好适合300台主机的场景。

## 代码示例

```bash
# 查看本机IP地址
ip addr show      # Linux
ipconfig /all     # Windows

# 计算子网
ipcalc 192.168.1.0/24     # Linux工具，需安装ipcalc
sipcalc 192.168.1.0/24    # 更详细的子网计算工具
```

```python
import ipaddress

# Python ipaddress模块处理IP地址
ip = ipaddress.IPv4Address('192.168.1.100')
network = ipaddress.IPv4Network('192.168.1.0/24')

print(f"IP地址: {ip}")
print(f"网络地址: {network.network_address}")
print(f"广播地址: {network.broadcast_address}")
print(f"子网掩码: {network.netmask}")
print(f"可用主机数: {network.num_addresses - 2}")
print(f"是否为私有地址: {ip.is_private}")
print(f"是否为环回地址: {ip.is_loopback}")

# 列出该网络中所有可用主机地址
hosts = list(network.hosts())
print(f"前5个可用地址: {[str(h) for h in hosts[:5]]}")
```
