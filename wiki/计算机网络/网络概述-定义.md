## 核心定义

计算机网络是指将地理位置不同的、具有独立功能的多个计算机及其外部设备，通过通信线路和网络设备连接起来，在网络操作系统、网络管理软件及网络通信协议的管理和协调下，实现资源共享和信息传递的系统。简单来说，计算机网络是"自治计算机的互连集合"。其中，"自治"意味着每台计算机具有独立的处理能力，不依赖于网络中的其他计算机；"互连"意味着计算机之间能够通过通信链路交换信息。一个完整的计算机网络由三个核心部分组成：（1）至少两台具有独立功能的计算机系统；（2）通信设备和通信线路构成的通信子网；（3）网络软件，包括通信协议、网络操作系统和网络应用软件。计算机网络的本质目标是实现资源共享，包括硬件资源（打印机、存储设备）、软件资源（数据库、应用软件）和数据资源（文件、信息）的共享。

## 关键结论

- 计算机网络的核心定义是"自治计算机的互连集合"，强调每台计算机的独立性和相互通信能力
- 计算机网络由通信子网（负责数据传输）和资源子网（负责数据处理）两大部分组成，这种二级结构是现代网络架构的基础
- 资源共享是计算机网络的最根本目标，包括硬件共享、软件共享和数据共享三个层面
- 网络协议是计算机网络正常运行的核心保障，它定义了通信双方必须遵循的规则和约定
- 网络的边缘部分（主机）和核心部分（路由器等交换节点）承担不同的功能角色，边缘负责产生和处理数据，核心负责数据的中继转发

## 易错点

1. **网络与互联网概念混淆**：计算机网络（network）指通过通信链路连接起来的计算机集合；互联网（internet）是网络的网络，即多个计算机网络通过路由器互联而成的更大规模的网络。将局域网等同于整个计算机网络系统是不准确的。

2. **通信子网与资源子网的边界**：通信子网由路由器和通信链路组成，负责数据传输；资源子网由主机和服务器组成，负责数据处理。初学者常把交换机误划入资源子网，实际上交换机属于通信子网。

3. **"自治"的误解**：自治不等于完全不依赖网络，而是指在脱离网络时计算机仍然可以独立运行其操作系统和本地程序。分布式系统中的从属节点不属于严格意义上的"自治计算机"。

4. **将计算机网络等同于分布式系统**：在分布式系统中，多台计算机的存在对用户是透明的，系统像一台虚拟的单机；而在计算机网络中，用户需要显式指定访问哪台计算机。两者的关键区别在于透明性。

## 例题

**例题1**：以下关于计算机网络定义的描述中，正确的是（ ）。
A. 计算机网络就是互联网
B. 计算机网络必须包含至少一台服务器
C. 计算机网络是自治计算机的互连集合
D. 计算机网络中所有计算机必须使用相同的操作系统

**解答**：选C。A错误，互联网是网络的网络，范围更大；B错误，计算机网络中的计算机可以是对等关系，不一定需要专门的服务器；D错误，网络中的计算机可以使用不同的操作系统，只要遵循相同的网络协议即可通信。

**例题2**：简述资源子网和通信子网的功能分工及其各自包含的主要设备。

**解答思路**：通信子网由通信控制处理机（如路由器、交换机）、通信线路和相关软件组成，负责完成网络数据传输、交换和通信控制任务。资源子网由主机系统、终端、终端控制器、连网外设和各种软件资源组成，负责全网的数据处理，向网络用户提供各种网络资源和网络服务。两者通过标准接口互连，实现了数据处理与数据通信的功能分离。

## 代码示例

```bash
# 使用ping命令测试网络连通性
# ping是最基本的网络诊断工具，利用ICMP协议检测主机可达性
ping -c 4 www.example.com

# 使用ifconfig/ip命令查看网络接口配置
ip addr show
# 或
ifconfig

# 使用netstat查看网络连接状态
netstat -tuln
```
