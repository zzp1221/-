## 核心定义

光线追踪（Ray Tracing）是一种基于几何光学的渲染方法，通过模拟光线的传播路径来计算图像。Whitted 风格光线追踪（Whitted-style Ray Tracing）是最经典的算法：从相机出发，通过每个像素发射一条光线（光线投射，Ray Casting），找到光线与场景最近的交点，然后在交点处计算光照。对于镜面反射和折射表面，递归地发射反射光线和折射光线，直到达到最大递归深度或光线未命中任何物体。光线的参数方程为 $\mathbf{r}(t) = \mathbf{o} + t\mathbf{d}$，其中 $\mathbf{o}$ 是原点，$\mathbf{d}$ 是方向，$t > 0$。

光线-物体求交（Ray-Object Intersection）是光线追踪的核心。光线-球体求交：将光线方程代入球体方程 $|\mathbf{r}(t) - \mathbf{c}|^2 = r^2$，得到关于 $t$ 的二次方程。光线-三角形求交：先求光线与三角形所在平面的交点，再判断交点是否在三角形内部（用重心坐标或边界函数）。光线-AABB 求交：利用 slab 方法，分别计算光线与轴对齐包围盒六个面的交点区间，取交集。Whitted 光线追踪可以自然地处理镜面反射、折射、阴影（从交点向光源发射阴影光线判断遮挡）等效果，但无法正确处理漫反射间的全局光照（需要路径追踪）。

## 关键结论

- 光线追踪的时间复杂度为 $O(N \times M)$，其中 $N$ 是像素数，$M$ 是物体数。使用加速结构可以将每个像素的求交复杂度降低到 $O(\log M)$
- 光线-三角形求交的 Möller-Trumbore 算法不需要预先计算平面方程，直接利用重心坐标求解，是实践中最常用的方法
- 折射光线的方向由 Snell 定律决定：$\eta_1 \sin\theta_1 = \eta_2 \sin\theta_2$，其中 $\eta$ 是折射率。当 $\sin\theta_2 > 1$ 时发生全内反射
- 阴影光线（Shadow Ray）从交点向光源发射，如果被遮挡则该点在阴影中。这是硬阴影的实现方式
- Whitted 光线追踪假设反射和折射是完美的镜面行为，无法模拟光泽反射（glossy reflection）和漫反射间的间接光照

## 易错点

1. **自相交问题**：光线从物体表面出发时，由于浮点精度误差，可能与同一物体再次相交。解决方法是给 $t$ 设置一个小的偏移量 $t_{min}$（如 $10^{-4}$），忽略太近的交点
2. **递归深度控制**：反射/折射光线需要递归追踪，但无限递归会导致栈溢出。通常设置最大递归深度（如 5-10 层），并根据反射率/折射率进行俄罗斯轮盘赌（Russian Roulette）终止
3. **折射率方向错误**：折射光线的方向需要根据光线是从外部进入物体还是从物体内部射出决定折射率的比值。进入物体时 $\eta = \eta_{物体} / \eta_{空气}}$，射出时反过来

## 例题

**题目**：一条光线从原点 $\mathbf{o} = (0, 0, 0)$ 沿方向 $\mathbf{d} = (0, 0, 1)$ 发射，一个球体的中心为 $\mathbf{c} = (0, 0, 5)$，半径 $r = 1$。求光线与球体的交点。

**解答**：

第一步，将光线方程代入球体方程：
$$|\mathbf{o} + t\mathbf{d} - \mathbf{c}|^2 = r^2$$
$$|(0,0,t) - (0,0,5)|^2 = 1$$
$$|0, 0, t-5|^2 = 1$$
$$(t-5)^2 = 1$$

第二步，解方程：
$$t - 5 = \pm 1$$
$$t_1 = 4, \quad t_2 = 6$$

第三步，取最近的正交点 $t = 4$：
$$\mathbf{p} = (0, 0, 0) + 4 \cdot (0, 0, 1) = (0, 0, 4)$$

交点为 $(0, 0, 4)$，距离原点 4 个单位。另一个交点 $(0, 0, 6)$ 是光线穿出球体的点。
