## 核心定义

顶点着色器（Vertex Shader）是渲染管线中第一个可编程阶段，对每个输入顶点独立执行。顶点着色器的核心任务是将顶点位置从模型空间变换到裁剪空间（`gl_Position = MVP * vec4(pos, 1.0)`），同时可以计算和传递其他顶点属性（法线变换、纹理坐标变换、颜色、雾化参数等）。顶点着色器必须输出裁剪空间位置，其余输出为可选。输入数据来自顶点缓冲对象（VBO），通过顶点属性指针（Vertex Attribute Pointer）描述每个顶点的数据布局（位置、法线、UV 等的偏移和步长）。实例化渲染（Instanced Rendering）允许同一网格的不同实例使用不同的变换矩阵，通过 `gl_InstanceID` 在着色器中索引实例数据。

图元装配（Primitive Assembly）是紧随顶点着色器的固定功能阶段，将顶点着色器输出的顶点按照绘制命令指定的图元类型（GL_TRIANGLES、GL_TRIANGLE_STRIP 等）组装为完整的图元。对于三角形，装配阶段会执行背面剔除（Back-Face Culling）：通过计算三角形的屏幕空间面积（或检查其法线方向与视线方向的点积）来判断正背面。如果三角形是背面（在默认正面定义下），则被剔除不进入后续阶段。正面的定义可以配置为顺时针（CW）或逆时针（CCW），默认 OpenGL 使用逆时针为正面。曲面细分（Tessellation）阶段在图元装配前后分别有控制点着色器（TCS）和评估着色器（TES），可以将低多边形网格细分为高精度曲面。

## 关键结论

- 顶点着色器是并行执行的，每个顶点的处理完全独立，无法访问其他顶点的数据。需要相邻顶点信息时，应在 CPU 端预计算或使用几何着色器
- 顶点属性的数量和类型由 VAO（顶点数组对象）描述，每个属性有独立的位置、类型、归一化标志和步长
- 背面剔除在图元装配阶段执行，在顶点着色器之后、光栅化之前。被剔除的三角形不进入光栅化，节省了片元着色器的计算
- 硬件实例化（Hardware Instancing）可以一次绘制调用渲染大量相同网格的不同实例，是渲染草地、粒子等场景的关键优化
- 顶点着色器可以输出自定义 varying 变量，这些变量在光栅化阶段会被重心坐标插值后传入片元着色器

## 易错点

1. **顶点属性布局错误**：VBO 中数据的内存布局（stride、offset）必须与着色器中的 layout 声明一致，否则数据解释错误。使用 `layout(location = 0)` 等显式绑定可以减少歧义
2. **忘记归一化法线**：从模型文件加载的法线可能未归一化。在顶点着色器中变换法线后必须重新归一化（`normalize(normalMatrix * normal)`），否则光照计算错误
3. **正面方向设置错误**：如果模型的三角形绕序与引擎期望不一致，背面剔除会错误地剔除正面三角形，导致物体"消失"或"inside-out"

## 例题

**题目**：一个顶点着色器接收顶点位置 `vec3 aPos` 和法线 `vec3 aNormal`，需要将位置变换到裁剪空间并将法线变换到世界空间后传给片元着色器。已知 `uniform mat4 MVP` 和 `uniform mat3 normalMatrix`（模型矩阵左上角 3x3 的逆转置），请编写顶点着色器。

**解答**：

```glsl
#version 330 core

// 顶点属性输入
layout(location = 0) in vec3 aPos;
layout(location = 1) in vec3 aNormal;

// 输出到片元着色器
out vec3 vNormal;
out vec3 vWorldPos;

// Uniform 变量
uniform mat4 MVP;
uniform mat4 model;
uniform mat3 normalMatrix;

void main() {
    // 变换到裁剪空间（必须输出）
    gl_Position = MVP * vec4(aPos, 1.0);

    // 变换法线到世界空间并归一化
    vNormal = normalize(normalMatrix * aNormal);

    // 变换位置到世界空间（用于光照计算）
    vWorldPos = vec3(model * vec4(aPos, 1.0));
}
```

关键点：
- `gl_Position` 是内置输出，必须赋值
- `normalMatrix` 是模型矩阵左上角 3x3 的逆转置矩阵，用于正确变换法线
- 法线变换后用 `normalize()` 确保单位化
- 世界空间位置通过 `model` 矩阵单独变换（不是 MVP）
