## 核心定义

四元数（Quaternion）是超复数的一种扩展，由一个实部和三个虚部组成，记为 $q = w + xi + yj + zk$，其中 $i^2 = j^2 = k^2 = ijk = -1$。在图形学中，通常写作 $q = (w, \mathbf{v})$，其中 $w$ 是标量部分，$\mathbf{v} = (x, y, z)$ 是向量部分。单位四元数（$|q| = 1$）可以表示三维旋转：绕单位轴 $\hat{\mathbf{n}}$ 旋转角度 $\theta$ 的四元数为 $q = (\cos\frac{\theta}{2}, \sin\frac{\theta}{2} \cdot \hat{\mathbf{n}})$。用四元数旋转点 $\mathbf{p}$ 的公式为 $\mathbf{p}' = q\mathbf{p}q^{-1} = q\mathbf{p}q^*$（单位四元数的逆等于共轭 $q^* = (w, -\mathbf{v})$）。

与欧拉角和旋转矩阵相比，四元数具有独特优势。欧拉角（Euler Angles）用三个角度 $(\phi, \theta, \psi)$ 分别表示绕三个轴的旋转，存在万向锁（Gimbal Lock）问题——当中间角度为 $90°$ 时，三个旋转轴退化为两个，丢失一个自由度。旋转矩阵虽然没有万向锁，但需要 9 个浮点数，且多次乘法后会产生数值漂移（非正交化），需要定期重新正交化。四元数只需要 4 个浮点数（存储更紧凑），支持球面线性插值（SLERP）实现平滑过渡，且不存在万向锁问题。四元数的乘法 $q_1 q_2$ 表示先应用 $q_2$ 再应用 $q_1$ 的复合旋转。

## 关键结论

- 单位四元数的集合在乘法下构成 $SO(3)$ 旋转群的双重覆盖：$q$ 和 $-q$ 表示相同的旋转，这是四元数表示旋转的基本性质
- SLERP（球面线性插值）公式 $\text{SLERP}(q_0, q_1, t) = \frac{\sin((1-t)\Omega)}{\sin\Omega}q_0 + \frac{\sin(t\Omega)}{\sin\Omega}q_1$，其中 $\cos\Omega = q_0 \cdot q_1$，保证等角速度旋转
- 四元数到旋转矩阵的转换：矩阵元素可以用四元数分量的二次组合表示，计算量远小于逐次三角函数计算
- 四元数乘法不满足交换律：$q_1 q_2 \neq q_2 q_1$，这反映了三维旋转本身不满足交换律的事实
- NLERP（归一化线性插值）是 SLERP 的低开销近似，在角度较小时误差可接受，常用于骨骼动画

## 易错点

1. **万向锁理解错误**：万向锁不是四元数的问题，而是欧拉角的问题。当中间旋转轴旋转 $90°$ 时，第一个和第三个旋转轴共线，导致一个自由度丢失。四元数通过避免分解为三个独立旋转来规避此问题
2. **四元数乘法顺序混淆**：$q_1 q_2$ 表示先 $q_2$ 后 $q_1$（与矩阵乘法约定一致），但在某些引擎文档中描述可能相反，需要查阅具体约定
3. **插值时未选择最短路径**：当 $q_0 \cdot q_1 < 0$ 时，应将其中一个四元数取反（$q_1 \leftarrow -q_1$），否则 SLERP 会走"长弧"路径，导致旋转超过 $180°$

## 例题

**题目**：求绕 y 轴旋转 $120°$ 的四元数，并用该四元数旋转点 $P(1, 0, 0)$。

**解答**：

第一步，构造四元数。旋转轴 $\hat{\mathbf{n}} = (0, 1, 0)$，角度 $\theta = 120°$：
$$q = (\cos 60°, \sin 60° \cdot (0, 1, 0)) = (0.5, 0, \frac{\sqrt{3}}{2}, 0)$$

第二步，将点 P 表示为纯四元数 $p = (0, 1, 0, 0)$。

第三步，计算共轭 $q^* = (0.5, 0, -\frac{\sqrt{3}}{2}, 0)$。

第四步，先计算 $qp$：
$$qp = (0.5, 0, \frac{\sqrt{3}}{2}, 0)(0, 1, 0, 0) = (-0 \cdot \frac{\sqrt{3}}{2}, 0.5 \cdot 1 + \frac{\sqrt{3}}{2} \cdot 0, 0.5 \cdot 0 - 0 \cdot 0, ...) $$

用公式展开得 $qp = (0, 0.5, 0, -\frac{\sqrt{3}}{2})$。

第五步，计算 $(qp)q^*$：
$$(qp)q^* = (0, 0.5, 0, -\frac{\sqrt{3}}{2})(0.5, 0, -\frac{\sqrt{3}}{2}, 0) = (-0.5, 0, 0, -\frac{\sqrt{3}}{2})$$

取向量部分并验证：绕 y 轴旋转 $120°$，点 $(1,0,0)$ 应变为 $(-0.5, 0, -\frac{\sqrt{3}}{2})$，即 $(\cos 120°, 0, -\sin 120°)$。结果正确。
