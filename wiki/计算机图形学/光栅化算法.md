## 核心定义

光栅化（Rasterization）是渲染管线中的固定功能阶段，将三角形等几何图元转换为离散的片元（fragment），供片元着色器处理。光栅化的核心问题是：确定三角形覆盖了哪些像素。主要有两种算法：扫描线算法（Scanline）和边界函数算法（Edge Function）。扫描线算法逐行扫描，利用三角形边的连续性计算每行的起止 x 坐标，适合 CPU 实现。现代 GPU 使用边界函数（Edge Function）方法：对于三角形的每条边，定义一个线性函数 $E_{AB}(x,y) = (x-x_A)(y_B-y_A) - (y-y_A)(x_B-x_A)$，如果 $E > 0$ 则点在边的左侧（假设逆时针绕序）。一个像素中心在三角形内部当且仅当它在三条边的同一侧。

重心坐标（Barycentric Coordinates）用于在三角形内部插值顶点属性。三角形内任意点 $P$ 可以表示为 $P = \alpha A + \beta B + \gamma C$，其中 $\alpha + \beta + \gamma = 1$，$\alpha, \beta, \gamma \geq 0$。重心坐标可以用面积比或边界函数计算：$\alpha = E_{BC}(P) / E_{BC}(A)$，$\beta = E_{CA}(P) / E_{CA}(B)$，$\gamma = 1 - \alpha - \beta$。光栅化阶段会将顶点着色器输出的所有属性（颜色、法线、UV 等）用重心坐标进行透视正确的插值，传递给片元着色器。需要注意的是，透视正确的插值不能直接在屏幕空间做线性插值，必须对 $1/z$ 做线性插值后取倒数：$\frac{v}{w}$ 在屏幕空间是线性的，可以正确插值。

## 关键结论

- 边界函数方法天然适合并行化：GPU 可以将三角形的包围盒分成小的 tile，每个 tile 并行测试其中的像素是否在三角形内
- 透视正确的属性插值公式：$f = \frac{\alpha f_0/w_0 + \beta f_1/w_1 + \gamma f_2/w_2}{\alpha/w_0 + \beta/w_1 + \gamma/w_2}$，其中 $w_i$ 是裁剪空间的 $w$ 分量
- 超采样（Supersampling）在每个像素内放置多个采样点，对每个采样点独立判断是否在三角形内，可以提高边缘质量
- 覆盖采样（Coverage Sample）是 MSAA 的实现方式：多个采样点共享一个着色结果，只有覆盖信息是每采样点独立的
- GPU 的光栅化器会保守地生成位于三角形边界的片元，以避免出现缝隙（crack），但可能导致重复绘制

## 易错点

1. **屏幕空间插值失真**：在屏幕空间直接对顶点属性做线性插值（忽略透视校正）会导致纹理扭曲。GPU 硬件自动处理透视校正，但手动实现光栅化时容易出错
2. **重心坐标精度问题**：当三角形非常小时（退化三角形），重心坐标计算可能出现数值不稳定，导致插值结果异常
3. **边界像素的归属问题**：当像素中心恰好落在三角形边上时，不同 API 的处理规则不同（top-left rule），可能导致相邻三角形在边界像素上不一致或重复着色

## 例题

**题目**：三角形三个顶点在屏幕空间的坐标为 $A(0,0)$, $B(4,0)$, $C(0,3)$。求像素中心 $(1,1)$ 的重心坐标，并判断该像素是否在三角形内。

**解答**：

第一步，计算三条边的边界函数值：
$$E_{AB}(x,y) = (x-0)(0-0) - (y-0)(4-0) = -4y$$
$$E_{BC}(x,y) = (x-4)(3-0) - (y-0)(0-4) = 3(x-4) + 4y = 3x + 4y - 12$$
$$E_{CA}(x,y) = (x-0)(0-3) - (y-3)(0-0) = -3x$$

第二步，在像素 $(1,1)$ 处计算：
$$E_{AB}(1,1) = -4 < 0$$
$$E_{BC}(1,1) = 3 + 4 - 12 = -5 < 0$$
$$E_{CA}(1,1) = -3 < 0$$

三个边界函数值同号（都为负），说明 $(1,1)$ 在三角形内部。

第三步，计算重心坐标：
$$\alpha = \frac{E_{BC}(P)}{E_{BC}(A)} = \frac{-5}{0+0-12} = \frac{5}{12}$$
$$\beta = \frac{E_{CA}(P)}{E_{CA}(B)} = \frac{-3}{-3 \cdot 4} = \frac{3}{12} = \frac{1}{4}$$
$$\gamma = 1 - \frac{5}{12} - \frac{1}{4} = 1 - \frac{5}{12} - \frac{3}{12} = \frac{4}{12} = \frac{1}{3}$$

验证：$\alpha + \beta + \gamma = \frac{5}{12} + \frac{3}{12} + \frac{4}{12} = 1$，结果正确。
