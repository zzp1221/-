## 核心定义

最大子数组问题（Maximum Subarray Problem）要求在给定的一维数组中找到连续子数组，使其元素之和最大。该问题由 Ulf Grenander 在1977年提出，最初是为了在一维数字图像中找到最大密度的矩形区域。分治解法将数组从中间等分为两半，最大子数组必然位于三种情况之一：完全在左半部分、完全在右半部分、跨越中点。跨越中点的情况需要 O(n) 时间计算，递归式为 T(n) = 2T(n/2) + O(n)，解得 O(n log n)。更优的 Kadane 算法（动态规划）将时间复杂度降至 O(n)，空间复杂度降至 O(1)。该问题在股票交易（一次买入卖出求最大收益）、信号处理、数据挖掘等领域有广泛应用。

## 关键结论

- 分治法解最大子数组的时间复杂度为 O(n log n)，空间复杂度为 O(log n)（递归栈）
- Kadane 算法的核心状态转移：dp[i] = max(arr[i], dp[i-1] + arr[i])，含义是以位置 i 结尾的最大子数组和
- 如果所有元素均为负数，则最大子数组和等于最大的单个元素（不能取空子数组）
- 二维扩展：最大子矩阵问题可在 O(n^3) 时间内使用 Kadane 算法逐行压缩求解
- 该问题的另一种描述为"最大连续子序列和"，常见于算法竞赛和面试题

## 易错点

1. 初始值设置错误：Kadane 算法中 `max_ending_here` 和 `max_so_far` 应初始化为第一个元素而非 0，否则在处理全负数组时会出错。
2. 跨中点情况的计算遗漏一端：必须同时向左右两个方向扫描，且左半部分从中点向左侧延伸，右半部分从中点+1向右侧延伸。
3. 将子数组误解为可以不连续：最大子数组问题要求元素是连续的。若不需要连续，则问题退化为选取所有正数即可，毫无算法设计意义。

## 例题

**例题1：** 给定数组 [-2, 1, -3, 4, -1, 2, 1, -5, 4]，求最大子数组和及对应子数组。

**解答：** 使用 Kadane 算法。遍历过程：[-2], [1], [-2], [4], [3], [5], [6], [1], [5]。最大值为 6，对应子数组为 [4, -1, 2, 1]。核心逻辑：每一步决定是继续扩展前面的子数组还是重新开始一个新的子数组。

**例题2：** 证明 Kadane 算法的正确性。

**解答：** 使用数学归纳法。设 dp[i] 为以 arr[i] 结尾的最大子数组和。归纳假设 dp[i-1] 正确计算出以 arr[i-1] 结尾的最大子数组和。对于 dp[i]，只有两种可能：单独 arr[i] 或 dp[i-1] 扩展加 arr[i]。取最大值即 dp[i]，递推正确。

## 代码示例

```python
# 最大子数组问题的多种解法

# 1. Kadane 算法 O(n), O(1)
def max_subarray_kadane(arr):
    if not arr:
        return 0, -1, -1
    max_so_far = arr[0]
    max_ending_here = arr[0]
    start = end = s = 0
    
    for i in range(1, len(arr)):
        if max_ending_here + arr[i] > arr[i]:
            max_ending_here += arr[i]
        else:
            max_ending_here = arr[i]
            s = i
        
        if max_ending_here > max_so_far:
            max_so_far = max_ending_here
            start = s
            end = i
    
    return max_so_far, start, end

# 2. 分治法 O(n log n)
def max_subarray_dc(arr, lo, hi):
    if lo == hi:
        return arr[lo], lo, hi
    
    mid = (lo + hi) // 2
    left_sum, left_lo, left_hi = max_subarray_dc(arr, lo, mid)
    right_sum, right_lo, right_hi = max_subarray_dc(arr, mid + 1, hi)
    cross_sum, cross_lo, cross_hi = max_crossing(arr, lo, mid, hi)
    
    if left_sum >= right_sum and left_sum >= cross_sum:
        return left_sum, left_lo, left_hi
    elif right_sum >= left_sum and right_sum >= cross_sum:
        return right_sum, right_lo, right_hi
    else:
        return cross_sum, cross_lo, cross_hi

def max_crossing(arr, lo, mid, hi):
    left_sum = float('-inf')
    s = 0
    max_left = mid
    for i in range(mid, lo - 1, -1):
        s += arr[i]
        if s > left_sum:
            left_sum = s
            max_left = i
    
    right_sum = float('-inf')
    s = 0
    max_right = mid + 1
    for i in range(mid + 1, hi + 1):
        s += arr[i]
        if s > right_sum:
            right_sum = s
            max_right = i
    
    return left_sum + right_sum, max_left, max_right

# 3. 最大子矩阵和 O(n^3)
def max_submatrix(matrix):
    if not matrix or not matrix[0]:
        return 0
    rows, cols = len(matrix), len(matrix[0])
    max_sum = float('-inf')
    
    for left in range(cols):
        col_sum = [0] * rows
        for right in range(left, cols):
            for i in range(rows):
                col_sum[i] += matrix[i][right]
            # 对 col_sum 运行 Kadane 算法
            cur = col_sum[0]
            best = cur
            for i in range(1, rows):
                cur = max(col_sum[i], cur + col_sum[i])
                best = max(best, cur)
            max_sum = max(max_sum, best)
    
    return max_sum
```
