## 核心定义

分治法（Divide and Conquer）是一种经典的算法设计范式，它将原问题递归地分解为若干个规模较小的同类子问题，独立求解这些子问题，然后将子问题的解合并为原问题的解。分治法包含三个标准步骤：分解（Divide），将原问题划分为若干个规模较小的子问题（通常是原问题规模的1/b），子问题与原问题形式相同；治理（Conquer），递归求解各个子问题，当子问题规模足够小时直接求解（基准条件）；合并（Combine），将子问题的解组合成原问题的解。分治法的正确性基于数学归纳法：假设子问题的解是正确的，证明合并步骤能产生正确的全局解。分治法之所以高效，是因为它将规模为n的问题分解为a个规模为n/b的子问题，当 b > 1 且 a > 1 时，递归树的深度为 O(log_b n)，从而获得对数或近似对数的性能。

## 关键结论

- 分治法的时间复杂度由递归式 T(n) = aT(n/b) + f(n) 描述，可用主定理求解
- 分解与合并步骤的工作量 f(n) 是影响分治法效率的关键因素
- 分治法天然适合并行计算，因为子问题彼此独立，可以在多核处理器上同时执行
- 分治法产生的子问题之间不应有重叠（否则应使用动态规划），子问题的规模应大致相等以保证平衡（Balanced）
- 分治法不仅适用于排序搜索，还广泛用于矩阵运算、计算几何、FFT、VLSI设计等

## 易错点

1. 子问题规模不平衡导致效率退化：如果每次划分极不均匀（如快速排序每次选到最小/最大元素），分治法的递归深度退化为 O(n)，复杂度从 O(n log n) 降到 O(n^2)。
2. 合并步骤被遗忘或实现错误：某些问题的难点不在分解而在合并（如最近点对问题），合并步骤的复杂度直接决定最终性能。
3. 分治法的"分"不是简单地将数据拆分：分解需要保证子问题与原问题具有相同的结构，且子问题之间不能相互依赖。

## 例题

**例题1：** 使用分治法求解最大子数组问题。

**解答：** 将数组 arr[lo..hi] 从中间 mid 分为两半。最大子数组要么完全在左半部分、要么完全在右半部分、要么跨越中点。前两者递归求解；第三者需 O(n) 时间向两边延伸扫描。递归式 T(n) = 2T(n/2) + O(n) = O(n log n)。可以通过 Kadane 算法进一步优化到 O(n)。

**例题2：** 利用分治法快速计算 x^n。

**解答：** 若 n 为偶数，x^n = (x^{n/2})^2；若 n 为奇数，x^n = x * (x^{(n-1)/2})^2。递归式 T(n) = T(n/2) + O(1)，解得 T(n) = O(log n)。这是快速幂（Fast Exponentiation）的核心思想。

## 代码示例

```python
# 分治法经典实现

# 1. 归并排序 O(n log n)
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])       # 分解 + 递归治理
    right = merge_sort(arr[mid:])
    return _merge(left, right)         # 合并 O(n)

def _merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i]); i += 1
        else:
            result.append(right[j]); j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result

# 2. 分治法求最大连续子数组和 O(n log n)
def max_subarray(arr, lo, hi):
    if lo == hi:
        return arr[lo] if arr[lo] > 0 else 0
    mid = (lo + hi) // 2
    # 递归求解左右两部分
    left_max = max_subarray(arr, lo, mid)
    right_max = max_subarray(arr, mid + 1, hi)
    # 跨越中点的最大子数组
    cross_max = _max_crossing(arr, lo, mid, hi)
    return max(left_max, right_max, cross_max)

def _max_crossing(arr, lo, mid, hi):
    left_sum = float('-inf')
    s = 0
    for i in range(mid, lo - 1, -1):
        s += arr[i]
        left_sum = max(left_sum, s)
    right_sum = float('-inf')
    s = 0
    for i in range(mid + 1, hi + 1):
        s += arr[i]
        right_sum = max(right_sum, s)
    return left_sum + right_sum

# 3. 分治法求 x^n (快速幂)  O(log n)
def fast_pow(x, n):
    if n == 0:
        return 1
    half = fast_pow(x, n // 2)
    if n % 2 == 0:
        return half * half
    else:
        return half * half * x
```
