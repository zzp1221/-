## 核心定义

最长公共子序列（Longest Common Subsequence, LCS）问题是经典的字符串比较问题，要求找出两个序列（或字符串）X 和 Y 的共同子序列中长度最长的一个。子序列（Subsequence）是从原序列中删除零个或多个元素（不改变剩余元素的顺序）得到的序列，不同于子串（Substring）要求连续。例如 X="ABCBDAB", Y="BDCABA"，LCS 为 "BCBA" 或 "BDAB"，长度为 4。LCS 问题具有典型的最优子结构和重叠子问题性质，是动态规划的标准例题。LCS 在生物信息学（DNA/蛋白质序列比对）、版本控制（文件 diff 算法）、自然语言处理（文本相似度）等领域有广泛应用。

## 关键结论

- 状态定义：dp[i][j] 表示 X[1..i] 和 Y[1..j] 的 LCS 长度
- 状态转移：若 X[i]==Y[j]，dp[i][j]=dp[i-1][j-1]+1；否则 dp[i][j]=max(dp[i-1][j], dp[i][j-1])
- 时间复杂度 O(mn)，空间复杂度 O(mn)（可优化至 O(min(m,n))）
- 最长公共子串（Longest Common Substring）是 LCS 的变体，要求连续，对应的转移方程略有不同
- LCS 与编辑距离（Edit Distance）关联紧密：编辑距离越小意味着 LCS 越长

## 易错点

1. 混淆子序列和子串：子序列不要求连续，子串要求连续。"ABCD" 是 "AXBYCD" 的子序列，但不是子串。
2. 边界条件处理：dp[0][j]=0, dp[i][0]=0。索引从 1 开始时字符串访问需减 1。
3. 回溯重构时的逻辑错误：从 dp[m][n] 反向走回 dp[0][0] 时，需要按状态转移的反方向移动。若 X[i]==Y[j] 则记录字符并向左上移动；否则根据 max 值向左或向上移动。

## 例题

**例题1：** 求 X="ABCBDAB", Y="BDCABA" 的 LCS 长度并重构 LCS。

**解答：** 构建 dp 表：dp[7][6]=4。回溯过程（从右下到左上）：dp[7][6] 由 dp[6][5]+1 得来（X[7]='B'==Y[6]='A'?不，由 max 得来），逐步追踪可得一个解 "BCBA"。

**例题2：** 将 LCS 问题推广到三个序列的版本。

**解答：** 状态 dp[i][j][k] 为 X[1..i], Y[1..j], Z[1..k] 的 LCS 长度。若 X[i]==Y[j]==Z[k] 则 dp[i][j][k]=dp[i-1][j-1][k-1]+1；否则取三个方向的最大值。时间复杂度 O(nmp)，空间 O(nmp)（三个序列长度分别为 n,m,p）。

## 代码示例

```python
def lcs_length(X, Y):
    """O(mn) 时间, O(mn) 空间"""
    m, n = len(X), len(Y)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if X[i - 1] == Y[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    return dp

def lcs_reconstruct(X, Y, dp):
    """回溯重构 LCS"""
    i, j = len(X), len(Y)
    result = []
    while i > 0 and j > 0:
        if X[i - 1] == Y[j - 1]:
            result.append(X[i - 1])
            i -= 1; j -= 1
        elif dp[i - 1][j] >= dp[i][j - 1]:
            i -= 1
        else:
            j -= 1
    return ''.join(reversed(result))

def lcs_optimized(X, Y):
    """空间优化: O(min(m,n))"""
    if len(X) < len(Y):
        X, Y = Y, X
    m, n = len(X), len(Y)
    prev = [0] * (n + 1)
    curr = [0] * (n + 1)
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if X[i - 1] == Y[j - 1]:
                curr[j] = prev[j - 1] + 1
            else:
                curr[j] = max(prev[j], curr[j - 1])
        prev, curr = curr, [0] * (n + 1)
    
    return prev[n]

# 示例
X, Y = "ABCBDAB", "BDCABA"
dp = lcs_length(X, Y)
print("LCS长度:", dp[len(X)][len(Y)])  # 4
print("LCS:", lcs_reconstruct(X, Y, dp))  # BCBA
print("空间优化版:", lcs_optimized(X, Y))  # 4
```
