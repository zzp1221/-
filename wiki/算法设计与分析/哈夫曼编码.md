## 核心定义

哈夫曼编码（Huffman Coding）是由 David Huffman 于 1952 年提出的贪心算法，用于构造最优前缀码（Optimal Prefix Code），实现数据的无损压缩。给定字符集 C 和每个字符出现的频率 f(c)，目标是为每个字符分配一个二进制编码，使得没有任何一个字符的编码是另一个字符编码的前缀（前缀性质），同时最小化文件编码后的总比特数 ∑ f(c) * len(code(c))。哈夫曼算法使用贪心策略构建一棵最优二叉树：建立一个以频率为键值的优先队列（最小堆），每次取出两个频率最小的节点合并为一个新节点（频率为两子节点之和），重复此过程直到只剩一个节点（即根节点）。树的叶节点代表字符，从根到叶的路径即为编码（左0右1或反之）。算法时间复杂度 O(n log n)（使用优先队列），空间复杂度 O(n)。

## 关键结论

- 哈夫曼编码是最优前缀码的证明依赖于贪心选择性质和最优子结构
- 哈夫曼编码的压缩率：对于频率分布极不均匀的字符集，压缩效果显著优于等长编码
- 平均码长满足 Shannon 熵下界：H <= 平均码长 < H+1，其中 H = -∑ p_i log_2 p_i
- 哈夫曼编码的变体：自适应哈夫曼编码（Adaptive Huffman，无需预知频率）、范式哈夫曼编码（Canonical Huffman，只需传输码长）
- 哈夫曼编码广泛应用于 JPEG、MP3、DEFLATE（gzip/zlib）、PNG 等压缩格式

## 易错点

1. 混淆哈夫曼树与二叉搜索树：哈夫曼树是根据频率构建的，不要求左子节点值小于右子节点值；它只关心加权路径长度最小。
2. 编码时必须记录码表（编码表/解码表）：解码端需要码字到字符的映射，通常通过传输哈夫曼树或使用范式哈夫曼仅传输码长。
3. 合并顺序错误：必须每次取出最小的两个。用普通数组而非优先队列实现会导致复杂度退化到 O(n^2)。

## 例题

**例题1：** 字符频率：A:5, B:9, C:12, D:13, E:16, F:45。构造哈夫曼树并计算加权路径长度。

**解答：** 过程：取 A(5)+B(9)=14（新节点）；14(AB)+C(12)=26；13(D)+14(AB)=27；16(E)+26(ABC)=42；27(ABD)+42(ABCE)=69；最后 45(F)+69=100。加权路径长度 = 5*4+9*4+12*3+13*3+16*3+45*1 = 20+36+36+39+48+45 = 224。若用等长 3 位编码则为 300 比特。

**例题2：** 证明哈夫曼算法的贪心选择性质。

**解答：** 设 x 和 y 是 C 中频率最低的两个字符。需证明存在一个最优前缀码使得 x 和 y 的码字长度相同且仅在最后一位不同（即它们是兄弟叶节点）。构造性证明：取任一最优编码树 T，交换最深层的两个叶与 x/y，交换后代价不会增加（因为 x,y 频率最小），得到包含 x,y 为兄弟的新最优树。

## 代码示例

```python
import heapq
from collections import Counter

class HuffmanNode:
    def __init__(self, char, freq):
        self.char = char
        self.freq = freq
        self.left = None
        self.right = None
    
    # 堆排序依据
    def __lt__(self, other):
        return self.freq < other.freq

def build_huffman_tree(text):
    """构建哈夫曼树"""
    freq = Counter(text)
    heap = [HuffmanNode(c, f) for c, f in freq.items()]
    heapq.heapify(heap)
    
    while len(heap) > 1:
        left = heapq.heappop(heap)
        right = heapq.heappop(heap)
        # 合并
        parent = HuffmanNode(None, left.freq + right.freq)
        parent.left = left
        parent.right = right
        heapq.heappush(heap, parent)
    
    return heap[0], freq

def build_codes(root):
    """从哈夫曼树生成编码表"""
    codes = {}
    def dfs(node, code):
        if node is None:
            return
        if node.char is not None:
            codes[node.char] = code
            return
        dfs(node.left, code + '0')
        dfs(node.right, code + '1')
    dfs(root, '')
    return codes

def huffman_encode(text):
    root, freq = build_huffman_tree(text)
    codes = build_codes(root)
    encoded = ''.join(codes[c] for c in text)
    return encoded, codes, root

def huffman_decode(encoded, root):
    """解码哈夫曼编码串"""
    result = []
    node = root
    for bit in encoded:
        node = node.left if bit == '0' else node.right
        if node.char is not None:
            result.append(node.char)
            node = root
    return ''.join(result)

# 示例
text = "aabbbccccddddd"
encoded, codes, root = huffman_encode(text)
print("频率:", Counter(text))
print("编码表:", codes)
print("编码后:", encoded)
print("解码验证:", huffman_decode(encoded, root))
print(f"压缩比: {len(text)*8}/{len(encoded)}")
```
