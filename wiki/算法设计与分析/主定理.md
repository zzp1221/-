## 核心定义

主定理（Master Theorem）是求解形如 T(n) = aT(n/b) + f(n) 的分治递归式的快速方法，也称主方法（Master Method）。其中 a >= 1 是子问题的个数，b > 1 是每次划分问题规模的缩减倍数，f(n) 是划分和合并子问题所需的时间。主定理通过比较 f(n) 与函数 n^{log_b a} 的增长率，将递归式的解划分为三种情况。主定理的精髓在于：分治算法的复杂度由"合并开销 f(n)"和"叶子节点开销 n^{log_b a}"中增长率更高的那一个决定；若两者相等（在多项式意义上），则还需乘一个对数因子。主定理极大简化了分治算法复杂度分析的流程。

## 关键结论

- 记 k = log_b a，即递归树的叶子节点数为 n^k
- Case 1（叶子主导）：若 f(n) = O(n^{k - ε})，ε > 0，则 T(n) = Θ(n^k)（合并开销小，复杂度由叶子节点决定）
- Case 2（均衡）：若 f(n) = Θ(n^k * log^w n)，则 T(n) = Θ(n^k * log^{w+1} n)（合并和叶子同等重要）
- Case 3（根主导）：若 f(n) = Ω(n^{k + ε})，ε > 0，且满足正则条件 af(n/b) <= cf(n), c < 1，则 T(n) = Θ(f(n))（合并开销主导）
- 非多项式差距：Case之间的差距必须是多项式级（n^ε 级别），对数差距不算，如 f(n) = n / log n 落在 Case 1 和 Case 2 的分隔线之外

## 易错点

1. 误判 Case 归属：f(n) = n * log n 和 n^{log_b a} = n 比较时，差距只有 log n（多项式差距为零），属于 Case 2 而非 Case 3。
2. 忘记了正则条件：Case 3 除了要求 f(n) 多项式地大于 n^{log_b a}，还必须验证正则条件 af(n/b) <= cf(n)。如果不满足（例如 f(n) = n * (2 - cos n)），主定理不适用。
3. 对 floor/ceil 的误解：当 b 不能整除 n 时使用 floor 或 ceil，主定理仍然有效。同样，对形如 T(n) = aT(n/b + c) + f(n) 的方程，在 n 足够大时也可以用主定理。

## 例题

**例题1：** T(n) = 9T(n/3) + n，求复杂度。

**解答：** a=9, b=3, f(n)=n。log_3 9 = 2。比较 f(n)=n 与 n^2 = n^2。n = O(n^{2 - 1}) = O(n)，取 ε=1。属于 Case 1。T(n) = Θ(n^2)。

**例题2：** T(n) = T(2n/3) + 1，求复杂度。

**解答：** a=1, b=3/2, f(n)=1。log_{3/2} 1 = 0。n^0 = 1。f(n) = 1 = Θ(n^0 * log^0 n)。属于 Case 2 (w=0)。T(n) = Θ(log n)。

## 代码示例

```python
import math

def master_theorem_solver(a, b, f_n_desc):
    """
    主定理可视化求解器
    a, b: 递归式参数  T(n) = aT(n/b) + f(n)
    """
    log_b_a = math.log(a, b)
    print(f"=== 主定理分析 ===")
    print(f"递归式: T(n) = {a}T(n/{b}) + f(n)")
    print(f"参数: a={a}, b={b}")
    print(f"关键指数: log_b(a) = log_{b}({a}) = {log_b_a:.4f}")
    print(f"核心比较函数: n^{log_b_a} = n^{log_b_a:.4f}")
    print()

    # 判定 Case
    print("请手动判断 Case 归属：")
    print(f"  Case 1: 若 f(n) = O(n^{log_b_a - ε}), T(n) = Θ(n^{log_b_a})")
    print(f"  Case 2: 若 f(n) = Θ(n^{log_b_a} log^k n), T(n) = Θ(n^{log_b_a} log^(k+1) n)")
    print(f"  Case 3: 若 f(n) = Ω(n^{log_b_a + ε}) 且正则条件, T(n) = Θ(f(n))")


# 经典案例分析
test_cases = [
    (2, 2, "n"),           # 归并排序 -> Θ(n log n)
    (4, 2, "n"),           # Strassen前的矩阵乘法 -> Θ(n^2)
    (1, 2, "1"),           # 二分查找 -> Θ(log n)
    (3, 2, "n^2"),         # Karatsuba -> Θ(n^1.585)
]

for a, b, f in test_cases:
    master_theorem_solver(a, b, f)
    print("-" * 50)
```
