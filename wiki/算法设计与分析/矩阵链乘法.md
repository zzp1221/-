## 核心定义

矩阵链乘法问题（Matrix Chain Multiplication, MCM）要求确定一组矩阵相乘的最优加括号顺序，使得标量乘法（Scalar Multiplication）的总次数最少。给定 n 个矩阵 A1, A2, ..., An，每个矩阵 Ai 的维度为 p_{i-1} x p_i。由于矩阵乘法满足结合律但不满足交换律，不同的加括号方式虽然产生相同的结果矩阵，但计算代价可能天差地别。例如 A: 10x100, B: 100x5, C: 5x50，((AB)C) 需 10*100*5 + 10*5*50 = 7500 次乘法，而 (A(BC)) 需 100*5*50 + 10*100*50 = 75000 次乘法，差距达 10 倍。MCM 是动态规划的经典例题，体现了"先在子问题上做最优选择，再组合成全局最优"的核心思想。

## 关键结论

- 穷举所有括号化方案的 Catalan 数为 C_{n-1}，呈指数级增长，动态规划将复杂度降至 O(n^3)
- 状态定义：dp[i][j] 表示计算从 Ai 到 Aj 的最优标量乘法次数
- 状态转移：dp[i][j] = min_{i <= k < j} { dp[i][k] + dp[k+1][j] + p_{i-1} * p_k * p_j }
- 填表顺序：按链长 L = j - i 从小到大的顺序计算（因为短链是长链的子问题）
- 除了计算最小代价还需记录分割点 k，以便重构最优括号化方案

## 易错点

1. 填表的方向错误：dp[i][j] 依赖 dp[i][k] 和 dp[k+1][j]，必须按链长递增顺序填表。直接双重循环 i/j 会访问未计算的状态。
2. 混淆维度的含义：p 数组有 n+1 个元素（p[0] 到 p[n]），Ai 的维度是 p[i-1] x p[i]。
3. 遗漏乘法代价的计算：每次合并左右两个子结果时，除子问题的代价外，还需加上当前合并的乘法次数 p[i-1]*p[k]*p[j]。

## 例题

**例题1：** 给定 p = [10, 30, 5, 60]，即 A1:10x30, A2:30x5, A3:5x60。求最优括号化和最小乘法次数。

**解答：** L=1: dp[1][1]=dp[2][2]=dp[3][3]=0。L=2: dp[1][2]=10*30*5=1500 (k=1)；dp[2][3]=30*5*60=9000 (k=2)。L=3: dp[1][3]=min(dp[1][1]+dp[2][3]+10*30*60=0+9000+18000=27000 (k=1), dp[1][2]+dp[3][3]+10*5*60=1500+0+3000=4500 (k=2)) = 4500。最优方案：((A1*A2)*A3)，4500次标量乘法。

**例题2：** 分析 MCM 的时间复杂度 O(n^3)。

**解答：** 三重循环：链长 L 从 2 到 n（O(n)），起点 i 从 1 到 n-L+1（O(n)），分割点 k 从 i 到 j-1（O(n)）。每层约为 n^2/2，k 的遍历平均为 n/3。所以总复杂度 = O(n^3)，常数因子约 1/6。

## 代码示例

```python
def matrix_chain_order(p):
    """
    p: 维度数组, p[0..n], Ai 维度 p[i-1] x p[i]
    返回: (最小代价表dp, 最优分割点表s)
    """
    n = len(p) - 1  # 矩阵个数
    dp = [[0] * (n + 1) for _ in range(n + 1)]
    s = [[0] * (n + 1) for _ in range(n + 1)]
    
    # L 为链长, 从2开始
    for L in range(2, n + 1):
        for i in range(1, n - L + 2):
            j = i + L - 1
            dp[i][j] = float('inf')
            for k in range(i, j):
                cost = dp[i][k] + dp[k + 1][j] + p[i - 1] * p[k] * p[j]
                if cost < dp[i][j]:
                    dp[i][j] = cost
                    s[i][j] = k
    
    return dp, s

def print_optimal_parens(s, i, j):
    """递归构建最优括号化方案"""
    if i == j:
        return f"A{i}"
    k = s[i][j]
    left = print_optimal_parens(s, i, k)
    right = print_optimal_parens(s, k + 1, j)
    return f"({left} * {right})"

# 示例
p = [10, 30, 5, 60]
dp, s = matrix_chain_order(p)
print("最小标量乘法次数:", dp[1][3])  # 4500
print("最优括号化:", print_optimal_parens(s, 1, 3))  # ((A1 * A2) * A3)
```
