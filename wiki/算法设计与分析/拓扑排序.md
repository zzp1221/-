## 核心定义

拓扑排序（Topological Sorting）是对有向无环图（DAG, Directed Acyclic Graph）的顶点进行线性排序，使得对于任意一条有向边 (u,v)，顶点 u 在排序中出现在顶点 v 之前。拓扑排序反映了图中顶点之间的偏序（Partial Order）关系，是任务调度、课程安排、编译依赖解析等问题的核心算法。拓扑排序存在的充要条件是图是 DAG。两种经典实现：Kahn 算法（BFS 变种），使用队列维护入度为 0 的顶点，每次取出一个顶点输出并从图中删除它的所有出边，更新邻居的入度；DFS 算法，后序遍历图并在遍历完成后将顶点压入栈中，最终栈的逆序即为拓扑排序。

## 关键结论

- 图必须是 DAG：若含有环，拓扑排序不存在（Kahn 算法中环内顶点入度永不降为零）
- Kahn 算法 O(V+E)：使用队列维护入度为零的顶点，可以检测环
- DFS 算法 O(V+E)：利用后序遍历时间戳（finish time）的逆序排列
- 拓扑排序不唯一：当多个顶点同时入度为 0 时，排序结果取决于选择顺序
- 应用场景：课程先修关系、Makefile 构建依赖、任务调度中的关键路径分析、PERT 图

## 易错点

1. 非连通图的处理：图中可能有多个连通分量，Kahn 和 DFS 都能正确处理（Kahn 初始入度为 0 的可能有多个图源；DFS 需对每个未访问顶点启动遍历）。
2. 环检测与拓扑排序的输出不矛盾：Kahn 算法若最终输出的顶点数 < |V|，则图含环。
3. 递归 DFS 的栈深度可能溢出：对于深链式的 DAG，需使用迭代 DFS 或 Kahn 算法。

## 例题

**例题1：** 给课程依赖关系：A->B, A->C, B->D, C->D。列出所有合法拓扑序。

**解答：** 构建DAG。Kahn：初始入度为 0 的是 A。输出 A 后 B,C 入度为 0。两种顺序：A,B,C,D 或 A,C,B,D。DFS 后序逆序也可得同样结果。

**例题2：** 使用拓扑排序检测有向图中的环路。

**解答：** Kahn 算法：若最终输出顶点数量 < |V|，剩余顶点必然构成环（每个环内顶点入度至少为 1，没有任何顶点能被输出）。DFS 方法：若在 DFS 期间遇到"灰色"状态的顶点（正在访问中），即存在后向边（Back Edge），相当于检测到环。

## 代码示例

```python
from collections import deque

# Kahn 算法 O(V+E)
def topological_sort_kahn(n, edges):
    """edges: [(u, v), ...]"""
    graph = [[] for _ in range(n)]
    indegree = [0] * n
    
    for u, v in edges:
        graph[u].append(v)
        indegree[v] += 1
    
    q = deque([i for i in range(n) if indegree[i] == 0])
    result = []
    
    while q:
        u = q.popleft()
        result.append(u)
        for v in graph[u]:
            indegree[v] -= 1
            if indegree[v] == 0:
                q.append(v)
    
    if len(result) < n:
        return None  # 有环
    return result

# DFS 算法 O(V+E)
def topological_sort_dfs(n, edges):
    graph = [[] for _ in range(n)]
    for u, v in edges:
        graph[u].append(v)
    
    WHITE, GRAY, BLACK = 0, 1, 2
    color = [WHITE] * n
    result = []
    
    def dfs(u):
        color[u] = GRAY
        for v in graph[u]:
            if color[v] == GRAY:
                return False  # 检测到环
            if color[v] == WHITE:
                if not dfs(v):
                    return False
        color[u] = BLACK
        result.append(u)
        return True
    
    for i in range(n):
        if color[i] == WHITE:
            if not dfs(i):
                return None  # 有环
    
    return list(reversed(result))  # 后序逆序

# 示例
n = 4
edges = [(0,1), (0,2), (1,3), (2,3)]
print("Kahn:", topological_sort_kahn(n, edges))
print("DFS:", topological_sort_dfs(n, edges))
```
