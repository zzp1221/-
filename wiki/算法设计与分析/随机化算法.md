## 核心定义

随机化算法（Randomized Algorithm）是在执行过程中引入随机选择的算法。与确定性算法不同，随机化算法的行为（包括运行时间和输出正确性）可能依赖于随机数生成器的输出。随机化算法分为两大类：Las Vegas 算法，保证输出总是正确的，但运行时间是随机变量（如随机化快速排序的期望时间复杂度为 O(n log n)）；Monte Carlo 算法，运行时间固定或不超过某个界，但输出以一定概率正确（如 Miller-Rabin 素性测试——若返回"合数"则一定正确，返回"素数"仅以高概率正确）。随机化算法的优势在于通常比确定性算法更简单、更高效，尤其在遇到确定性算法的最坏输入时，随机化可以"平滑"性能。

## 关键结论

- Las Vegas 算法：总是返回正确结果，期望时间有界（随机快排期望 O(n log n)）
- Monte Carlo 算法：固定时间，结果有概率误差，通常可通过独立重复运行将错误概率降至任意小（如 Miller-Rabin 重复 k 次，错误率 < 4^{-k}）
- 随机化算法的分析需要概率论工具：期望值、指示器随机变量、Chernoff 界等
- 去随机化（Derandomization）研究如何将随机化算法转化为确定性算法，常使用条件期望方法
- 关键应用：随机快排、跳跃表（Skip List）、随机化哈希（Universal Hashing）、随机化选择算法（QuickSelect）

## 易错点

1. Monte Carlo 和 Las Vegas 的混淆：Las Vegas 保证正确性但时间可变；Monte Carlo 保证时间但正确性可变。两者可相互转化。
2. 概率分析中容易误用"平均情况"：随机化算法的最坏情况分析考虑的是随机数的分布而非输入的分布，对任何输入都给出期望保证。
3. 随机数生成器（RNG）质量影响算法效果：需要使用高质量的伪随机数生成器（如 Mersenne Twister），避免使用低质量的 Rand()。

## 例题

**例题1：** 分析随机化快速排序的期望时间复杂度。

**解答：** 设 X_{ij} 为指示器随机变量，表示第 i 小和第 j 小的元素在排序过程中被比较。分析得 P(X_{ij}=1) = 2/(j-i+1)。总比较次数期望 = sum_{i<j} 2/(j-i+1) = O(n log n)。注意这是针对任何输入的最坏期望时间。

**例题2：** 用 Monte Carlo 方法估算圆周率 π。写算法并分析精度。

**解答：** 在 [-1,1]^2 正方形内随机均匀采样 N 个点，统计落在单位圆内的点数 M。π ≈ 4M/N。根据中心极限定理，误差 ~ O(1/√N)。N=10^6 时约 3 位有效数字。

## 代码示例

```python
import random

# Las Vegas: 随机化快速排序
def randomized_quicksort(arr, lo, hi):
    if lo >= hi:
        return
    pivot_idx = random.randint(lo, hi)
    arr[pivot_idx], arr[hi] = arr[hi], arr[pivot_idx]
    p = partition(arr, lo, hi)
    randomized_quicksort(arr, lo, p - 1)
    randomized_quicksort(arr, p + 1, hi)

def partition(arr, lo, hi):
    pivot = arr[hi]
    i = lo - 1
    for j in range(lo, hi):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i + 1], arr[hi] = arr[hi], arr[i + 1]
    return i + 1

# Monte Carlo: Miller-Rabin 素性测试
def miller_rabin(n, k=10):
    if n < 2:
        return False
    if n == 2 or n == 3:
        return True
    if n % 2 == 0:
        return False
    
    r, d = 0, n - 1
    while d % 2 == 0:
        r += 1
        d //= 2
    
    def witness(a):
        x = pow(a, d, n)  # 快速幂取模
        if x == 1 or x == n - 1:
            return True  # 可能为素数
        for _ in range(r - 1):
            x = (x * x) % n
            if x == n - 1:
                return True
        return False
    
    for _ in range(k):
        a = random.randint(2, n - 2)
        if not witness(a):
            return False  # 确定为合数
    return True  # 高概率为素数

# Monte Carlo: 估算圆周率
def estimate_pi(n):
    inside = 0
    for _ in range(n):
        x = random.uniform(-1, 1)
        y = random.uniform(-1, 1)
        if x * x + y * y <= 1:
            inside += 1
    return 4 * inside / n

print(estimate_pi(100000))
print(miller_rabin(2147483647, 20))  # 大质数检测
```
