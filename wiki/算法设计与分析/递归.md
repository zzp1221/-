## 核心定义

递归（Recursion）是一种算法设计技术，指函数在其定义中直接或间接调用自身的过程。一个正确的递归算法必须包含两个基本要素：基准条件（Base Case），即不再递归调用的终止条件，用于防止无限递归；递归条件（Recursive Case），即将原问题分解为规模更小的同类子问题并递归求解。递归的理论基础是数学归纳法：基准条件对应归纳基础，递归条件对应归纳步骤，递归调用的正确性建立在更小子问题已正确求解的假设之上。递归将一个复杂问题逐步化简到简单情况，体现了"分而治之"的思想，是许多高效算法（如分治、动态规划）的基础技术。然而递归也带来额外的调用栈开销，可能导致栈溢出或重复计算。

## 关键结论

- 递归函数的内存开销主要来自调用栈（Call Stack），每层递归需要保存返回地址和局部变量
- 尾递归（Tail Recursion）是递归调用位于函数末尾的特殊形式，可被编译器优化为迭代，消除栈开销
- 递归式（Recurrence）是描述递归算法时间复杂度的方程，形如 T(n) = aT(n/b) + f(n)，求解放法有代入法、递归树法和主定理
- 递归并非总是最佳选择：当递归深度过大（如 n=10^6 时链表反转递归）可能导致栈溢出；有些递归存在大量重复计算（如简单斐波那契递归），应改用迭代或记忆化
- 递归与迭代可以互相转换：任何递归算法都可以通过显式栈结构转化为迭代形式

## 易错点

1. 忘记基准条件导致无限递归：递归函数必须确保每次递归调用都在向基准条件靠近，否则会栈溢出（StackOverflowError）。典型错误包括：参数不递减、基准条件永远不满足、相互递归中没有形成终止路径。
2. 递归深度过大导致栈溢出：操作系统对线程栈大小有限制（通常约8MB），深度为10^6的递归必然溢出。对于大数据集应改用迭代或通过尾递归优化来避免。
3. 将递归的空间复杂度误认为是O(1)：每次递归调用都会在栈上分配新的栈帧，其空间复杂度至少为O(递归深度)。例如递归版二分查找空间O(log n)，不是O(1)。

## 例题

**例题1：** 写出 Hanoi 塔问题的递归解法并分析时间复杂度。

**解答：** 将 n 个盘子从 A 移到 C 借助 B：
1. 将 n-1 个盘子从 A 移到 B 借助 C（递归）
2. 将最大的盘子从 A 移到 C（直接）
3. 将 n-1 个盘子从 B 移到 C 借助 A（递归）

递归式：T(n) = 2T(n-1) + 1, T(1) = 1。解得 T(n) = 2^n - 1 = O(2^n)。

**例题2：** 使用递归计算阶乘 n!，分析空间复杂度。

**解答：**
```python
def factorial(n):
    if n <= 1:      # 基准条件
        return 1
    return n * factorial(n - 1)  # 递归条件
```
时间复杂度 O(n)，空间复杂度 O(n)（递归深度为 n）。若使用尾递归优化或迭代实现，空间复杂度可降至 O(1)。

## 代码示例

```python
# 递归算法示例集合

# 1. 经典递归：斐波那契数列（含记忆化对比）
def fib_recursive(n):
    """朴素递归, O(2^n) 时间, O(n) 空间"""
    if n <= 1:
        return n
    return fib_recursive(n-1) + fib_recursive(n-2)

def fib_memo(n, memo=None):
    """记忆化递归, O(n) 时间, O(n) 空间"""
    if memo is None:
        memo = {}
    if n <= 1:
        return n
    if n not in memo:
        memo[n] = fib_memo(n-1, memo) + fib_memo(n-2, memo)
    return memo[n]

# 2. 尾递归求阶乘
def factorial_tail(n, acc=1):
    """尾递归, 编译器可优化为 O(1) 空间"""
    if n <= 1:
        return acc
    return factorial_tail(n-1, acc * n)

# 3. 递归式可视化：递归树分析
def recurrence_tree_example(n):
    """
    T(n) = 2T(n/2) + n, T(1) = 1
    归并排序递归式，每层总工作量 = n
    树高 = log_2 n
    总复杂度 = O(n log n)
    """
    if n <= 1:
        return 1
    return 2 * recurrence_tree_example(n // 2) + n

# 4. 组合枚举：生成所有子集 O(n * 2^n)
def subsets(nums):
    result = []
    def backtrack(start, current):
        result.append(current[:])              # 记录当前子集
        for i in range(start, len(nums)):
            current.append(nums[i])
            backtrack(i + 1, current)          # 递归探索
            current.pop()                      # 回溯
    backtrack(0, [])
    return result
```
