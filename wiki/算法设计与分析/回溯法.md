## 核心定义

回溯法（Backtracking）是一种系统搜索问题解空间的算法范式，可以视为带剪枝的深度优先搜索。它从部分解出发，逐步构建候选解。在每一步，算法尝试扩展当前部分解；若发现当前部分解不可能扩展为完整解（不满足约束条件或界限条件），则"回溯"到上一步并尝试其他选项。回溯法将问题的解空间组织为一棵"状态空间树"（State-Space Tree），通过深度优先的方式遍历这棵树，利用约束函数和限界函数进行剪枝，避免对不可能产生解的分支进行无意义探索。回溯法的典型应用包括 n 皇后问题、图的着色、哈密顿回路、子集和问题、0-1 背包问题等。回溯法在最坏情况下的时间复杂度通常为指数级，但通过合理剪枝可在实践中处理中等规模问题。

## 关键结论

- 回溯法的两种剪枝方式：约束函数（Constraint Function）剪去不满足显式或隐式约束的子树；限界函数（Bounding Function）剪去不可能产生优于当前最优解的子树
- 状态空间树的结构直接影响回溯效率：良好的解空间表示可减小搜索规模
- 回溯与分支限界的区别：回溯法是 DFS + 剪枝，分支限界法是 BFS/Best-First + 剪枝
- 回溯法的递归实现简洁直观，迭代实现更灵活（可自定义回溯粒度）
- 排列树和子集树是回溯法的两种典型解空间结构：排列树大小为 O(n!)，子集树大小为 O(2^n)

## 易错点

1. 忘记回溯时恢复现场：递归调用后必须撤销对全局状态的修改（如移除刚添加的元素），否则下一分支会在污染的状态上探索。
2. 剪枝条件写得过于宽松或过于严格：过于宽松导致剪枝不足效率低下；过于严格可能错误地剪去全局最优解所在的分支。
3. 混淆回溯法和分支限界法：回溯法一旦找到解就记录（可继续搜索更优解），分支限界法需要证明当前分支的上/下界已超越当前最优才剪枝。

## 例题

**例题1：** 用回溯法求解 4 皇后问题。

**解答：** 状态空间树为排列树。按行放置皇后，对每行（0~3）依次尝试各列。约束条件：不与已放置皇后在同一列、同一对角线。当行=4 时找到一个解。回溯过程：第 0 行列 0，第 1 行尝试列 2 可行；第 2 行逐一尝试均冲突，回溯到第 1 行换列 3；依此类推找到解。

**例题2：** 回溯法解子集和问题：S={5,10,12,13,15,18}, target=30，求所有子集。

**解答：** 含剪枝：对排序后的集合从第一个元素开始，每步有"选"和"不选"两种分支。约束函数：sum > target 则剪枝（因为已排序，后续更大）。限界函数：已选和 + 剩余全选 < target 则剪枝。最终找到 {5,10,15}, {12,18}。

## 代码示例

```python
# N皇后问题回溯法
def solve_n_queens(n):
    results = []
    # board[i] = j 表示第 i 行皇后在第 j 列
    board = [-1] * n
    # 辅助数组加速冲突检查
    cols = [False] * n
    diag1 = [False] * (2 * n - 1)  # 主对角线 (row-col+n-1)
    diag2 = [False] * (2 * n - 1)  # 副对角线 (row+col)
    
    def backtrack(row):
        if row == n:
            results.append(board[:])
            return
        for col in range(n):
            if cols[col] or diag1[row - col + n - 1] or diag2[row + col]:
                continue  # 约束剪枝
            # 做选择
            board[row] = col
            cols[col] = diag1[row - col + n - 1] = diag2[row + col] = True
            backtrack(row + 1)   # 递归探索
            # 回溯恢复
            cols[col] = diag1[row - col + n - 1] = diag2[row + col] = False
            board[row] = -1
    
    backtrack(0)
    return results

# 子集和问题回溯法（含剪枝）
def subset_sum_backtrack(nums, target):
    nums.sort()
    results = []
    
    def backtrack(start, current, remaining_sum):
        # 找到合法解
        if sum(current) == target:
            results.append(current[:])
            return
        if start >= len(nums):
            return
        
        for i in range(start, len(nums)):
            x = nums[i]
            # 约束剪枝：加入后超过目标
            if sum(current) + x > target:
                break
            # 限界剪枝：加上后面所有都不够
            if sum(current) + remaining_sum < target:
                break
            
            current.append(x)
            backtrack(i + 1, current, remaining_sum - x)
            current.pop()  # 回溯
    
    backtrack(0, [], sum(nums))
    return results

print("4皇后解数量:", len(solve_n_queens(4)))  # 2
print("子集和解:", subset_sum_backtrack([5,10,12,13,15,18], 30))
```
