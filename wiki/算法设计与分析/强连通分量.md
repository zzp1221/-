## 核心定义

在有向图中，若从顶点 u 可以到达 v 且从 v 可以到达 u，则 u 和 v 是强连通的（Strongly Connected）。强连通分量（SCC）是满足强连通关系的极大顶点子集。将有向图的所有 SCC 缩为一个点后，得到的 DAG 称为该图的"分量图"（Component Graph）。求解 SCC 有两种经典线性时间算法：Kosaraju 算法（两趟 DFS），第一趟在原图做 DFS 并记录完成时间（finish time），第二趟在转置图（反向所有边）上按 finish time 降序做 DFS，每棵 DFS 树即为一个 SCC；Tarjan 算法（一趟 DFS），在 DFS 过程中维护 dfn（发现时间）和 low（通过后代边能回溯到的最早 dfn），当 dfn[u] = low[u] 时 u 及其上方节点构成一个 SCC。

## 关键结论

- 分量图必定是 DAG：将每个 SCC 缩为一个超级顶点，边方向保持原图方向，不存在环路
- Kosaraju 需要两次完整 DFS，时间复杂度 O(V+E)，额外空间 O(V)（存储 finish time）
- Tarjan 仅需一次 DFS，时间复杂度 O(V+E)，需额外 O(V) 栈空间
- SCC 的重要应用：2-SAT 问题可线性时间内规约为 SCC；有向图可达性分析；程序流图的循环检测
- 一个有向图的 SCC 数量至少为 1（整个图强连通的情况，如强连通图），最多为 |V|（有向无环图）

## 易错点

1. Kosaraju 第二遍需在"转置图"上进行：若在原图上再做 DFS，得到的不是 SCC。
2. Tarjan 算法中 low 的定义容易混淆：low[u] = min(dfn[u], low[v] for v in children, dfn[v] for edge (u,v) in stack)。其中前向边和横叉边对 low 更新有不同处理。
3. SCC 缩点后编号使用：分量图是一个 DAG，可以对其进行拓扑排序，常用于后续处理。

## 例题

**例题1：** 图 G: 1->2, 2->3, 3->1, 2->4, 4->5, 5->4。找出所有的 SCC。

**解答：** SCC1 = {1,2,3}（三者形成环，相互可达）；SCC2 = {4,5}（4和5形成环相互可达）。缩点后得到 DAG: SCC1 -> SCC2。

**例题2：** 证明分量图是 DAG。

**解答：** 设缩点后分量图有环 C1->C2->...->Ck->C1。由 SCC 缩点定义，C1 中存在某点可到 C2 中某点，C2 中可到 C3 中某点，...，Ck 可到 C1。则 C1,...,Ck 中所有点相互可达，它们应属于同一个 SCC，与极大性矛盾。

## 代码示例

```python
# Kosaraju 算法
def kosaraju(n, edges):
    # 构建邻接表
    graph = [[] for _ in range(n)]
    rev_graph = [[] for _ in range(n)]
    for u, v in edges:
        graph[u].append(v)
        rev_graph[v].append(u)
    
    # 第一遍 DFS：记录完成顺序
    visited = [False] * n
    finish_order = []
    
    def dfs1(u):
        visited[u] = True
        for v in graph[u]:
            if not visited[v]:
                dfs1(v)
        finish_order.append(u)
    
    for i in range(n):
        if not visited[i]:
            dfs1(i)
    
    # 第二遍 DFS：在转置图上按 finish 逆序做 DFS
    visited = [False] * n
    scc = []
    
    def dfs2(u, comp):
        visited[u] = True
        comp.append(u)
        for v in rev_graph[u]:
            if not visited[v]:
                dfs2(v, comp)
    
    for u in reversed(finish_order):
        if not visited[u]:
            comp = []
            dfs2(u, comp)
            scc.append(comp)
    
    return scc

# Tarjan 算法
def tarjan(n, edges):
    graph = [[] for _ in range(n)]
    for u, v in edges:
        graph[u].append(v)
    
    scc = []
    dfn = [-1] * n
    low = [-1] * n
    in_stack = [False] * n
    stack = []
    timer = 0
    
    def dfs(u):
        nonlocal timer
        dfn[u] = low[u] = timer
        timer += 1
        stack.append(u)
        in_stack[u] = True
        
        for v in graph[u]:
            if dfn[v] == -1:
                dfs(v)
                low[u] = min(low[u], low[v])
            elif in_stack[v]:
                low[u] = min(low[u], dfn[v])
        
        if low[u] == dfn[u]:  # 是 SCC 根节点
            comp = []
            while True:
                w = stack.pop()
                in_stack[w] = False
                comp.append(w)
                if w == u:
                    break
            scc.append(comp)
    
    for i in range(n):
        if dfn[i] == -1:
            dfs(i)
    
    return scc

n = 5
edges = [(0,1),(1,2),(2,0),(1,3),(3,4),(4,3)]
print("Kosaraju:", kosaraju(n, edges))
print("Tarjan:", tarjan(n, edges))
```
