## 核心定义

图的着色问题（Graph Coloring）要求为图的每个顶点分配一种颜色，使得相邻顶点颜色不同。m-着色判定问题：给定图 G 和 m 种颜色，判断是否存在合法着色方案。着色优化问题：求最少所需颜色数，即图的色数（Chromatic Number）χ(G)。图的着色问题是 NPC 问题（对 k>=3 的 k-着色判定是 NPC 的；二着色是 P 的，等同于判断图是否是二分图）。求解 m-着色问题的回溯法按顶点顺序依次为每个顶点尝试 m 种颜色，每当赋色时检查与已着色邻居是否冲突，冲突则剪枝回溯。复杂度 O(m^|V|)，但剪枝后实际远小。贪心着色算法（按某种顺序为每个顶点分配第一种可用颜色）虽然不保证最优（χ(G)），但给出一个上界。

## 关键结论

- 四色定理：任意平面图可用 4 种颜色合法着色（1976年 Appel 和 Haken 用计算机辅助证明）
- 图是二分图当且仅当可用 2 种颜色着色，等价于不含奇环
- Welsh-Powell 上界：χ(G) <= max_i min{i-1, deg(v_i)}+1（按度数降序贪心着色）
- 回溯法搜索顺序优化：优先为度数最大的顶点着色可减少分支因子
- 图着色应用：寄存器分配（编译器）、考试排课、频率分配、数独求解

## 易错点

1. 回溯法解图着色时不能只判断当前顶点的颜色约束，需检查所有邻居。
2. 二分图判断与 2-着色的混淆：DFS/BFS 交替着色，若发现冲突说明存在奇环（非二分图）。
3. 贪心着色的结果依赖于顶点处理顺序：不同顺序可能产生不同的颜色数。

## 例题

**例题1：** 图 G=(V,E)：V={1,2,3,4}, E={(1,2),(1,3),(1,4),(2,3),(3,4)}。用 3 种颜色，判断是否可着色并给出方案。

**解答：** 回溯法尝试：1=色1，2尝试色2可行，3尝试色1与1冲突、色2与2冲突、色3可行，4与1,3邻接分别用色2,3，尝试色1可行。方案：1=1, 2=2, 3=3, 4=1。

**例题2：** 证明二分图判断等价于 2-着色问题。

**解答：** 正向：若图是二分图，用 BFS 对每层交替着色（层0色1，层1色2...），同层无边故不冲突。反向：若 2-着色成功，将同色顶点划为同一部，可得二分图划分。

## 代码示例

```python
def graph_coloring_backtrack(graph, m):
    """
    graph: 邻接表
    m: 可用颜色数
    返回：着色方案（None 表示不可着色）
    """
    n = len(graph)
    colors = [-1] * n  # colors[i] = 顶点i的颜色(0~m-1)
    
    def is_safe(v, c):
        for neighbor in graph[v]:
            if colors[neighbor] == c:
                return False
        return True
    
    def backtrack(v):
        if v == n:
            return True  # 全部着色完成
        for c in range(m):
            if is_safe(v, c):
                colors[v] = c
                if backtrack(v + 1):
                    return True
                colors[v] = -1  # 回溯
        return False
    
    if backtrack(0):
        return colors
    return None

def greedy_graph_coloring(graph):
    """贪心着色：按度数降序，O(V^2)"""
    n = len(graph)
    # 按度数降序排列顶点
    order = sorted(range(n), key=lambda v: -len(graph[v]))
    colors = [-1] * n
    
    for v in order:
        # 收集邻居已使用的颜色
        used = set()
        for neighbor in graph[v]:
            if colors[neighbor] != -1:
                used.add(colors[neighbor])
        # 分配最小的未使用颜色（色数 <= deg(v)+1）
        c = 0
        while c in used:
            c += 1
        colors[v] = c
    
    return colors

def is_bipartite(graph):
    """判断是否是二分图（2-着色）O(V+E)"""
    n = len(graph)
    color = [-1] * n
    
    def bfs(start):
        from collections import deque
        q = deque([start])
        color[start] = 0
        while q:
            u = q.popleft()
            for v in graph[u]:
                if color[v] == -1:
                    color[v] = 1 - color[u]
                    q.append(v)
                elif color[v] == color[u]:
                    return False  # 冲突，有奇环
        return True
    
    for i in range(n):
        if color[i] == -1:
            if not bfs(i):
                return False
    return True
```
