## 核心定义

哈密顿回路（Hamiltonian Cycle）是图论中的经典问题：在无向图（或有向图）中寻找一条经过每个顶点恰好一次且最终返回起点的回路。该问题以爱尔兰数学家 William Rowan Hamilton（1857年提出 Icosian 游戏）命名。判定问题"图中是否存在哈密顿回路"是 Karp 的 21 个 NPC 问题之一，证明了哈密顿回路的 NP 完全性。回溯法求解的基本思路：从某一顶点出发，按深度优先搜索依次选择未访问的相邻顶点，当路径长度达到 |V| 时检查是否可回到起点，若能则找到一个解。回溯法的最坏复杂度 O(n!)，通过剪枝（如选择度数最小的顶点优先探索、dead-end 检查）提高实际效率。哈密顿路径是不要求返回起点的版本，同样是 NPC。

## 关键结论

- 哈密顿回路的判定问题是 NPC 的，甚至对 3-正则平面图、网格图等特殊图仍是 NPC
- 狄拉克定理（Dirac's Theorem, 1952）：若 n>=3 且每个顶点度数 >= n/2，则图存在哈密顿回路（充分非必要条件）
- 奥尔定理（Ore's Theorem）：若对任意不相邻的顶点 u,v 有 deg(u)+deg(v) >= n，则存在哈密顿回路
- 哈密顿回路与旅行商问题（TSP）的关系：TSP 是带有权重的哈密顿回路优化版本
- 回溯法的两种剪枝策略：度数剪枝（若某未访问顶点度数 < 2 则不可行）和连通性剪枝

## 易错点

1. 哈密顿回路与欧拉回路混淆：哈密顿回路要求经过每个顶点一次，欧拉回路要求经过每条边一次。两者判定复杂度截然不同（欧拉是有多项式算法的 P 问题）。
2. 狄拉克和奥尔定理只是充分条件：不满足这些条件的图仍然可能有哈密顿回路。
3. 回溯法起始顶点固定可减少 (n-1)!/2 的搜索量，因为回路可以旋转和反转。

## 例题

**例题1：** 图 G=(V,E): V={1,2,3,4,5}, E={(1,2),(2,3),(3,4),(4,5),(5,1),(2,5),(3,5)}。回溯法求哈密顿回路。

**解答：** 固定起始顶点为 1。从 1->2->3->4->5->1 是一条哈密顿回路。若 1->5->3->4->? 需要回溯，最终找到解。验证每条边存在且每个顶点恰好访问一次。

**例题2：** 证明哈密顿回路 <=_p TSP，即哈密顿回路是 TSP 的特殊情况。

**解答：** 给定图 G=(V,E)，构建 TSP 实例：完全图 K_n，原边赋权 1，非原边赋权 2。则 G 有哈密顿回路当且仅当 TSP 存在代价为 n 的回路（每条边权均为 1）。这是多项式时间的规约。

## 代码示例

```python
def hamiltonian_cycle_backtrack(graph):
    """
    graph: 邻接表
    返回：哈密顿回路（顶点序列列表）或 None
    """
    n = len(graph)
    path = [0]  # 固定从顶点0出发
    visited = [False] * n
    visited[0] = True
    
    def is_safe(v, pos):
        """检查是否可将 v 放在 path[pos]"""
        # v 必须是 path[-1] 的邻居
        if v not in graph[path[-1]]:
            return False
        # v 不能已访问过
        if visited[v]:
            return False
        return True
    
    def backtrack(pos):
        if pos == n:
            # 检查是否可以回到起点
            return path[-1] in graph[path[0]]
        
        for v in range(n):
            if is_safe(v, pos):
                path.append(v)
                visited[v] = True
                if backtrack(pos + 1):
                    return True
                visited[v] = False
                path.pop()
        return False
    
    if backtrack(1):
        return path + [0]  # 回到起点
    return None

def has_hamiltonian_dirac(graph):
    """狄拉克定理快速判断（充分条件）"""
    n = len(graph)
    if n < 3:
        return False
    return all(len(graph[v]) >= n // 2 for v in range(n))

# 示例
graph = {
    0: [1, 4],
    1: [0, 2, 4],
    2: [1, 3, 4],
    3: [2, 4],
    4: [0, 1, 2, 3]
}
print(hamiltonian_cycle_backtrack(graph))  # [0, 1, 2, 3, 4, 0]
```
