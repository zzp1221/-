## 核心定义

完全背包问题（Unbounded/Complete Knapsack Problem）是 0-1 背包的变种：每种物品可以取无限次（即没有数量限制），而非仅能取一次。给定 n 种物品，每种有重量 w_i 和价值 v_i，背包容量为 W，目标仍是在总重量不超过 W 的前提下最大化总价值。问题的 DP 转移与 0-1 背包高度相似：dp[i][j] = max(dp[i-1][j], dp[i][j-w_i] + v_i)。关键区别在于第二项使用的是 dp[i][j-w_i]（同行的当前容量减去 w_i）而非 dp[i-1][j-w_i]，这体现了"同种物品可以重复选择"的特性。空间优化为 O(W) 时，内层循环必须正向遍历（从 w_i 到 W），这与 0-1 背包的反向遍历形成鲜明对比。

## 关键结论

- 完全背包的递推式：dp[i][j] = max(dp[i-1][j], dp[i][j-w_i] + v_i)，反映了可多次选取同一物品
- 一维空间优化时正向遍历：for j in range(w_i, W+1)，允许当前物品在同一轮迭代中被多次考虑
- 时间复杂度 O(nW)，空间复杂度 O(W)（一维优化版）
- 完全背包可以用贪心法配合 DP 加速：如果物品按价值密度排序后能在某个阈值后变为完全背包，使用混合策略
- 硬币找零（Coin Change）问题是完全背包的特殊形式：价值 v_i 不重要，求最小硬币数量

## 易错点

1. 正向遍历与反向遍历的混淆：0-1背包反向遍历（防止重复），完全背包正向遍历（允许重复）。这是面试和考试中的高频考点。
2. 最大值类问题和计数类问题的初始值不同：求最大价值时 dp 初始化为 0；求方案数时 dp[0]=1, dp[rest]=0。
3. 循环顺序的影响：对于二维完全背包（总重量和总体积两维约束），三重循环的顺序会影响正确性，通常物品维度放最外层。

## 例题

**例题1：** 3种物品：(2,3), (3,4), (4,5), W=8。求最大价值。

**解答：** 物品种类可重复选取。最优方案：物品(2,3)取 4 次，价值 12，总重 8；或物品(4,5)取 2 次，价值 10。更大价值为 12。注意对比 0-1 背包（每种只能取一次）的最优价值 9。

**例题2：** 硬币找零：coins=[1,2,5], amount=11。求最少硬币数。

**解答：** 完全背包，dp[j] = min(dp[j], dp[j-coin]+1)，初始化 dp[0]=0, 其他为 INF。dp[11]=3（5+5+1）。

## 代码示例

```python
def unbounded_knapsack(W, weights, values):
    """完全背包 O(nW) O(W)"""
    dp = [0] * (W + 1)
    for i in range(len(weights)):
        w, v = weights[i], values[i]
        # 正向遍历允许重复选取
        for j in range(w, W + 1):
            dp[j] = max(dp[j], dp[j - w] + v)
    return dp[W]

def coin_change(coins, amount):
    """硬币找零（最少硬币数）"""
    INF = float('inf')
    dp = [INF] * (amount + 1)
    dp[0] = 0
    for coin in coins:
        for j in range(coin, amount + 1):
            dp[j] = min(dp[j], dp[j - coin] + 1)
    return dp[amount] if dp[amount] != INF else -1

def coin_change_ways(coins, amount):
    """硬币找零方案数"""
    dp = [0] * (amount + 1)
    dp[0] = 1
    for coin in coins:
        for j in range(coin, amount + 1):
            dp[j] += dp[j - coin]
    return dp[amount]

# 比较 0-1 背包与完全背包的关键差异
def compare():
    W, w, v = 8, [2, 3, 4, 5], [3, 4, 5, 6]
    # 0-1: 反向遍历
    dp01 = [0] * (W + 1)
    for i in range(len(w)):
        for j in range(W, w[i] - 1, -1):
            dp01[j] = max(dp01[j], dp01[j - w[i]] + v[i])
    # 完全: 正向遍历
    dp_full = [0] * (W + 1)
    for i in range(len(w)):
        for j in range(w[i], W + 1):
            dp_full[j] = max(dp_full[j], dp_full[j - w[i]] + v[i])
    print(f"0-1背包最大值: {dp01[W]}")    # 9
    print(f"完全背包最大值: {dp_full[W]}")  # 12

compare()
```
