## 核心定义

近似算法（Approximation Algorithm）是处理 NP 难优化问题的实用方法。当精确求解 NP 难问题需要指数时间时，近似算法在多项式时间内找到解，并保证解的质量与最优解在一定的比例之内。近似算法的核心性能指标是近似比（Approximation Ratio）ρ：对于最小化问题，ρ = ALGO/OPT >= 1；对于最大化问题，ρ = OPT/ALGO >= 1。ρ 越小（接近 1）说明近似质量越高。PTAS（Polynomial-Time Approximation Scheme）是一类更强的近似算法，对于任意给定的 ε > 0，可在关于输入规模 n 的多项式时间内得到 (1+ε) 近似解（时间复杂度可以是关于 1/ε 的任意函数）。FPTAS（Fully Polynomial-Time Approximation Scheme）要求时间复杂对于 n 和 1/ε 都是多项式的，是近似算法的最强形式。

## 关键结论

- 贪心算法是近似算法的重要来源：分数背包的最优贪心策略直接给出最优解；0-1 背包的贪心（按密度）给出 2-近似
- 顶点覆盖问题的 2-近似：每次选取一条未覆盖边的两个端点加入覆盖集，近似比为 2
- 旅行商问题（度量 TSP）：若满足三角不等式，Christofides 算法可获得 1.5-近似
- 某些问题具有不可近似性：集合覆盖问题不能有 (1-ε)log n 近似比（除非 P=NP）
- PTAS 示例：任意维度下的欧几里德 TSP 存在 PTAS；FPTAS 示例：0-1 背包存在 O(n^3/ε) 时间的 FPTAS

## 易错点

1. PTAS 的时间复杂度可能为 n^{1/ε} 或 2^{1/ε} n^2，虽对每个固定 ε 是多项式的，但当 ε 很小时实际不可行。
2. 近似比是最坏情况保证，实际运行结果可能远好于近似比，但分析时必须按最坏情况来保证。
3. 非度量 TSP（不满足三角不等式）不可近似：存在 gap 型归约证明若 P≠NP 则没有常数因子近似。

## 例题

**例题1：** 设计顶点覆盖的 2-近似算法并分析近似比。

**解答：** 算法：反复选取图中任意一条边 (u,v)，将 u 和 v 加入覆盖集，删除与 u 或 v 关联的所有边和顶点，直到无边剩余。设最优覆盖大小为 OPT，算法选中的每条边互不相交且至少需要 1 个端点覆盖，故 OPT >= 边数。算法每边取 2 端点，ALGO = 2 * 边数 <= 2 * OPT。近似比为 2。

**例题2：** 0-1 背包的 FPTAS 设计思路。

**解答：** 将物品价值按比例缩放并取整：v'_i = floor(v_i / M)，M = (ε * max(v_i)) / n。缩放后物品价值均为小整数，使用 DP（O(n^2 * max(v'))）求解，再映射回原价值。可证明解的相对误差不超过 ε，时间复杂度 O(n^3/ε)。

## 代码示例

```python
# 顶点覆盖 2-近似算法
def vertex_cover_2approx(edges, n):
    """贪心去边法，每次取一条边的两个端点"""
    remaining = set(range(n))
    cover = set()
    edge_set = set(edges)
    
    while edge_set:
        u, v = edge_set.pop()
        cover.add(u)
        cover.add(v)
        # 删除所有关联边
        to_remove = []
        for e in edge_set:
            if e[0] == u or e[0] == v or e[1] == u or e[1] == v:
                to_remove.append(e)
        for e in to_remove:
            edge_set.discard(e)
    
    return cover

# 集合覆盖的贪心近似 O(log n)
def set_cover_greedy(universe, subsets):
    """贪心：每次选择覆盖最多未覆盖元素的子集"""
    uncovered = set(universe)
    result = []
    
    while uncovered:
        best = max(subsets, key=lambda s: len(set(s) & uncovered))
        result.append(best)
        uncovered -= set(best)
    
    return result
```
