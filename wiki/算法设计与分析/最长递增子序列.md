## 核心定义

最长递增子序列（Longest Increasing Subsequence, LIS）问题要求在给定序列中找到一个严格递增的子序列，使得其长度最长。例如序列 [10, 9, 2, 5, 3, 7, 101, 18] 的 LIS 为 [2, 3, 7, 101] 或 [2, 3, 7, 18]，长度为 4。该问题有两种经典解法：动态规划 O(n^2) 解法，状态定义为 dp[i] = 以第 i 个元素结尾的 LIS 长度，转移方程为 dp[i] = max{ dp[j] + 1 | j < i and arr[j] < arr[i] }；贪心+二分 O(n log n) 解法，维护一个数组 tails，其中 tails[k] 表示长度为 k+1 的递增子序列的最小可能末尾值，遍历过程中使用二分查找更新 tails。LIS 是理解"动态规划与贪心/二分的结合"的重要例题。

## 关键结论

- DP 解法 O(n^2)：dp[i] 需要枚举前面所有小于 arr[i] 的元素，O(n^2) 时间 O(n) 空间
- 耐心排序（Patience Sorting）解法 O(n log n)：维护 tails 数组，对于每个 x 使用二分查找找到 tails 中第一个 >= x 的位置并替换，tails 的长度即为 LIS 长度
- 若要求返回具体的 LIS 序列，需要额外的 prev 或 indices 数组记录前驱信息
- LIS 的变体包括：最长非递减子序列、最长递减子序列、最长波浪子序列（Wiggle Subsequence）
- LIS 与 LCS 的关系：序列 A 和其排序去重后序列 B 的 LCS 即为 A 的 LIS

## 易错点

1. 混淆严格递增和非严格递增的区别：tails 数组中使用的是 lower_bound（第一个 >= x）实现严格递增，若需求非严格递增则用 upper_bound。
2. tails 数组中的元素并非最终 LIS 序列：tails 仅用于维护长度，不代表任何一段真实的递增子序列。若要重构 LIS，需要单独维护前驱信息。
3. 忽略了重复元素的情况：对于含重复元素的序列，"严格递增"会跳过相等元素，而"非严格递增"（最长不下降）的二分判断条件不同。

## 例题

**例题1：** 用 O(n log n) 解法求 [10, 9, 2, 5, 3, 7, 101, 18] 的 LIS。

**解答：** 遍历过程 tails 的变化：x=10->[10]; x=9->[9]; x=2->[2]; x=5->[2,5]; x=3->[2,3]; x=7->[2,3,7]; x=101->[2,3,7,101]; x=18->[2,3,7,18]。最终 tails 长度为 4，即为 LIS 长度。

**例题2：** LIS 问题的"俄罗斯套娃信封"变体：给定 n 个信封 (w,h)，一个信封能套入另一个当 w_i < w_j 且 h_i < h_j，求最多能套几层。

**解答：** 先对宽度升序排序（宽度相同时高度降序以避免同宽度误判），然后对高度序列求 LIS，O(n log n)。

## 代码示例

```python
import bisect

def lis_dp(nums):
    """DP解法 O(n^2)"""
    n = len(nums)
    if n == 0:
        return 0
    dp = [1] * n
    for i in range(n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

def lis_patience(nums):
    """耐心排序法（贪心+二分）O(n log n)"""
    tails = []
    for x in nums:
        # bisect_left: 找第一个 >= x 的位置
        idx = bisect.bisect_left(tails, x)
        if idx == len(tails):
            tails.append(x)
        else:
            tails[idx] = x
    return len(tails)

def lis_reconstruct(nums):
    """重构 LIS 序列 O(n log n)"""
    n = len(nums)
    if n == 0:
        return []
    tails = []        # tails[k] = 长度为k+1的IS的最小末尾值
    tails_idx = []    # tails_idx[k] = 上述末尾值在原数组中的索引
    prev = [-1] * n   # prev[i] = 以nums[i]结尾的LIS中前一个元素的索引
    
    for i, x in enumerate(nums):
        idx = bisect.bisect_left(tails, x)
        if idx == len(tails):
            tails.append(x)
            tails_idx.append(i)
        else:
            tails[idx] = x
            tails_idx[idx] = i
        if idx > 0:
            prev[i] = tails_idx[idx - 1]
    
    # 重构
    result = []
    cur = tails_idx[-1]
    while cur != -1:
        result.append(nums[cur])
        cur = prev[cur]
    return list(reversed(result))

# 示例
nums = [10, 9, 2, 5, 3, 7, 101, 18]
print("LIS长度(DP):", lis_dp(nums))        # 4
print("LIS长度(贪心):", lis_patience(nums))  # 4
print("LIS序列:", lis_reconstruct(nums))    # [2,3,7,101] 或 [2,3,7,18]
```
