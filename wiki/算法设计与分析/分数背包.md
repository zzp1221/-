## 核心定义

分数背包问题（Fractional Knapsack Problem）是 0-1 背包问题的松弛版本：物品可以被分割，即可以取物品的任意比例（而非只能全取或全不取）。给定 n 种物品，每种有重量 w_i 和价值 v_i，背包容量为 W，可以将物品 i 的某个比例 x_i ∈ [0,1] 装入背包，获得价值 x_i * v_i。由于物品可分割，可以利用贪心算法获得最优解：将所有物品按价值密度（v_i / w_i）降序排列，从密度最高的物品开始装入，直到背包装满或物品耗尽；若某物品不能完全装入，则取恰好填满剩余容量的比例。该贪心策略正确性的证明基于"分数背包具有贪心选择性质和最优子结构"这一事实。

## 关键结论

- 贪心策略：按 v/w 降序依次取物品，可以取部分，O(n log n) 时间（排序主导）
- 分数背包的贪心策略与 0-1 背包形成鲜明对比：前者只需贪心，后者需要 DP
- 分数背包可以用带权中位数（Weighted Median）优化至 O(n) 时间
- 分数背包的最优子结构：取出那个局部最优的物品部分后，剩余问题仍是同等规模的分数背包
- 应用场景：投资组合（可部分投资）、资源分配（可切片分配）、液体混合问题

## 易错点

1. 将 0-1 背包用贪心求解：0-1 背包不可分割，贪心不能保证最优。例如 W=10, (w,v):(6,7),(5,5),(5,5)。按密度贪心取 (6,7) 后剩余 4 无法再装任何物品，价值 7；而最优选取后两件得 10。
2. 分数背包中排序后无需考虑是否超过了容量无法取整：可取的物品总是当前密度最高的，只需要取适量填满即可。
3. 浮点精度问题：在实际编码中取物品的分数比例时需注意浮点运算精度。

## 例题

**例题1：** 物品 (10,60), (20,100), (30,120)，W=50。用贪心求最大价值。

**解答：** 计算密度：(10,60) v/w=6, (20,100) v/w=5, (30,120) v/w=4。按密度排序：(10,60), (20,100), (30,120)。取第 1 件：容量剩 40，价值 60。取第 2 件：容量剩 20，价值 160。取第 3 件的 20/30=2/3：价值 160+120*(2/3)=240。总最大价值=240。

**例题2：** 证明分数背包的贪心选择性质。

**解答：** 设物品 1 价值密度最高。在任一最优解中，若物品 1 未被充分利用（未装满容量且未取物品 1），则可取更多物品 1 并减少其他密度较低的物品，总价值不降。用"剪切-粘贴"：将最优解中密度低的物品替换为密度高的物品 1，价值增加或不变，故存在包含物品 1 的最优解。

## 代码示例

```python
def fractional_knapsack(W, weights, values):
    """分数背包贪心 O(n log n)"""
    items = [(v / w, w, v) for w, v in zip(weights, values)]
    items.sort(reverse=True)  # 按价值密度降序
    
    total_value = 0.0
    remaining = W
    
    for density, w, v in items:
        if remaining >= w:
            remaining -= w
            total_value += v
        else:
            total_value += v * (remaining / w)
            break
    
    return total_value

def fractional_knapsack_with_fractions(W, weights, values):
    """返回最大价值和每件物品选取比例"""
    items = [(v / w, w, v, i) for i, (w, v) in enumerate(zip(weights, values))]
    items.sort(reverse=True)
    
    total_value = 0.0
    fractions = [0.0] * len(weights)
    remaining = W
    
    for density, w, v, idx in items:
        if remaining >= w:
            remaining -= w
            total_value += v
            fractions[idx] = 1.0
        else:
            fractions[idx] = remaining / w
            total_value += v * fractions[idx]
            break
    
    return total_value, fractions

# 示例
W = 50
weights = [10, 20, 30]
values = [60, 100, 120]
print(fractional_knapsack(W, weights, values))  # 240.0
total, frac = fractional_knapsack_with_fractions(W, weights, values)
print(f"总价值: {total}, 选取比例: {frac}")
```
