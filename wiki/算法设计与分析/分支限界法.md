## 核心定义

分支限界法（Branch and Bound, B&B）是一种用于求解组合优化问题（特别是 NP 难问题）的系统搜索方法。与回溯法采用 DFS 不同，分支限界法通常使用广度优先搜索（BFS）或最佳优先搜索（Best-First Search），利用优先队列选择最有希望的分支进行扩展。其核心机制是维护全局最优解的值（上界或下界）作为限界标准：对于每个待扩展的活节点，计算其目标函数的一个界限值；若该界限值不可能产生比当前最优更优的解（如最大化问题中上界 <= 当前最优），则将该节点剪枝（杀死），不再扩展其子树。分支限界法用于求解最优化问题（求最大/最小值），而非仅仅求可行解。典型应用包括 0-1 背包、旅行商问题（TSP）、任务分配问题等。

## 关键结论

- 分支限界法包含三个核心组件：分支策略（如何划分子问题）、限界函数（计算上界/下界）、搜索策略（FIFO/LIFO/优先级队列）
- 最大化问题使用上界函数进行剪枝，最小化问题使用下界函数进行剪枝
- 分支限界法比回溯法更适合求最优解：它在搜索过程中持续评估子问题的潜力，优先探索最有希望的方向
- 松弛技术是计算限界函数的关键：通过放松某些约束（如 0-1 背包允许分数选择），快速计算宽松但有意义的上/下界
- 分支限界法常用队列变种：FIFO（先进先出）、LIFO（栈式，类似回溯）、优先队列（以限界值优先）

## 易错点

1. 混淆上界和下界：最大化问题中，上界是乐观估计（实际不可能超过），下界是当前最优解的值。剪枝条件是上界 <= 当前最优。
2. 限界函数过于松散：宽松的限界函数使剪枝效果变差，搜索接近穷举，丧失 B&B 的优势。
3. 未正确初始化和更新全局最优：全局最优的初始值应从一个简单启发式解获得。

## 例题

**例题1：** 用分支限界法解 0-1 背包：W=10, (w,v)=[(4,40),(7,42),(5,25),(3,12)]。画节点搜索过程。

**解答：** 最大化问题，使用"分数背包"的值作为上界。按价值密度排序物品。初始上界：全部按分数装可获 40+42+25+12*1/3 ≈ 107。分支：选/不选物品1。选物品1后剩余容量 6，上界=40+42*6/7≈76。不选物品1后上界=42+25+12≈79。当前最优下界=0。按上界优先扩展，最终找到最优价值=65（物品1+物品2的分数部分？不，应严格0-1），具体分支过程略。

**例题2：** 对比回溯法和分支限界法求解 0-1 背包时的差异。

**解答：** 回溯法采用 DFS，一旦找到合法解就记录（下界），回溯过程中利用限界剪枝。分支限界法使用优先级队列，优先探索上界最高的节点，可以更快找到最优解并利用该解剪枝。分支限界通常比回溯更早剪除大量分支。

## 代码示例

```python
import heapq

def knapsack_branch_bound(W, weights, values):
    """分支限界法解0-1背包（最大化）"""
    n = len(weights)
    # 按价值密度排序
    items = sorted(zip(weights, values, range(n)), 
                   key=lambda x: x[1]/x[0], reverse=True)
    
    best_value = 0
    best_taken = 0
    
    # 分数背包上界计算函数
    def fractional_bound(i, remaining):
        bound = 0.0
        for j in range(i, n):
            w, v, _ = items[j]
            if remaining >= w:
                remaining -= w
                bound += v
            else:
                bound += v * (remaining / w)
                break
        return bound
    
    # 优先队列: (-upper_bound, level, value, weight, taken_mask)
    heap = []
    bound = fractional_bound(0, W)
    heapq.heappush(heap, (-bound, 0, 0, 0, 0))
    
    while heap:
        neg_bound, level, value, weight, taken = heapq.heappop(heap)
        upper = -neg_bound
        
        if upper <= best_value:  # 限界剪枝
            continue
        if level == n:
            if value > best_value:
                best_value = value
                best_taken = taken
            continue
        
        w, v, idx = items[level]
        
        # 分支1：不选
        bound = value + fractional_bound(level + 1, W - weight)
        if bound > best_value:
            heapq.heappush(heap, (-bound, level + 1, value, weight, taken))
        
        # 分支2：选（若可行）
        if weight + w <= W:
            bound = value + v + fractional_bound(level + 1, W - weight - w)
            if bound > best_value:
                heapq.heappush(heap, (-bound, level + 1, 
                             value + v, weight + w, taken | (1 << idx)))
    
    return best_value

# 示例
W = 10
weights = [4, 7, 5, 3]
values = [40, 42, 25, 12]
print("最优价值:", knapsack_branch_bound(W, weights, values))  # 65
```
