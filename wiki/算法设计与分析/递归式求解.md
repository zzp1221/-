## 核心定义

递归式（Recurrence）是描述分治算法时间复杂度的一类方程，通常形式为 T(n) = aT(n/b) + f(n)，其中 a >= 1 表示子问题的个数，b > 1 表示子问题规模缩减的比例，f(n) 表示分解和合并子问题所花费的时间。求解递归式有三种经典方法：代入法（Substitution Method），也称为"猜测-验证"法，先猜测解的形式，再用数学归纳法证明其正确性，常用于验证解的有效性；递归树法（Recursion Tree Method），将递归展开成树状结构，计算每一层的工作量和总层数，直观地求和得到最终复杂度，是最直观的方法；主定理（Master Theorem），针对特定形式的递归式提供"查表式"的快速求解公式，是最实用的工具。

## 关键结论

- 代入法的核心技巧在于归纳假设的选择，常需要加减低阶项来"匹配"归纳步骤
- 递归树法遵循三步：画出递归树、计算每层工作量（包括子问题计算和合并开销）、累加所有层的工作量得到总复杂度
- 主定理 Case 1：若 f(n) = O(n^{log_b a - ε})，则 T(n) = Θ(n^{log_b a})
- 主定理 Case 2：若 f(n) = Θ(n^{log_b a} * log^k n)，则 T(n) = Θ(n^{log_b a} * log^{k+1} n)
- 主定理 Case 3：若 f(n) = Ω(n^{log_b a + ε}) 且满足正则条件 af(n/b) <= cf(n) (c<1)，则 T(n) = Θ(f(n))

## 易错点

1. 主定理的三个Case不是穷尽的：当 f(n) 落在 n^{log_b a} 和其他函数之间的"缝隙"中时（如 f(n) = n^{log_b a} / log n），主定理不适用。此时需要用递归树或代入法。
2. Case 3的正则条件常被遗忘：只满足 f(n) = Ω(n^{log_b a + ε}) 还不够，必须验证存在 c < 1 使得 af(n/b) <= cf(n) 成立。典型反例是 f(n) = n + n*sin(n)，缺乏正则性。
3. 代入法猜错边界导致证明卡壳：对 O 界的归纳证明需要"收紧"常数。例如证明 T(n) = O(n) 时如果归纳假设 T(k) <= ck，代入得 T(n) <= cn + 1，需要将假设改为 T(k) <= ck - d 来吸收常数项。

## 例题

**例题1：** 使用主定理求解 T(n) = 4T(n/2) + n。

**解答：** a=4, b=2, f(n)=n。log_b a = log_2 4 = 2。比较 f(n)=n 与 n^{log_b a}=n^2：由于 n = O(n^{2-ε})，取 ε=0.5，则 n = O(n^{1.5})。属于 Case 1。因此 T(n) = Θ(n^2)。

**例题2：** 使用主定理求解 T(n) = 2T(n/2) + n log n。

**解答：** a=2, b=2, f(n)=n log n。log_b a = log_2 2 = 1。f(n) = n log n = Θ(n^1 * log^1 n)，属于 Case 2（k=1）。因此 T(n) = Θ(n log^2 n)。

# 递归式求解可视化：递归树法
def recurrence_tree_analysis(n, a, b):
    """
    演示递归树分析 T(n) = a T(n/b) + f(n)
    返回：树的总层数、每层节点数、每层工作量
    """
    layers = []
    level = 0
    current_n = n
    nodes = 1  # 当前层节点数
    
    print(f"递归式: T(n) = {a} T(n/{b}) + f(n)")
    print(f"log_{b}({a}) = {__import__('math').log(a, b):.2f}")
    print()
    
    while current_n >= 1:
        # 本层每个节点的规模为 current_n
        # 本层共有 nodes 个节点
        # 本层总工作量 = nodes * f(current_n)（以f(n)=n为例）
        workload = nodes * current_n
        layers.append((level, nodes, current_n, workload))
        print(f"第{level}层: {nodes}个节点, 每节点规模={current_n:.0f}, 本层工作量={workload}")
        
        level += 1
        nodes *= a
        current_n /= b
    
    total = sum(l[3] for l in layers)
    print(f"\n总工作量 = {total}")
    return layers

# 示例: T(n) = 3T(n/4) + n  -> log_4(3) ≈ 0.79
recurrence_tree_analysis(1024, 3, 4)
