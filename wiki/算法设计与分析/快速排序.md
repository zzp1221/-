## 核心定义

快速排序（Quick Sort）是由 Tony Hoare 于1961年提出的基于分治法的排序算法。其核心操作是划分（Partition）：从数组中选择一个元素作为主元（Pivot），将数组重新排列，使得所有小于主元的元素放在主元左侧，所有大于主元的元素放在主元右侧，主元自身处于最终排序位置。然后递归地对左右两个子数组重复这一过程。快速排序的平均时间复杂度为 O(n log n)，最坏情况为 O(n^2)（当每次划分极不平衡时），但由于其常数因子小、具有缓存友好的就地排序特性，实践中通常优于归并排序和堆排序。快速排序是不稳定排序，因为划分过程中的交换操作可能改变相等元素的相对顺序。

## 关键结论

- 快速排序的平均时间复杂度为 O(n log n)，且隐含常数很小（约1.39 n log n次比较），使其在实践中很快
- 最坏情况 O(n^2) 发生在每次选到最小或最大元素作为主元时，此时递归树退化为线性链
- 随机化快速排序通过对主元引入随机性，使最坏情况的概率降至指数级小，期望时间复杂度为 O(n log n)
- 三数取中（Median-of-Three）是工业界常用的主元选择策略，取首、中、尾三元素的中值作为主元
- 快速排序是原地排序，空间复杂度主要来自递归调用栈：平均 O(log n)，最坏 O(n)

## 易错点

1. 划分算法的边界条件容易出错：Lomuto划分方案虽然简单但效率略低，Hoare划分方案效率更高但对指针移动的理解要求更精确。一步之差可能导致死循环或错误划分。
2. 忽略递归深度优化：在生产实践中，通常会先递归较小的一侧，对较大一侧进行尾递归或迭代处理，将栈深度控制在 O(log n)。
3. 对有大量重复元素的数组直接使用标准快速排序：当所有元素相同时，分区极不平衡。此时应使用三路快速排序（3-Way Partitioning），将数组分为 < pivot、= pivot、> pivot 三部分。

## 例题

**例题1：** 分析快速排序的最坏情况并举例说明。

**解答：** 当每次选主元都是数组的最值（如已排序数组且固定选末尾元素），划分后一边为空一边为 n-1，递归式 T(n) = T(n-1) + T(0) + O(n) = T(n-1) + O(n)，展开得 T(n) = O(n^2)。例如对 [1,2,3,4,5] 选末尾 5 为主元，划分后左侧 [1,2,3,4] 右侧 []，极不平衡。

**例题2：** 证明随机化快速排序的期望时间复杂度为 O(n log n)。

**解答：** 随机选择主元，任意两个元素被比较的概率为 2/(j-i+1)（它们在最终排序中相邻的概率）。期望比较次数 = sum_{i<j} 2/(j-i+1) = O(n log n)。也可以用递归式 T(n) = 1/n * sum_{k=1}^n (T(k-1) + T(n-k)) + O(n)，解得 T(n) = O(n log n)。

## 代码示例

```python
import random

def quicksort(arr, lo, hi):
    """Lomuto划分方案的快速排序"""
    if lo >= hi:
        return
    pivot_idx = partition(arr, lo, hi)
    quicksort(arr, lo, pivot_idx - 1)
    quicksort(arr, pivot_idx + 1, hi)

def partition(arr, lo, hi):
    """Lomuto 划分: 选最后一个元素为主元"""
    pivot = arr[hi]
    i = lo - 1  # i指向小于pivot的最后一个元素
    for j in range(lo, hi):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i + 1], arr[hi] = arr[hi], arr[i + 1]
    return i + 1

def randomized_quicksort(arr, lo, hi):
    """随机化快速排序: 随机选主元并交换到末尾"""
    if lo >= hi:
        return
    rand_idx = random.randint(lo, hi)
    arr[rand_idx], arr[hi] = arr[hi], arr[rand_idx]
    pivot_idx = partition(arr, lo, hi)
    randomized_quicksort(arr, lo, pivot_idx - 1)
    randomized_quicksort(arr, pivot_idx + 1, hi)

def quicksort_3way(arr, lo, hi):
    """三路划分快速排序: 适合大量重复元素"""
    if lo >= hi:
        return
    lt, gt = lo, hi
    pivot = arr[lo]
    i = lo
    while i <= gt:
        if arr[i] < pivot:
            arr[lt], arr[i] = arr[i], arr[lt]
            lt += 1; i += 1
        elif arr[i] > pivot:
            arr[gt], arr[i] = arr[i], arr[gt]
            gt -= 1
        else:
            i += 1
    quicksort_3way(arr, lo, lt - 1)
    quicksort_3way(arr, gt + 1, hi)

# 非递归版本（用显式栈）
def quicksort_iterative(arr):
    stack = [(0, len(arr) - 1)]
    while stack:
        lo, hi = stack.pop()
        if lo < hi:
            p = partition(arr, lo, hi)
            # 先压入较大的区间（保证栈深度为 O(log n)）
            if p - lo > hi - p:
                stack.append((lo, p - 1))
                stack.append((p + 1, hi))
            else:
                stack.append((p + 1, hi))
                stack.append((lo, p - 1))
```
