## 核心定义

0-1背包问题（0-1 Knapsack Problem）是组合优化中的经典问题：给定 n 件物品，每件物品 i 有重量 w_i 和价值 v_i，以及一个容量为 W 的背包，每件物品要么完整装入背包（取1件），要么不装入（取0件），不能分割。目标是在总重量不超过 W 的前提下，使装入物品的总价值最大化。该问题是典型的 NP 难问题，但可使用动态规划在伪多项式时间 O(nW) 内求解。0-1背包问题的 DP 状态定义为 dp[i][j] = 考虑前 i 件物品且背包容量为 j 时的最大价值，转移方程为：dp[i][j] = max(dp[i-1][j], dp[i-1][j-w_i] + v_i)，即要么不放第 i 件，要么放（需剩余容量充足）。该问题体现了动态规划中"选或不选"的经典决策模式。

## 关键结论

- 0-1背包是"伪多项式"算法：O(nW) 时间，W 以数字形式而非比特长度衡量，故是 NP 难问题在一定范围内的可行解法
- 空间优化：使用一维数组可将空间从 O(nW) 降至 O(W)，但内层循环必须反向遍历（从 W 到 w_i）以避免同一物品被多次使用
- 完全背包（物品可取无限件）与 0-1 背包的唯一区别是内层循环正向遍历
- 0-1背包是最简单的背包模型，其变种包括多重背包、分组背包、有依赖背包等
- 当 W 很大（如 10^9）而 n 较小时，DP 不可行，需使用 meet-in-the-middle 或分支限界法

## 易错点

1. 空间优化时内层循环方向错误：正向遍历会让每件物品被多次使用（变成完全背包），必须从 W 向 w_i 递减遍历。
2. dp 初始化为 0 导致误将不装满的情况视为合法：若要求背包恰好装满，dp[0] 初始化为 0，其余 dp[j] 初始化为负无穷。
3. 混淆 0-1 背包和分数背包：分数背包可用贪心算法 O(n log n)，而 0-1 背包没有贪心解法。

## 例题

**例题1：** 4件物品：w=[2,3,4,5], v=[3,4,5,6], W=8。求最大价值。

**解答：** 构建 dp 表：dp[4][8] 经过递推。选物品 1(2,3) 和 物品 4(5,6) 可得价值 9；或选物品 2(3,4) 和物品 3(4,5) 也得价值 9。最大价值为 9，对应重量 2+5=7 或 3+4=7。

**例题2：** 为什么 0-1 背包不能用贪心算法？

**解答：** 反例：W=10, 物品1(6,7), 物品2(5,5), 物品3(5,5)。按价值密度（v/w）排序：1>2=3。贪心选物品1后剩余容量4无法装入2或3，总价值7。但最优解为选物品2+3，总价值10。不满足贪心选择性质。

## 代码示例

```python
def knapsack_01(W, weights, values):
    """0-1背包 DP O(nW) 时间, O(W) 空间"""
    n = len(weights)
    dp = [0] * (W + 1)
    
    for i in range(n):
        w, v = weights[i], values[i]
        # 必须反向遍历，防止同一物品被多次使用
        for j in range(W, w - 1, -1):
            dp[j] = max(dp[j], dp[j - w] + v)
    
    return dp[W]

def knapsack_01_with_items(W, weights, values):
    """返回最大价值和选中的物品"""
    n = len(weights)
    dp = [[0] * (W + 1) for _ in range(n + 1)]
    
    for i in range(1, n + 1):
        w, v = weights[i - 1], values[i - 1]
        for j in range(W + 1):
            if j >= w:
                dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - w] + v)
            else:
                dp[i][j] = dp[i - 1][j]
    
    # 回溯选中的物品
    selected = []
    j = W
    for i in range(n, 0, -1):
        if dp[i][j] != dp[i - 1][j]:
            selected.append(i - 1)
            j -= weights[i - 1]
    
    return dp[n][W], selected

# 示例
W = 8
weights = [2, 3, 4, 5]
values = [3, 4, 5, 6]
max_val, items = knapsack_01_with_items(W, weights, values)
print(f"最大价值: {max_val}")    # 9
print(f"选中物品索引: {items}")  # [3, 0] 或 [2, 1]
```
