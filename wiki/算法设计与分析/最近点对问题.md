## 核心定义

最近点对问题（Closest Pair of Points）要求在平面上给定的 n 个点中，找到距离最小的一对点（距离通常指欧几里德距离）。暴力解法需要检查所有 O(n^2) 个点对的组合。分治解法将时间复杂度降至 O(n log n)：首先按 x 坐标排序所有点，从中间等分为左右两个点集，递归求出左半部分和右半部分的最近点对距离 δ；然后在中间宽度为 2δ 的带状区域内检查跨越中线的点对。关键是证明在宽 2δ、高 δ 的矩形内最多只能容纳有限个（通常 6-8 个）点，因此带状区的检查只需 O(n) 时间。递归式 T(n) = 2T(n/2) + O(n) = O(n log n)。

## 关键结论

- 分治法的时间复杂度 O(n log n) 由排序开销主导，可在预排序后降至 O(n log n)
- 带状区分析是算法的核心：在 d x 2d 的矩形内最多容纳 8 个点（更精细分析为 6 个），保证了线性时间的检查
- 当点集退化（多点共线）时，算法仍正确且高效
- 高维扩展：k 维最近点对问题可用类似思路但复杂度更高（常使用 KD-Tree 等数据结构）
- 该问题有广泛的应用场景：碰撞检测、聚类分析、空中交通管制、VLSI 布局优化等

## 易错点

1. 带状区的边界不是固定的 x 坐标区间：带状区是以中线 x 坐标为基准，宽度为 2δ 的竖条。只检查 x 坐标在线内的点，不包括线外。
2. 内部循环退出条件不正确：对带状区内的每个点，只需检查其后续有限个点（y 坐标差超过 δ 即可跳出），不能无限制扫描。
3. 忘记更新全局最短距离 δ：在计算跨越中线的点对后，需用更小的距离更新 δ，影响后续带状区的判断。

## 例题

**例题1：** 给定 6 个点 A(0,0), B(2,2), C(3,1), D(4,3), E(1,4), F(5,0)，用分治法求最近点对。

**解答：** 排序 x 坐标：A(0,0), E(1,4), B(2,2), C(3,1), D(4,3), F(5,0)。中点 x=2.5。左半 A,E,B δ_L = min_dist(A,B)=√8。右半 C,D,F δ_R = min_dist(C,F)=√5。δ = min(√8,√5)=√5。带状区内点：B,C,D,E。检查跨中线点对，发现 C(3,1) 和 B(2,2) 距离 √2 < √5，更新 δ=√2。最终最近点对为 B 和 C。

**例题2：** 证明在 d x 2d 矩形内最多容纳 8 个点。

**解答：** 将矩形的短边 d 等分为 2 份，长边 2d 等分为 4 份，形成 8 个 d/2 x d/2 的小正方形。每个小正方形的对角线长度为 d/√2 < d，故每个小正方形内最多一个点（否则两个点的距离 < d，矛盾）。因此矩形内最多 8 个点。

## 代码示例

```python
import math

def closest_pair(points):
    """分治法求最近点对 O(n log n)"""
    px = sorted(points, key=lambda p: p[0])  # 按x排序
    py = sorted(points, key=lambda p: p[1])  # 按y排序
    return _closest_pair_rec(px, py)

def _closest_pair_rec(px, py):
    n = len(px)
    if n <= 3:
        return _brute_force(px)
    
    mid = n // 2
    mid_x = px[mid][0]
    
    # 分别构建左右子集的py序列
    pyl = [p for p in py if p[0] <= mid_x]
    pyr = [p for p in py if p[0] > mid_x]
    
    # 递归求解
    dl, pl, ql = _closest_pair_rec(px[:mid], pyl)
    dr, pr, qr = _closest_pair_rec(px[mid:], pyr)
    
    # 取左右最小值
    d = dl
    best_pair = (pl, ql)
    if dr < d:
        d = dr
        best_pair = (pr, qr)
    
    # 构建带状区
    strip = [p for p in py if abs(p[0] - mid_x) < d]
    
    # 带状区内检查（关键优化：只检查有限个邻居）
    for i in range(len(strip)):
        j = i + 1
        while j < len(strip) and strip[j][1] - strip[i][1] < d:
            dist = _dist(strip[i], strip[j])
            if dist < d:
                d = dist
                best_pair = (strip[i], strip[j])
            j += 1
    # 题目需返回距离最小的两点
    return d, best_pair[0], best_pair[1]

def _dist(p1, p2):
    return math.sqrt((p1[0]-p2[0])**2 + (p1[1]-p2[1])**2)

def _brute_force(points):
    n = len(points)
    best = float('inf')
    best_pair = None
    for i in range(n):
        for j in range(i+1, n):
            d = _dist(points[i], points[j])
            if d < best:
                best = d
                best_pair = (points[i], points[j])
    return best, best_pair[0], best_pair[1]
```
