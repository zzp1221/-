## 核心定义

组合数学（Combinatorics）是研究离散对象的计数、排列和结构的数学分支，是计算机科学算法分析和大规模系统设计的数学基础。核心概念包括：(1) **计数原理**：加法原理（互斥事件的计数相加）和乘法原理（独立步骤的计数相乘）；(2) **排列**（Permutation）：从 n 个不同元素中取 r 个元素按顺序排列，记 P(n,r) = n!/(n−r)!；(3) **组合**（Combination）：从 n 个不同元素中取 r 个元素不考虑顺序，记 C(n,r) = n!/(r!(n−r)!) 即二项式系数；(4) **容斥原理**（Inclusion-Exclusion Principle）：|A₁∪A₂∪...∪Aₙ| = Σ|Aᵢ| − Σ|Aᵢ∩Aⱼ| + Σ|Aᵢ∩Aⱼ∩Aₖ| − ... + (−1)ⁿ⁺¹|A₁∩...∩Aₙ|，用于计算至少满足一个条件的元素个数；(5) **鸽巢原理**（Pigeonhole Principle）：若 n+1 个物体放入 n 个盒子，则至少有一个盒子含至少 2 个物体——推广形式：若 N 个物体放入 k 个盒子，则至少有一个盒子含至少 ⌈N/k⌉ 个物体；(6) **生成函数**（Generating Function）：将序列编码为形式幂级数 a₀ + a₁x + a₂x² + ...，通过函数运算求解递推关系和计数问题；(7) **递推关系**（Recurrence Relation）：用前面若干项定义后面项的等式，如 Fibonacci 数列 Fₙ = Fₙ₋₁ + Fₙ₋₂。

## 关键结论

- 二项式定理：(x+y)ⁿ = Σ_{k=0}ⁿ C(n,k) xⁿ⁻ᵏ yᵏ；二项式系数满足 Pascal 恒等式 C(n,k) = C(n−1,k−1) + C(n−1,k)
- 可重复排列：从 n 个元素中取 r 个排列（可重复），共 nʳ 种；可重复组合：C(n+r−1, r) 种
- 容斥原理可计算错排数 Dₙ = n!·Σ_{k=0}ⁿ (−1)ᵏ/k!，错排问题有近似 Dₙ ≈ n!/e
- Catalan数 Cₙ = C(2n,n)/(n+1) 计数二叉树、合法括号序列和三角剖分等组合结构
- 线性齐次递推关系 aₙ = c₁aₙ₋₁ + ... + cₖaₙ₋ₖ 可通过特征方程 rᵏ − c₁rᵏ⁻¹ − ... − cₖ = 0 求解
- 生成函数将递推关系转化为代数方程，是解决组合计数问题的系统化方法

## 易错点

1. 排列组合的选择条件混淆：排列有顺序（P(n,r)），组合无顺序（C(n,r)），同题中需仔细判别
2. 容斥原理符号交错的记忆：第一项 +，第二项 −（减去重复计算的两两交集），第三项 +（加回被过度减去的三三交集）...
3. 鸽巢原理中的"至少"含义：保证了存在性但未给出具体哪个盒子——这是一种非构造性的存在性证明工具

## 例题

**例题1**：5 本不同的书分给 3 个学生，每个学生至少得 1 本，多少种分法？

**解答**：总分配数（每本书可以给任意学生）= 3^5。需减去有人未得书的情况。用容斥原理：设 Aᵢ 为第 i 个学生未得书的情况。|Aᵢ| = 2^5。|Aᵢ∩Aⱼ| = 1^5 = 1。|A₁∩A₂∩A₃| = 0。所求 = 3^5 − C(3,1)×2^5 + C(3,2)×1^5 − C(3,3)×0 = 243 − 96 + 3 − 0 = 150。

**例题2**：求解递推关系 aₙ = 5aₙ₋₁ − 6aₙ₋₂，a₀ = 1, a₁ = 2。

**解答**：特征方程 r² − 5r + 6 = 0，两根 r₁=2, r₂=3。通解 aₙ = α·2ⁿ + β·3ⁿ。代入初始条件：a₀=α+β=1, a₁=2α+3β=2。解得 α=1, β=0。故 aₙ = 2ⁿ。

## 代码示例

```python
from math import comb, factorial

def P(n, r):
    """排列数"""
    return factorial(n) // factorial(n - r)

def C(n, r):
    """组合数"""
    return comb(n, r)

def inclusion_exclusion(sets):
    """容斥原理求并集的元素数"""
    from itertools import combinations
    n = len(sets)
    total = 0
    for k in range(1, n + 1):
        sign = 1 if k % 2 == 1 else -1
        for combo in combinations(range(n), k):
            intersection = sets[combo[0]].copy()
            for i in combo[1:]:
                intersection &= sets[i]
            total += sign * len(intersection)
    return total

# Fibonacci递推关系
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

print(f"F(10) = {fibonacci(10)}")  # 55
```
