## 核心定义

集合（Set）是离散数学乃至整个现代数学的基石概念，指具有某种特定性质的、确定的、彼此可以区分的对象的汇集。集合中的每个对象称为该集合的**元素**（Element）。若对象 a 属于集合 A，记作 a ∈ A；否则记作 a ∉ A。集合由其中的元素唯一确定——这是**外延公理**（Axiom of Extensionality）：两个集合相等的充要条件是它们具有完全相同的元素，与元素的排列顺序和重复次数无关（{1, 2, 3} = {3, 1, 2} = {1, 1, 2, 3}）。集合的表示方法有三种：(1) **列举法**（Roster Method）：直接列出所有元素，如 A = {1, 2, 3, 4}；(2) **描述法**（Set Builder Notation）：用谓词刻画元素性质，如 B = {x | x 是小于 5 的正整数}；(3) **文氏图**（Venn Diagram）：用平面图形直观表示。空集（Empty Set, ∅）是不含任何元素的集合，是任何集合的子集。**万有集**（Universal Set, U）是包含所讨论的全部元素的集合，依赖于具体语境。

## 关键结论

- 集合中元素的"无序性"和"互异性"是两个基本特征：{a, b} = {b, a}，且同一元素不会因多次出现而改变集合
- 空集 ∅ 是唯一的：任何两个空集拥有完全相同的元素（都没有元素），由外延公理它们相等
- 属于关系 ∈ 与子集关系 ⊆ 是根本不同的：a ∈ A 是元素与集合的关系，A ⊆ B 是集合与集合的关系
- 罗素悖论（Russell's Paradox）表明不是任何条件都能定义一个集合——因此现代集合论使用公理化方法（ZFC公理系统）
- 集合论的基本运算包括：并 ∪、交 ∩、差 −、补 ᶜ、对称差 ⊕

## 易错点

1. 混淆 a 与 {a}：a 是元素，{a} 是以 a 为唯一元素的单元素集。a ∈ {a}，但一般 a ≠ {a}（除非在非良基集合论中）
2. 混淆空集 ∅ 与包含空集的集合 {∅}：∅ 有 0 个元素，{∅} 有 1 个元素（即 ∅）。|∅| = 0, |{∅}| = 1
3. 忘记忆集 ∅ 是任何集合的子集：对任何集合 A，∅ ⊆ A 总是成立（空真，vacuous truth）

## 例题

**例题1**：列出集合 A = {x | x ∈ ℕ, x < 10, x 是质数} 的所有元素，并用列举法表示。

**解答**：小于 10 的质数有 2, 3, 5, 7。故 A = {2, 3, 5, 7}，|A| = 4。

**例题2**：判断命题真假：若 A = {∅, {∅}}，则 (a) ∅ ∈ A, (b) ∅ ⊆ A, (c) {∅} ∈ A, (d) {∅} ⊆ A。

**解答**：
(a) 真：∅ 是 A 的第一个元素。
(b) 真：空集是任何集合的子集。
(c) 真：{∅} 是 A 的第二个元素。
(d) 真：{∅} 的唯一元素是 ∅，而 ∅ ∈ A，故 {∅} ⊆ A。

## 代码示例

```python
# Python中的集合操作
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}
U = set(range(1, 11))  # 万有集

# 基本运算
union = A | B              # 并集: {1,2,3,4,5,6}
intersection = A & B       # 交集: {3,4}
difference = A - B         # 差集: {1,2}
complement = U - A         # 补集: {5,6,7,8,9,10}
sym_diff = A ^ B           # 对称差: {1,2,5,6}

# 验证集合的无序性和互异性
C = {3, 2, 1, 4, 4, 1}
print(f"A == C: {A == C}")  # True

# 子集判断
print(f"∅ ⊆ A: {set().issubset(A)}")  # True
```
