## 核心定义

**陪集**（Coset）是群论中对群进行"划分"的基本工具。设 H ≤ G，对任意 a ∈ G，集合 aH = {a*h | h ∈ H} 称为 H 的**左陪集**（Left Coset），Ha = {h*a | h ∈ H} 称为 H 的**右陪集**（Right Coset）。左陪集构成 G 的一个划分（任意两个左陪集要么相等要么不相交），且所有左陪集具有相同的基数 |H|，由此导出 **Lagrange 定理**：|H| 整除 |G|，指数 [G:H] = |G|/|H|。

**正规子群**（Normal Subgroup）是满足 ∀g∈G, gHg⁻¹ = H（等价于 gH = Hg）的子群，记作 H ⊴ G。正规子群的左陪集和右陪集完全一致，这是构造商群的前提。

**商群**（Quotient Group / Factor Group）G/H 定义为 H 的所有陪集的集合 {gH | g ∈ G}，配合运算 (aH)(bH) = (ab)H。当 H ⊴ G 时，此运算良定义。商群的阶 |G/H| = [G:H] = |G|/|H|。商群是群同态第一定理（Fundamental Homomorphism Theorem）的核心：若 φ: G → K 是群同态，则 ker φ ⊴ G 且 G/ker φ ≅ im φ。

## 关键结论

- 左陪集划分 G 为等势的不相交子集，每个子集与 H 等势
- Lagrange 定理：|G| = |H| × [G:H]，子群的阶必整除群的阶
- 子群 H 是正规的当且仅当 ∀g∈G, gHg⁻¹ ⊆ H
- 阿贝尔群的每个子群都是正规子群（因为 gH = Hg 自动满足）
- 商群构造类似"折叠"或"取模"操作：ℤ/nℤ ≅ ℤₙ
- 同态基本定理：群的每个同态像同构于一个商群

## 易错点

1. 混淆左陪集与右陪集：一般而言 aH ≠ Ha，仅在正规子群时两者相等
2. 误认为商群 G/H 的元素是 G 的元素：G/H 的元素是陪集（H 的陪集），它们是 G 的子集而不是 G 的单个元素
3. Lagrange 定理只说 |H| 整除 |G|，但逆命题不成立：n 整除 |G| 不一定存在 n 阶子群（如 A₄ 有 12 个元素但没有 6 阶子群）

## 例题

**例题1**：G = (ℤ, +)，H = 3ℤ = {..., -6, -3, 0, 3, 6, ...}。列出 H 的所有陪集并构造商群 ℤ/3ℤ。

**解答**：H 的陪集（因为 ℤ 是阿贝尔群，左右陪集相同）：0+H = {...,-6,-3,0,3,6,...}，1+H = {...,-5,-2,1,4,7,...}，2+H = {...,-4,-1,2,5,8,...}。商群 ℤ/3ℤ = {0+H, 1+H, 2+H} ≅ ℤ₃。

**例题2**：证明指数为 2 的子群必为正规子群。

**解答**：设 H ≤ G 且 [G:H] = 2。对任意 g ∈ G，若 g ∈ H，则 gH = H = Hg。若 g ∉ H，由于恰有两个左陪集（H 和 gH），也恰有两个右陪集（H 和 Hg）。又因 H 是公有的，故 gH = G\H = Hg。所以 gH = Hg 对所有 g 成立，H ⊴ G。

## 代码示例

```python
def left_cosets(G_elements, G_op, H_subset):
    """计算子群H的所有左陪集"""
    cosets = []
    remaining = set(G_elements)
    while remaining:
        g = remaining.pop()
        coset = {G_op(g, h) for h in H_subset}
        cosets.append(coset)
        remaining -= coset
    return cosets

# ℤ₁₂ 中 H={0,4,8}
G = list(range(12))
H = {0, 4, 8}
op = lambda a, b: (a + b) % 12
cosets = left_cosets(G, op, H)
print(f"H的陪集(商群元素): {[sorted(c) for c in cosets]}")
# [{0,4,8}, {1,5,9}, {2,6,10}, {3,7,11}]
print(f"Lagrange: |G|={len(G)}, |H|={len(H)}, 指数={len(G)//len(H)}")
```
