## 核心定义

复合函数（Function Composition）是将两个函数串联执行而产生的新的函数。给定函数 f: A → B 和 g: B → C，二者的复合记作 g∘f: A → C，定义为 (g∘f)(a) = g(f(a))——即先用 f 作用于 a，再用 g 作用于结果。复合的"顺序"需要注意：g∘f 读作"g 复合 f"或"先 f 后 g"，函数符号的书写顺序是反向的。复合函数定义的前提条件是 f 的陪域等于 g 的定义域（或至少 f 的像 ⊆ g 的定义域）。函数复合满足**结合律**：(h∘g)∘f = h∘(g∘f)（只要复合的定义合法），但不满足交换律（g∘f ≠ f∘g 一般情形成立）。**恒等函数** id_A: A → A, id_A(a) = a，是函数复合的单位元：f∘id_A = f, id_B∘f = f。一个函数 f: A → B 的**逆函数** f⁻¹: B → A 满足 f⁻¹∘f = id_A 和 f∘f⁻¹ = id_B——逆函数存在的充要条件是 f 为双射。

## 关键结论

- 复合函数保持单射性、满射性和双射性：若 f 和 g 都是单（满/双）射，则 g∘f 也是单（满/双）射
- 逆命题成立但有限制：若 g∘f 是单射，则 f 必为单射（g 不一定）；若 g∘f 是满射，则 g 必为满射（f 不一定）
- 双射的逆是唯一的：(f⁻¹)⁻¹ = f
- 复合函数的逆：(g∘f)⁻¹ = f⁻¹∘g⁻¹（注意顺序反转，类似于矩阵乘法中 (AB)⁻¹ = B⁻¹A⁻¹）
- 函数复合构成一个半群（在 A → A 的自映射上）或更丰富的代数结构（双射全体构成群，称为对称群）

## 易错点

1. 混淆复合顺序：g∘f 是先 f 后 g，但符号书写上 g 写在 f 前面——这是函数复合中最常见的困惑来源
2. 误以为 f∘g = g∘f：一般情形成立，例如 f(x) = x+1, g(x) = x²，(g∘f)(x) = (x+1)² 而 (f∘g)(x) = x²+1，完全不同的函数
3. 认为 g∘f 合法则 f∘g 也合法：前者要求 ran(f) ⊆ dom(g)，后者要求 ran(g) ⊆ dom(f)，两者独立

## 例题

**例题1**：设 f(x) = x+2, g(x) = 3x, h(x) = x²。求 (h∘g∘f)(1) 并验证结合律。

**解答**：
(h∘g∘f)(1) = h(g(f(1))) = h(g(3)) = h(9) = 81。

结合律验证：(h∘g)∘f = h∘(g∘f) = h∘g∘f，三重复合括号可省略。

**例题2**：证明若 f: A→B 和 g: B→C 都是双射，则 g∘f 是双射且 (g∘f)⁻¹ = f⁻¹∘g⁻¹。

**解答**：由双射的复合保双射性，g∘f 是双射。验证逆公式：(f⁻¹∘g⁻¹)∘(g∘f) = f⁻¹∘(g⁻¹∘g)∘f = f⁻¹∘id_B∘f = f⁻¹∘f = id_A。类似地 (g∘f)∘(f⁻¹∘g⁻¹) = id_C。由逆的唯一性即得 (g∘f)⁻¹ = f⁻¹∘g⁻¹。

## 代码示例

```python
from functools import reduce

def compose(*functions):
    """多个函数的复合: compose(f, g, h)(x) = f(g(h(x)))"""
    def composed(x):
        result = x
        for f in reversed(functions):
            result = f(result)
        return result
    return composed

def f(x): return x + 2
def g(x): return 3 * x
def h(x): return x ** 2

h_g_f = compose(h, g, f)
print(f"(h∘g∘f)(1) = {h_g_f(1)}")  # 81

# 验证结合律: (h∘g)∘f = h∘(g∘f)
h_g = compose(h, g)
g_f = compose(g, f)
lhs = compose(h_g, f)(5)
rhs = compose(h, g_f)(5)
print(f"结合律验证: {lhs == rhs}")  # True
```
