## 核心定义

量词（Quantifier）是一阶逻辑中用于表达论域中个体数量特征的关键符号。最基础的两种量词是**全称量词** ∀（Universal Quantifier）和**存在量词** ∃（Existential Quantifier）。∀x P(x) 读作"对所有 x，P(x) 成立"，当且仅当论域中每一个体都满足 P 时为真。∃x P(x) 读作"存在 x 使得 P(x) 成立"，当且仅当论域中至少有一个个体满足 P 时为真。量词紧邻的变元称为**指导变元**，量词的作用范围称为**辖域**（Scope）。变元的自由出现（Free Occurrence）与约束出现（Bound Occurrence）是核心概念：在量词辖域内的指导变元为约束出现，否则为自由出现。含有自由变元的公式不是命题（真值不确定），不含任何自由变元的公式称为**语句**（Sentence / Closed Formula），才是真正的命题。全称量词与存在量词可以通过否定相互定义：¬∀x P(x) ≡ ∃x ¬P(x)，¬∃x P(x) ≡ ∀x ¬P(x)。

## 关键结论

- 全称量词 ∀ 对应自然语言中的"所有""每个""任意""一切"
- 存在量词 ∃ 对应自然语言中的"存在""有""至少有一个""某些"
- ∃!x P(x)（存在唯一 x 使得 P(x)）可以定义为 ∃x (P(x) ∧ ∀y (P(y) → y = x))，即存在且唯一
- 量词的辖域和约束关系决定了一个公式的逻辑结构，改变辖域会影响语义
- 量词顺序至关重要：∀x∃y P(x,y) ≠ ∃y∀x P(x,y)。前者说每个 x 都有某个 y（可能不同），后者说存在同一个 y 对所有的 x 都成立
- 一阶逻辑的量词只约束个体变元，不能约束谓词变元——这是一阶与二阶逻辑的本质区别

## 易错点

1. 混淆量词顺序："每个人都爱某个人"是 ∀x∃y Loves(x,y)，而"有某个人被所有人爱"是 ∃y∀x Loves(x,y)，两者语义不同
2. 错误处理所有与存在的否定：¬∀x P(x) 等价于 ∃x ¬P(x)，而不是 ∀x ¬P(x)。"并非所有人都及格"并不意味着"所有人都不及格"
3. 辖域判断错误：∀x P(x) → Q(x) 中，∀x 的辖域仅为 P(x)，最后的 x（Q(x)中）是自由变元——这与 ∀x (P(x) → Q(x)) 不同

## 例题

**例题1**：设论域为全体实数，P(x): x > 0, Q(x): x² > 0。判断下列命题的真值。
(a) ∀x (P(x) → Q(x))
(b) ∀x (Q(x) → P(x))
(c) ∃x (P(x) ∧ ¬Q(x))

**解答**：
(a) 真。任何正数的平方都大于 0。
(b) 假。x = -1 时，Q(-1) 为真（1>0），但 P(-1) 为假（-1≯0）。
(c) 假。任何正数的平方都大于 0，不存在反例。

**例题2**：用谓词和量词形式化"不存在最大的素数"。

设论域为正整数。谓词 P(x): x 是素数，G(x,y): x > y。
形式化：¬∃x (P(x) ∧ ∀y (P(y) → (x = y ∨ G(x, y))))
或等价地：∀x (P(x) → ∃y (P(y) ∧ G(y, x)))
即"对每个素数，总存在更大的素数"。

## 代码示例

```python
# 量词的Python实现
from typing import Callable, Iterable, TypeVar

T = TypeVar('T')

def forall(domain: Iterable[T], predicate: Callable[[T], bool]) -> bool:
    """全称量词 ∀: 域中所有元素满足谓词"""
    return all(predicate(x) for x in domain)

def exists(domain: Iterable[T], predicate: Callable[[T], bool]) -> bool:
    """存在量词 ∃: 域中存在元素满足谓词"""
    return any(predicate(x) for x in domain)

# 验证 De Morgan 律: ¬∀xP(x) ≡ ∃x¬P(x)
domain = range(1, 101)
def is_even(x): return x % 2 == 0

lhs = not forall(domain, is_even)
rhs = exists(domain, lambda x: not is_even(x))
print(f"¬∀xEven(x) ≡ ∃x¬Even(x): {lhs == rhs}")

# 量词顺序的重要性
D = [1, 2, 3]
# ∀x∃y: x <= y — 每个x有某个y >= x (总是真)
f1 = all(any(x <= y for y in D) for x in D)
# ∃y∀x: x <= y — 存在某个y，所有x都 <= y (y=3为真)
f2 = any(all(x <= y for x in D) for y in D)
print(f"∀x∃y(x<=y): {f1}, ∃y∀x(x<=y): {f2}")
```
