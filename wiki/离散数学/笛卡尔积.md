## 核心定义

笛卡尔积（Cartesian Product）是集合论中构造有序结构的基本运算。对于两个集合 A 和 B，它们的笛卡尔积 A × B 定义为所有第一分量取自 A、第二分量取自 B 的有序对（Ordered Pair）的集合：A × B = {(a, b) | a ∈ A, b ∈ B}。有序对 (a, b) 的核心性质是：(a, b) = (c, d) 当且仅当 a = c 且 b = d（这与无序对 {a, b} 不同，后者不受顺序影响）。笛卡尔积不满足交换律（A × B ≠ B × A 一般情形成立）但满足对并交差的分配律。推广到 n 个集合的情形，n 重笛卡尔积 A₁ × A₂ × ... × Aₙ 是所有 n 元有序组 (a₁, a₂, ..., aₙ) 的集合，其中 aᵢ ∈ Aᵢ。特别地，当所有 Aᵢ 相同（都等于 A）时，记 Aⁿ = A × A × ... × A（n 次）。笛卡尔积是定义**关系**（Relation）和**函数**（Function）的基础结构。

## 关键结论

- |A × B| = |A| × |B|（对有限集，笛卡尔积的基数等于基数的乘积），这解释了其名称中"积"的由来
- 空集的笛卡尔积为空：A × ∅ = ∅ × A = ∅
- 笛卡尔积对并集满足分配律：A × (B ∪ C) = (A × B) ∪ (A × C)，类似对交集也满足分配律
- 有序对的集合论严格定义（Kuratowski定义）：(a, b) = {{a}, {a, b}}，该定义能够证明有序对的核心性质
- ℝ² = ℝ × ℝ 是平面直角坐标系的数学基础——每个点由两个坐标（x,y）唯一确定

## 易错点

1. 笛卡尔积不满足交换律：A × B ≠ B × A。例如 {1} × {a} = {(1,a)}，而 {a} × {1} = {(a,1)}，两者不同
2. 混淆有序对 (a,b) 与无序对 {a,b}：当 a≠b 时 {a,b} = {b,a}，但 (a,b) ≠ (b,a)
3. 误用分配律形式：A × (B ∩ C) = (A × B) ∩ (A × C) 正确，但 (A × B) ∩ (C × D) ≠ (A ∩ C) × (B ∩ D) 实为 = (A ∩ C) × (B ∩ D) ——这个其实是正确的，需要检查

## 例题

**例题1**：设 A = {1, 2}, B = {x, y}。求 A × B 和 B × A，并比较。

**解答**：
A × B = {(1,x), (1,y), (2,x), (2,y)}
B × A = {(x,1), (x,2), (y,1), (y,2)}
A × B ≠ B × A，|A × B| = |B × A| = 4。
验证了交换律不成立。

**例题2**：证明 A × (B ∩ C) = (A × B) ∩ (A × C)。

**解答**：
设 (a, x) ∈ A × (B ∩ C)，则 a ∈ A 且 x ∈ B ∩ C，即 x ∈ B 且 x ∈ C。故 (a, x) ∈ A × B 且 (a, x) ∈ A × C，即 (a, x) ∈ (A × B) ∩ (A × C)。反向包含同理。所以等式成立。

## 代码示例

```python
from itertools import product

A = {1, 2, 3}
B = {'x', 'y'}

# 笛卡尔积
cartesian_product = set(product(A, B))
print(f"A × B = {cartesian_product}")
print(f"|A × B| = {len(cartesian_product)}")  # 6

# 不满足交换律
B_cross_A = set(product(B, A))
print(f"A × B == B × A: {cartesian_product == B_cross_A}")  # False

# 分配律验证: A × (B ∪ C)
C = {'a', 'b'}
lhs = set(product(A, B | C))
rhs = set(product(A, B)) | set(product(A, C))
print(f"A×(B∪C) = (A×B)∪(A×C): {lhs == rhs}")  # True

# n重笛卡尔积
print(f"A³ 的元素个数: {len(list(product(A, repeat=3)))}")  # 27 = 3³
```
