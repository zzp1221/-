## 核心定义

自动机理论（Automata Theory）是理论计算机科学的基石，研究抽象计算模型及其能力边界。**确定性有限自动机**（DFA, Deterministic Finite Automaton）是一个五元组 M = (Q, Σ, δ, q₀, F)，其中 Q 是有限状态集，Σ 是输入字母表，δ: Q×Σ → Q 是转移函数（确定性的），q₀∈Q 是初始状态，F⊆Q 是接受状态集。**非确定性有限自动机**（NFA, Nondeterministic Finite Automaton）将转移函数改为 δ: Q×Σ→P(Q)，允许多个可能的下一状态，也可有 ε-转移（不消耗输入的状态转移）。DFA 和 NFA 在表达能力上**等价**：任何 NFA 可转化为等价的 DFA（子集构造法），因此它们都恰好识别**正则语言**（Regular Language）。**正则表达式**是正则语言的代数表示。**图灵机**（Turing Machine）是具有无限读写带的自动机，可以模拟任何算法可计算的过程，其所能识别的语言类为**递归可枚举语言**（Recursively Enumerable Languages）。图灵机是Church-Turing论题的核心：任何"算法上可计算的"函数都可以被某台图灵机计算。

## 关键结论

- DFA 和 NFA 等价（都识别正则语言），但 NFA 可能比 DFA 指数地更简洁（状态数 O(n) vs O(2ⁿ)）
- 正则语言对并、交、补、连接、Kleene星号封闭——这是正则表达式的代数基础
- 正则语言的泵引理（Pumping Lemma）提供了一种证明某语言不是正则的方法：若 L 正则，则存在 p 使得任何长度 ≥ p 的串 w∈L 可分解为 w=xyz 满足 |xy|≤p、|y|>0、且 xyⁱz∈L
- DFA 最小化算法（Moore或Hopcroft算法）可将任意 DFA 转化为等价的最小 DFA（状态数最少且同构意义下唯一）
- 图灵机 = DFA + 无限可读写带；通用图灵机可模拟任何其他图灵机——这是现代通用计算机的理论模型
- 停机问题（Halting Problem）不可判定：不存在算法判断任意图灵机是否在给定输入上停机

## 易错点

1. 混淆 NFA 状态转移的不确定性：从同一状态读同一输入符可转移到多个不同状态——但 NFA 接受一串当且仅当存在某条路径到达接受状态
2. DFA 到 NFA 转换忽略死状态：子集构造法中，所有从未到达的状态子集可被删除或合并
3. 正则语言泵引理的逆用：泵引理是正则语言**必要**条件，满足泵引理的语言不一定是正则的（充分性不成立）

## 例题

**例题1**：构造 DFA 识别语言 L = {w | w 中 'a' 的个数是 3 的倍数}（字母表 Σ = {a, b}）。

**解答**：状态 Q = {q₀, q₁, q₂} 表示当前计数值 mod 3：q₀（计数≡0，接受状态）、q₁（≡1）、q₂（≡2）。转移：读 a 时状态+1（mod 3），读 b 时状态不变。初始状态 = q₀，F = {q₀}。例如 w="aab" 的路径：q₀→a→q₁→a→q₂→b→q₂，q₂≠q₀，拒绝。

**例题2**：用泵引理证明 L = {0ⁿ1ⁿ | n ≥ 1} 不是正则语言。

**解答**：假设 L 正则，泵引理给出泵长度 p。取 s = 0ᵖ1ᵖ ∈ L, |s|≥p。将 s 分解为 xyz，|xy|≤p 且 |y|>0。由于 |xy|≤p，xy 全在 0 区域，故 y 仅含 0。泵 y（重复）后 xy²z 的 0 多于 1，不在 L 中，矛盾。故 L 不是正则语言。

## 代码示例

```python
class DFA:
    def __init__(self, states, alphabet, transitions, start, accepts):
        self.states = states
        self.alphabet = alphabet
        self.transitions = transitions  # {(q, a): q'}
        self.start = start
        self.accepts = accepts
    
    def run(self, input_str):
        state = self.start
        for char in input_str:
            state = self.transitions.get((state, char))
            if state is None:
                return False
        return state in self.accepts

# DFA: a的个数是3的倍数
dfa_mod3 = DFA(
    states={'q0', 'q1', 'q2'},
    alphabet={'a', 'b'},
    transitions={('q0','a'):'q1', ('q0','b'):'q0',
                 ('q1','a'):'q2', ('q1','b'):'q1',
                 ('q2','a'):'q0', ('q2','b'):'q2'},
    start='q0',
    accepts={'q0'}
)

for w in ['', 'a', 'aa', 'aaa', 'aab', 'baa', 'aaab']:
    print(f"DFA接受 '{w}': {dfa_mod3.run(w)}")
```
