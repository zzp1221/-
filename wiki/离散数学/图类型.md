## 核心定义

图（Graph）是离散数学中描述对象之间二元关系的基本数学模型。一个图 G = (V, E) 由顶点集 V（Vertex Set）和边集 E（Edge Set）组成。根据边的类型和特征，图可分为多种类型：(1) **无向图**（Undirected Graph）：边是无序对 {u, v}，表示对称关系；(2) **有向图**（Directed Graph / Digraph）：边是有序对 (u, v)，表示方向性关系；(3) **简单图**（Simple Graph）：无自环（Loop）、无重复边（多重边）；(4) **多重图**（Multigraph）：允许多重边；(5) **完全图** Kₙ：n 个顶点的简单图中每对不同的顶点都有一条边相连，共 n(n-1)/2 条边；(6) **二分图**（Bipartite Graph）：顶点集可分为两个不相交子集 X 和 Y，使得每条边连接 X 中一个顶点和 Y 中一个顶点；(7) **平面图**（Planar Graph）：能画在平面上且边仅相交于顶点的图；(8) **正则图**（Regular Graph）：所有顶点的度都相等，若度均为 k 则称 k-正则图；(9) **零图**：无边图；**平凡图**：仅有一个顶点。

## 关键结论

- 简单图不允许自环和多重边，是图论中最基本的研究对象
- 完全二部图 K_{m,n}：|X|=m, |Y|=n，每个 X 中的顶点与 Y 中所有顶点相连，共 m×n 条边
- 完全图 K₅ 和完全二部图 K₃,₃ 是判定平面性的关键——Kuratowski定理指出一个图是平面图当且仅当它不包含 K₅ 或 K₃,₃ 的细分
- 图的补图 Ḡ：与 G 有相同顶点集，边恰好是 G 中所没有的简单图边
- 子图、生成子图、诱导子图是图的三种基本包含关系，各有不同的定义严谨度

## 易错点

1. 混淆有向边与无向边在图中的表示：无向图中 {u,v} = {v,u} 是同一条边，有向图中 (u,v) 和 (v,u) 是两条不同的弧
2. 认为是图就必须连通：空图（无边图）、非连通图都是合法的图，图的定义中不隐含连通性
3. 忽略简单图的限制条件：说"完全图有 n(n-1)/2 条边"的前提是简单图，在允许多重边的情况下边数不受此限制

## 例题

**例题1**：证明在任意 6 个人中，或者有 3 个人两两相识，或者有 3 个人两两不相识。（Ramsey 理论经典例子）

**解答**：以 6 个顶点表示 6 个人，相识关系用红边表示，不相识用蓝边表示（完全图 K₆ 的边二染色）。取任一顶点 v，它与其他 5 个顶点有 5 条边，由鸽巢原理至少 3 条同色（设为红色）。设这 3 条边的另一端点为 a, b, c。若 a, b, c 之间有红边，则该红边与 v 构成红色三角形（三人两两相识）；若 a, b, c 之间全是蓝边，则 a, b, c 构成蓝色三角形（三人两两不相识）。证毕。

**例题2**：判断 Petersen 图是否为二分图。Petersen 图有 10 个顶点，每个顶点度数为 3。

**解答**：二分图不含奇圈。Petersen 图含有长度为 5 的圈（奇圈），故不是二分图。

## 代码示例

```python
class Graph:
    def __init__(self, vertices, directed=False):
        self.V = vertices
        self.directed = directed
        self.adj = {v: set() for v in vertices}
    
    def add_edge(self, u, v):
        self.adj[u].add(v)
        if not self.directed:
            self.adj[v].add(u)
    
    def is_bipartite(self):
        """BFS染色法判断二分图"""
        color = {}
        for start in self.V:
            if start not in color:
                color[start] = 0
                queue = [start]
                for u in queue:
                    for v in self.adj[u]:
                        if v not in color:
                            color[v] = 1 - color[u]
                            queue.append(v)
                        elif color[v] == color[u]:
                            return False
        return True

# 测试: C5 是奇圈，不是二分图
g = Graph(range(5))
for i in range(5):
    g.add_edge(i, (i+1) % 5)
print(f"C5是二分图: {g.is_bipartite()}")  # False
```
