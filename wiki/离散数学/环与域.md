## 核心定义

**环**（Ring）是具有两种二元运算（通常称为加法 + 和乘法 ·）的代数结构 (R, +, ·)，满足：(1) (R, +) 是阿贝尔群（加法群）；(2) (R, ·) 是半群（乘法满足结合律）；(3) 乘法对加法满足分配律（a·(b+c) = a·b + a·c 且 (a+b)·c = a·c + b·c）。若乘法还满足交换律，则称**交换环**（Commutative Ring）；若存在乘法单位元 1（1·a = a·1 = a），则称**含幺环**（Ring with Unity）。整数环 (ℤ, +, ·) 是最典型的交换含幺环。

**域**（Field）是一个满足以下额外条件的交换含幺环：每个非零元素 a 都有乘法逆元 a⁻¹（使得 a·a⁻¹ = 1）。换言之，域的非零元素关于乘法构成阿贝尔群。有理数 ℚ、实数 ℝ、复数 ℂ 都是域；整数 ℤ 不是域（例如 2 没有乘法逆元在 ℤ 中）。**有限域**（Finite Field / Galois Field）是元素个数有限的域，记作 GF(pⁿ) 或 F_{pⁿ}，其中 p 为素数。有限域在密码学（AES 使用 GF(2⁸)）、编码理论（Reed-Solomon码）和计算机科学中有核心应用。

## 关键结论

- 环中加法单位元 0 满足 0·a = a·0 = 0（零乘任何元得零）
- 整环（Integral Domain）是无零因子的交换含幺环：若 a·b = 0，则 a = 0 或 b = 0
- 除环（Division Ring）中每个非零元有乘法逆元但不要求乘法交换（如四元数 ℍ）
- Wedderburn小定理：任何有限除环必为域（即有限除环的乘法自动交换）
- 有限域 GF(pⁿ) 中元素的个数必为质数幂 pⁿ；特征为 p 且对任何元素 a 有 a^{pⁿ} = a
- 域上的多项式环 F[x] 是主理想整环（PID），类似于整数环有唯一分解性

## 易错点

1. 混淆环与域：域 = 环 + 非零元关于乘法构成群。例如 ℤ 是环但不是域（2 在 ℤ 中无乘法逆元）
2. 认为有限环一定是域：ℤ₆（模6整数环）是有限环，但不是域因为 2×3 ≡ 0(mod 6)，故 ℤ₆ 不是整环（有零因子），当然更不是域。ℤₚ（p 素数）才是有限域
3. 混淆 GF(2ⁿ) 与 ℤ_{2ⁿ}：GF(8) ≠ ℤ₈（ℤ₈ 有零因子 2×4=0 mod 8）；GF(8) 是含 8 个元素的域，结构不同于 ℤ₈

## 例题

**例题1**：证明在任意环 R 中，0·a = 0 对所有 a ∈ R 成立。

**解答**：由分配律，0·a = (0+0)·a = 0·a + 0·a。两边加 -(0·a) 得 0 = 0·a。同理可证 a·0 = 0。

**例题2**：构造有 4 个元素的有限域 GF(4)。

**解答**：GF(4) 不能是 ℤ₄（因为 4 不是素数，ℤ₄ 不是整环）。GF(4) = {0, 1, α, α+1}，其中 α 是方程 x²+x+1=0（在 GF(2) 上不可约）的根。加法按 GF(2) 向量加法（分量异或），乘法由 α² = α+1 确定。以 α 为生成元的乘法群：α¹=α, α²=α+1, α³=1。|GF(4)*| = 3，乘法群为 3 阶循环群。

## 代码示例

```python
class ZpField:
    """模p (p素数) 的有限域 GF(p) = ℤ_p"""
    def __init__(self, p):
        if not self._is_prime(p):
            raise ValueError("p必须是素数")
        self.p = p
    
    def _is_prime(self, n):
        if n < 2: return False
        for i in range(2, int(n**0.5)+1):
            if n % i == 0: return False
        return True
    
    def add(self, a, b): return (a + b) % self.p
    def mul(self, a, b): return (a * b) % self.p
    def inv(self, a):
        """模p乘法逆元 (费马小定理)"""
        if a % self.p == 0:
            raise ValueError("0无逆元")
        return pow(a, self.p - 2, self.p)
    
    def elements(self):
        return list(range(self.p))

# GF(7)
gf7 = ZpField(7)
print(f"3的逆元在GF(7)中: {gf7.inv(3)}")  # 5 (因3×5=15≡1 mod 7)
print(f"验证: 3×{gf7.inv(3)} mod 7 = {gf7.mul(3, gf7.inv(3))}")  # 1
```
