## 核心定义

**子集**（Subset）：对集合 A 和 B，若 A 的每个元素都是 B 的元素（即 ∀x(x∈A → x∈B)），则称 A 是 B 的子集，记作 A ⊆ B。若 A ⊆ B 且 A ≠ B，则称 A 是 B 的**真子集**（Proper Subset），记作 A ⊂ B 或 A ⊊ B。**幂集**（Power Set）是集合 A 的所有子集构成的集合，记作 P(A) 或 2^A：P(A) = {X | X ⊆ A}。若 |A| = n，则 |P(A)| = 2^n——此结论可用数学归纳法严格证明。幂集的构造可以按子集的基数分层进行：空集、所有单元素子集、所有双元素子集...直到 A 本身。幂集是后续学习关系、函数、测度论、拓扑学等概念的基础。特别地，P(∅) = {∅}（含一个元素），而不是 ∅；P({∅}) = {∅, {∅}}。幂集的大小以指数速度增长：当 n=10 时幂集已有 1024 个元素。

## 关键结论

- 任何集合 A 都有两个平凡子集：∅ 和 A 自身（∀A: ∅ ⊆ A, A ⊆ A）
- 子集关系 ⊆ 是偏序关系：自反（A ⊆ A）、反对称（A ⊆ B ∧ B ⊆ A → A = B）、传递（A ⊆ B ∧ B ⊆ C → A ⊆ C）
- 幂集中元素个数与 |A| 满足指数关系 |P(A)| = 2^{|A|}——这本质上是组合数学中每个元素"选或不选"的乘法原理
- 幂集的基数严格大于原集合的基数（Cantor定理）：对任何集合 A（包括无穷集），|P(A)| > |A|
- 幂集构造是迭代的：P(P(A)) = P²(A) 有 2^(2^n) 个元素

## 易错点

1. 将子集关系 ⊆ 与属于关系 ∈ 混淆：A ⊆ B 要求 A 的每个元素都在 B 中，而 A ∈ B 要求 A 整体是 B 的一个元素。例如 {1} ⊆ {1, 2} 为真，但 {1} ∈ {1, 2} 为假
2. 误认为 P(A) 包含 A 的所有元素：P(A) 的元素是 A 的**子集**，而非 A 的元素本身。例如 P({1,2}) = {∅, {1}, {2}, {1,2}}，不包含 1 或 2
3. 忽略空集的幂集：P(∅) = {∅}，有 1 个元素，这符合公式 2^0 = 1

## 例题

**例题1**：求集合 A = {a, b, c} 的幂集 P(A)。

**解答**：|A| = 3，|P(A)| = 2³ = 8。
P(A) = {∅, {a}, {b}, {c}, {a,b}, {a,c}, {b,c}, {a,b,c}}。

按子集大小排列：
- 0个元素：∅
- 1个元素：{a}, {b}, {c}
- 2个元素：{a,b}, {a,c}, {b,c}
- 3个元素：{a,b,c}

**例题2**：若 |A| = 5，问 P(A) 有多少个元素？其中有多少个含恰好 2 个元素的子集？

**解答**：|P(A)| = 2^5 = 32。含恰好 2 个元素的子集数为 C(5,2) = 10（从 5 个元素中选 2 个的组合数）。

## 代码示例

```python
from itertools import combinations

def power_set(s):
    """生成集合s的幂集"""
    s = list(s)
    n = len(s)
    result = []
    for r in range(n + 1):
        result.extend(combinations(s, r))
    return [set(combo) for combo in result]

A = {'a', 'b', 'c'}
P_A = power_set(A)
print(f"P({{a,b,c}})有 {len(P_A)} 个元素: {P_A}")
# 输出: 8个元素

# 验证 |P(A)| = 2^|A|
for n in range(0, 8):
    test_set = set(range(n))
    assert len(power_set(test_set)) == 2**n
print("|P(A)| = 2^n 验证通过 (n=0..7)")
```
