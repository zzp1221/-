## 核心定义

解释（Interpretation）是一阶逻辑的语义学核心概念，指为形式语言中的符号赋予具体含义的映射。一个一阶逻辑的解释 I 由以下要素构成：(1) 一个非空集合 D，称为**论域**（Domain of Discourse / Universe）；(2) 对每个个体常元符号 c，指派 D 中的一个元素 c^I ∈ D；(3) 对每个 n 元函数符号 f，指派一个函数 f^I: Dⁿ → D；(4) 对每个 n 元谓词符号 P，指派一个 n 元关系 P^I ⊆ Dⁿ（即 D 上使 P 为真的 n 元组的集合）。**赋值**（Assignment / Valuation）是临时性的补充：将个体变元映射到 D 中的元素。一个公式 φ 在解释 I 下（相对于某个赋值）为真，记作 I ⊨ φ。如果一个公式在所有解释下均为真，称为**有效式**（Valid Formula，一阶永真式）。如果存在某个解释使 φ 为真，则 φ 是**可满足的**（Satisfiable）。解释的概念使得我们能精确讨论一阶逻辑公式的语义合法性和逻辑蕴含关系。

## 关键结论

- 同一个一阶逻辑公式在不同的解释下可以有不同的真值，这正体现了谓词逻辑较命题逻辑更强的表达能力
- 解释中的论域 D 可以是有穷集（如{1,2,3}）也可以是无穷集（如实数集 R），不同论域下的语义可能截然不同
- 有效式（如 ∀x P(x) → P(a)）在任何解释下均为真，构成一阶逻辑的定理
- 论域为空时，全称语句无意地真（Vacuous Truth），这时 ∀x P(x) 总是真的而 ∃x P(x) 总是假的
- 一阶逻辑的紧致性定理（Compactness）：若公式集 Γ 的每个有穷子集都有模型，则 Γ 自身也有模型
- Lowenheim-Skolem定理表明一阶逻辑无法刻画无穷集合的确切基数（不能区分可数无穷与不可数无穷）

## 易错点

1. 将"解释"理解为自然语言的翻译：逻辑解释是严格的数学映射（将符号映射到集合论实体），而非将其翻译为汉语或英语
2. 混淆"可满足"与"有效"：P(a) 是可满足的（存在使 P(a) 为真的解释）但不是有效的（也存在使 P(a) 为假的解释）
3. 忘记论域非空的约束：一阶逻辑的标准语义要求论域非空，∀x P(x) ⊨ ∃x P(x) 只在非空论域下成立

## 例题

**例题1**：给定公式 φ = ∀x∃y R(x, y)，构造两个不同的解释分别使 φ 为真和为假。

**解答**：
使 φ 为真的解释：论域 D = ℕ（自然数），R(x, y) 解释为 "y = x + 1"（每个自然数都有后继）。
使 φ 为假的解释：论域 D = {0, 1, ..., 9}，R(x, y) 解释为 "y = x - 1"（当 x = 0 时，不存在 y 使 y = -1 ∈ D）。

**例题2**：证明公式 ∃x ∀y (P(y) → P(x)) 在任何非空论域的解释下均为真。

**解答**：若存在某个个体使 ¬P(x) 为真，取它为 x0，则对任意 y，P(y) → P(x0) 等价于 P(y) → F，仅当 P(y)=F 时为真。但这不能保证对所有 y 都成立（可能有某个 y0 使 P(y0)=T，此时 P(y0) → P(x0) ≡ T → F ≡ F）。若所有个体使 P(x) 为真，则任取一个 x0，对任意 y 有 P(y) → P(x0) ≡ P(y) → T ≡ T。故公式在任何非空论域解释下为真，是有效式。

## 代码示例

```python
from itertools import product
from typing import Set, List, Callable, Dict, Tuple

# 构造一阶逻辑的解释
class Interpretation:
    def __init__(self, domain: Set[int]):
        self.domain = domain
    
    def eval_unary(self, predicate: Callable[[int], bool], 
                   x: int) -> bool:
        """一元谓词在当前论域下的真值"""
        return predicate(x)
    
    def eval_forall(self, predicate: Callable[[int], bool]) -> bool:
        """全称量词求值"""
        return all(predicate(x) for x in self.domain)
    
    def eval_exists(self, predicate: Callable[[int], bool]) -> bool:
        """存在量词求值"""
        return any(predicate(x) for x in self.domain)

# 构造两个不同的解释
interp1 = Interpretation(set(range(10)))
interp2 = Interpretation(set(range(100)))

# 同一个谓词在不同解释下的评估
def is_even(x): return x % 2 == 0

print(f"论域[0..9]: ∃x Even(x) = {interp1.eval_exists(is_even)}")
print(f"论域[0..99]: ∃x Even(x) = {interp2.eval_exists(is_even)}")
print(f"论域[0..9]: ∀x Even(x) = {interp1.eval_forall(is_even)}")
```
