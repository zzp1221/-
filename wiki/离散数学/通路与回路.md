## 核心定义

通路（Walk）是图论中最基本的动态概念，描述从顶点到顶点沿边行进的路线。在无向图 G = (V, E) 中，从 v₀ 到 vₖ 的一条通路是顶点和边的交替序列 v₀, e₁, v₁, e₂, ..., eₖ, vₖ，其中每条边 eᵢ 介于 vᵢ₋₁ 和 vᵢ 之间。根据限制条件的不同，有以下精细分类：(1) **通路**（Walk）：无任何额外限制，边和顶点均可重复；(2) **迹**（Trail）：边不重复的通路（顶点可重复）；(3) **路径**（Path）：顶点不重复的通路（从而边也不重复）；(4) **闭通路**（Closed Walk）：起止点相同的通路；(5) **回路/闭迹**（Circuit/Closed Trail）：边不重复的闭通路；(6) **圈**（Cycle）：除起止点外顶点不重复的闭通路（即顶点不重复的回路）。路径长度等于所经过的边数。图中两个顶点 u 和 v 之间的**最短路径**长度称为它们之间的**距离** d(u, v)。在加权图中，路径的权重为各边权重之和，最短路径问题（Dijkstra算法、Floyd算法）是图论中的经典算法问题。

## 关键结论

- 长度 k 的路径包含 k+1 个顶点和 k 条边
- 简单图中，若顶点数 n ≥ 2，则最长简单路径的长度不超过 n−1
- 每个 u-v 通路包含一个 u-v 路径（可通过删除循环和重复部分获得）——这是路径存在性定理
- 如果存在从 u 到 v 的长度为奇数（偶数）的通路，则存在长度不超过 n−1 的奇数（偶数）路径——奇偶性分析在图的二染色问题中有重要意义
- 任何无向图中，若最小度数 δ(G) ≥ 2，则图必定包含一个圈

## 易错点

1. 混淆"路"（Path）的不同定义：离散数学中 Path 严格指顶点不重复的通路，而日常语言中"一条路线"可能允许重复
2. 将"闭迹"（Circuit）与"圈"（Cycle）混淆：闭迹允许内部顶点重复，圈不允许——例如图"8字形"中沿内外圈走一圈是闭迹但不是圈
3. 认为通路必须包含边：长度为 0 的平凡通路（仅一个顶点）也是合法的通路

## 例题

**例题1**：在一个有 n 个顶点的简单无向图中，证明任何长度超过 n−1 的通路必然包含一个圈。

**解答**：设通路 W = v₀, e₁, ..., vₖ 长度 k > n−1，则经过 k+1 > n 个顶点。由鸽巢原理，至少有一个顶点重复出现。设第一个重复出现的顶点为 vᵢ = vⱼ（i < j），则 vᵢ 到 vⱼ 之间的子序列构成一个圈（因为 vᵢ 到 vⱼ−1 之间的顶点均不重复，这是"第一个重复"的保证）。

**例题2**：证明在任何图中，从 u 到 v 存在通路当且仅当存在 uv-路径。

**解答**：（⇐）路径是通路的特例，显然。（⇒）取 uv 通路中最短的一条 W。若 W 中有重复顶点 vᵢ = vⱼ，则删除 vᵢ 到 vⱼ 之间的子通路，得到一条更短的 uv 通路，与"最短"矛盾。故最短通路无重复顶点，即为路径。

## 代码示例

```python
from collections import deque

def bfs_shortest_path(graph, start, end):
    """BFS求无权图最短路径"""
    if start == end:
        return [start]
    queue = deque([[start]])
    visited = {start}
    
    while queue:
        path = queue.popleft()
        node = path[-1]
        for neighbor in graph[node]:
            if neighbor == end:
                return path + [neighbor]
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(path + [neighbor])
    return None

# 简单图: 0-1-2-3-4
g = {0: [1], 1: [0, 2], 2: [1, 3], 3: [2, 4], 4: [3]}
print(f"0到4的最短路径: {bfs_shortest_path(g, 0, 4)}")
# 输出: [0, 1, 2, 3, 4]
```
