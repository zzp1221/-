## 核心定义

联结词（Logical Connectives）是命题逻辑中用于将原子命题组合为复合命题的逻辑运算符。常用联结词有五个：**否定**（¬，非）、**合取**（∧，且）、**析取**（∨，或）、**蕴含**（→，如果...则...）、**等价**（↔，当且仅当）。每个联结词都由其真值表精确定义，其语义是纯真值函数的——复合命题的真值完全由子命题的真值和联结词的运算规则决定，不依赖于子命题的具体内容。联结词可以嵌套使用，通过括号明确运算优先级。优先级从高到低通常为：¬ > ∧ > ∨ > → > ↔。联结词的功能完备性（Functional Completeness）是一个重要概念：{¬, ∧, ∨} 构成一个功能完备集，可以表达所有布尔函数；{¬, ∧} 和 {¬, ∨} 各自也是功能完备集；甚至 NAND（↑，与非）和 NOR（↓，或非）各自单独就构成功能完备集。

## 关键结论

- 否定 ¬p：当 p 为真时 ¬p 为假，反之亦然，这是唯一的一元联结词
- 合取 p ∧ q：当且仅当 p 和 q 同时为真时才为真，体现了"且"的严格性
- 析取 p ∨ q：当 p 和 q 至少有一个为真时就为真，注意这是"可兼或"（inclusive OR）
- 蕴含 p → q：当 p 为真而 q 为假时整个蕴含式为假，其余情况为真；p 为假时蕴含式自动为真（虚真，Vacuous Truth）
- 等价 p ↔ q：当 p 和 q 真值相同时为真，是双向蕴含 (p → q) ∧ (q → p) 的简写
- 异或 p ⊕ q：当 p 和 q 真值不同时为真，即"不可兼或"，可表示为 (p ∨ q) ∧ ¬(p ∧ q)

## 易错点

1. 混淆"或"的日常用法与逻辑析取：日常语言中的"或"有时表"不可兼或"（如"要么你去要么我去"），但逻辑析取 ∨ 默认是"可兼或"（inclusive OR），两者都去也为真
2. 蕴含式的理解误区：p → q 不要求 p 和 q 有因果关系；"如果 1+1=3，则地球是圆的"在逻辑上为真（因为前件为假），这在日常直觉中可能显得奇怪
3. 混淆 → 与 ↔ 的方向性：p → q 只保证 q 是 p 的必要条件，而不是充分条件；只有 p ↔ q 才表示充要条件

## 例题

**例题1**：设 p: "今天下雨"，q: "今天冷"。用自然语言表达下列命题。
(a) ¬p
(b) p ∧ q
(c) p ∨ q
(d) ¬p ∧ ¬q
(e) p → q
(f) p ↔ q

**解答**：
(a) 今天不下雨。
(b) 今天既下雨又冷。
(c) 今天下雨或者冷（或两者都有）。
(d) 今天既不下雨也不冷。
(e) 如果今天下雨，那么今天冷。
(f) 今天下雨当且仅当今天冷。

**例题2**：用真值表证明 ¬(p ∧ q) ≡ ¬p ∨ ¬q（De Morgan律）。

| p | q | p ∧ q | ¬(p ∧ q) | ¬p | ¬q | ¬p ∨ ¬q |
|---|---|-------|----------|----|----|---------|
| T | T | T     | F        | F  | F  | F       |
| T | F | F     | T        | F  | T  | T       |
| F | T | F     | T        | T  | F  | T       |
| F | F | F     | T        | T  | T  | T       |

¬(p ∧ q) 和 ¬p ∨ ¬q 在所有真值指派下结果相同，故等价成立。

## 代码示例

```python
# 联结词的Python实现
def neg(p: bool) -> bool:
    """否定"""
    return not p

def conj(p: bool, q: bool) -> bool:
    """合取"""
    return p and q

def disj(p: bool, q: bool) -> bool:
    """析取（可兼或）"""
    return p or q

def impl(p: bool, q: bool) -> bool:
    """蕴含"""
    return (not p) or q

def equiv(p: bool, q: bool) -> bool:
    """等价"""
    return p == q

def nand(p: bool, q: bool) -> bool:
    """与非，单独构成功能完备集"""
    return not (p and q)

# 验证De Morgan律
print("验证 ¬(p∧q) ≡ ¬p∨¬q:")
for p in [True, False]:
    for q in [True, False]:
        lhs = not (p and q)
        rhs = (not p) or (not q)
        assert lhs == rhs, f"De Morgan律不成立: p={p}, q={q}"
print("验证通过！")
```
