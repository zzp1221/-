## 核心定义

关系矩阵（Relation Matrix）是二元关系的矩阵表示形式，为关系的计算机存储和性质分析提供了代数化手段。对于有限集合 A = {a₁, a₂, ..., aₘ} 到 B = {b₁, b₂, ..., bₙ} 的关系 R，其关系矩阵 M_R 是一个 m×n 的布尔矩阵（0-1 矩阵），定义为：(M_R)_{ij} = 1 若 (aᵢ, bⱼ) ∈ R，否则为 0。当 A = B 时，M_R 为 n×n 方阵。利用关系矩阵，可以通过矩阵运算（布尔矩阵乘法）判定和计算关系性质：自反关系等价于 M_R 的对角线全为 1；对称关系等价于 M_R = M_R^T（转置）；反对称关系等价于 M_R 与 M_R^T 的对应非对角元不同时为 1；传递关系等价于 M_R² ≤ M_R（布尔意义下）。关系矩阵还可以通过**关系图**（有向图）直观表示：每个元素对应一个顶点，(a,b) ∈ R 对应从 a 到 b 的有向边——关系矩阵也就是该有向图的邻接矩阵。

## 关键结论

- M_R 的第 i 行：描述了元素 aᵢ 与 B 中哪些元素相关
- M_R 的第 j 列：描述了 A 中哪些元素与 bⱼ 相关
- 布尔矩阵乘法 ⊙：M_R² = M_R ⊙ M_R，其中 (M_R²)_{ij} = ∨ₖ ((M_R)_{ik} ∧ (M_R)_{kj})，即存在长度为 2 的路径
- 传递闭包 t(R) 的关系矩阵为 M_R ∨ M_R² ∨ M_R³ ∨ ... ∨ M_R^n（其中 n = |A|）
- Warshall 算法是计算传递闭包的标准方法，时间复杂度 O(n³)，空间复杂度 O(n²)
- 复合关系 R∘S 的关系矩阵：M_{R∘S} = M_S ⊙ M_R（注意顺序：先 S 后 R）

## 易错点

1. 混乱矩阵下标与元素的对应：M_R 的行对应 A 的元素（前件），列对应 B 的元素（后件），复合时易搞混
2. 布尔矩阵乘法与普通矩阵乘法的区别：布尔乘法用 ∧（与）代替乘法，∨（或）代替加法
3. 传递性矩阵判定的误区：M_R² ≤ M_R 是传递性的必要条件但要注意 ≤ 是逐元素比较（0≤0, 0≤1, 1≤1 均为真，1≤0 为假）

## 例题

**例题1**：设 A = {1, 2, 3}，关系 R = {(1,1), (1,2), (2,3)}。写出 M_R 并用矩阵乘法求 M_R²，判断 R 是否传递。

**解答**：
```
M_R = [[1, 1, 0],
       [0, 0, 1],
       [0, 0, 0]]

M_R² = M_R ⊙ M_R = [[1, 1, 1],
                     [0, 0, 0],
                     [0, 0, 0]]
```
M_R² 的 (1,3) 位置为 1 而 M_R 中 (1,3) 为 0，故 M_R² ≰ M_R，R 不是传递的。

**例题2**：利用关系矩阵证明：若关系 R 对称，则对任意正整数 k，Rᵏ 也对称。

**解答**：R 对称 ⇔ M_R = M_R^T。由布尔矩阵乘法，(M_R²)^T = (M_R ⊙ M_R)^T = M_R^T ⊙ M_R^T = M_R ⊙ M_R = M_R²，故 R² 对称。由数学归纳法可以扩展到任意正整数 k。

## 代码示例

```python
import numpy as np

def bool_mat_mult(A, B):
    """布尔矩阵乘法"""
    n = A.shape[0]
    C = np.zeros((n, n), dtype=int)
    for i in range(n):
        for j in range(n):
            C[i, j] = int(any(A[i, k] and B[k, j] for k in range(n)))
    return C

def is_reflexive(M):
    return all(M[i, i] == 1 for i in range(len(M)))

def is_symmetric(M):
    return np.array_equal(M, M.T)

def is_transitive(M):
    M2 = bool_mat_mult(M, M)
    return all(M2[i, j] <= M[i, j] 
               for i in range(len(M)) for j in range(len(M)))

# 示例
M = np.array([[1, 1, 0], [0, 0, 1], [0, 0, 0]])
print(f"自反: {is_reflexive(M)}")
print(f"对称: {is_symmetric(M)}")
print(f"传递: {is_transitive(M)}")
```
