## 核心定义

一阶逻辑推理（First-Order Inference）是将命题逻辑的推理规则扩展到包含量词和谓词的一阶逻辑中。除了保留命题逻辑的所有推理规则外，一阶逻辑新增了四条**量词推理规则**：(1) **全称实例化**（Universal Instantiation, UI）：从 ∀x P(x) 可推出任意个体常元 c 对应的 P(c)；(2) **全称泛化**（Universal Generalization, UG）：如果对任意选取的（非特定的）个体 c 能推出 P(c)，则可推出 ∀x P(x)；(3) **存在实例化**（Existential Instantiation, EI）：从 ∃x P(x) 可推出对某一个"新"的个体常元 c 有 P(c)，c 不能是之前出现过或隐含特定含义的常元；(4) **存在泛化**（Existential Generalization, EG）：从 P(c)（c 为某个体常元）可推出 ∃x P(x)。一阶逻辑推理最常用的方法论是**归结原理**（Resolution）：首先将公式化为斯柯伦范式（Skolem Normal Form），消除存在量词，再化为合取范式，最后通过合一（Unification）和归结（Resolution）推导出空子句（表示矛盾）。

## 关键结论

- 全称实例化 UI 是最自然的量词消去规则：若"所有人都会死"为真，则"苏格拉底会死"为真
- UG 规则有严格的适用条件：c 必须是"任意选取"的，不能依赖于任何特定假设；若 c 出现在前提或仍有效的假设中，UG 不能使用
- EI 规则要求引入的常元必须是"全新的"（fresh constant），不能与已有常元相同，否则可能产生错误结论
- 一阶逻辑的归结原理是完备的：若 Γ ⊨ φ，则从 Γ ∪ {¬φ} 通过归结可导出空子句。但一阶归结是不可判定的——不存在算法断定任意一阶公式是否可满足（Church-Turing定理）
- 合一（Unification）是使两个表达式在适当替换后变为相等的算法，是一阶归结的核心操作

## 易错点

1. EI 使用后未引入新常元而用了已存在的常元：从 ∃x P(x) 和 ∃x Q(x) 不能推出 ∃x (P(x) ∧ Q(x))，因为两次 EI 必须使用不同的常元
2. UG 使用条件不满足：在假设下引入 c 并推导 P(c) 后，若 c 出现在未消除的假设中，不能使用 UG 得到 ∀x P(x)
3. 忘记处理量词辖域：在应用 UI 或 EG 之前需确保量词的辖域被正确理解和处理

## 例题

**例题1**：用一阶逻辑推理证明"所有人都会死，苏格拉底是人，所以苏格拉底会死"。

形式化：
1. ∀x (Man(x) → Mortal(x))  （前提）
2. Man(Socrates)            （前提）
3. Man(Socrates) → Mortal(Socrates)  （1, UI：将 x 实例化为 Socrates）
4. Mortal(Socrates)         （2, 3, Modus Ponens） ∎

**例题2**：判断以下推理是否有效："有人喜欢所有课程，所以所有课程都有人喜欢。"

形式化：∃x ∀y Likes(x, y) ⊨ ∀y ∃x Likes(x, y)。

证明：
1. ∃x ∀y Likes(x, y)  （前提）
2. ∀y Likes(c, y)      （1, EI，c 为新常元）
3. Likes(c, a)          （2, UI，a 为任意个体）
4. ∃x Likes(x, a)       （3, EG）
5. ∀y ∃x Likes(x, y)    （4, UG，a 是任意的） ∎

推理有效。

## 代码示例

```python
# 一阶逻辑推理规则的Python模拟
class FolProver:
    def __init__(self, domain):
        self.domain = list(domain)
        self.premises = []
    
    def forall(self, pred):
        """∀x P(x)"""
        return all(pred(x) for x in self.domain)
    
    def exists(self, pred):
        """∃x P(x)"""
        return any(pred(x) for x in self.domain)
    
    def universal_instantiation(self, forall_pred, constant):
        """UI: ∀x P(x) ⊨ P(c)"""
        return forall_pred(constant)
    
    def existential_generalization(self, pred_c, constant):
        """EG: P(c) ⊨ ∃x P(x)"""
        return self.exists(pred_c)

# 验证经典三段论: ∀x(Man(x)→Mortal(x)), Man(Socrates) ⊨ Mortal(Socrates)
domain = ["Socrates", "Plato", "Aristotle"]
man = lambda x: x == "Socrates"
mortal = lambda x: True  # 所有人都会死

# 验证前提
premise1_holds = all(not man(x) or mortal(x) for x in domain)
premise2_holds = man("Socrates")
conclusion_holds = mortal("Socrates")

print(f"三段论有效: {premise1_holds and premise2_holds <= conclusion_holds}")
```
