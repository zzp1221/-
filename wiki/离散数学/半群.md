## 核心定义

半群（Semigroup）是最简单的代数系统之一，由一个非空集合 S 和 S 上的一个二元运算 * 组成，记为 (S, *)，该运算满足**结合律**：对任意 a, b, c ∈ S，有 (a * b) * c = a * (b * c)。半群不要求存在单位元，也不要求运算满足交换律。若运算还满足交换律（a * b = b * a），则称为**交换半群**或**阿贝尔半群**（Commutative Semigroup）。若存在元素 e ∈ S 使得对任意 a ∈ S 有 e * a = a * e = a，则 (S, *, e) 称为**独异点**（Monoid / 幺半群）。半群是一个广泛而基础的代数结构：自然数关于加法 (ℕ, +) 是半群也是独异点（0 为单位元），正整数关于加法 (ℤ⁺, +) 是半群但不是独异点（无单位元）。半群理论在形式语言和自动机理论中扮演核心角色——一个语言的句法半群（Syntactic Semigroup）是判断该语言是否为正则语言的关键工具。

## 关键结论

- 结合律要求 a*b*c 的括号可以任意放置而不影响结果，从而可以无歧义地书写多个元素的连乘
- 半群的幂运算：aⁿ = a * a * ... * a（n 次），满足 aᵐ * aⁿ = aᵐ⁺ⁿ 和 (aᵐ)ⁿ = aᵐⁿ
- 任一半群都可以通过添加单位元（如果尚未有）"完备化"为一个独异点
- 有限半群必定包含幂等元（Idempotent：a*a = a），这是半群理论中的一个重要定理（由鸽巢原理证明）
- 半群（及独异点）是范畴论中"单一对象范畴"的代数对应物
- 计算机科学中，字符串的连接运算构成自由独异点；变换半群对应自动机的状态转移

## 易错点

1. 混淆半群与群：半群只要求结合律，群还需要单位元和每个元素的逆元。半群是最"宽松"的代数结构
2. 混淆幺半群（独异点）与群：独异点有单位元但没有逆元——例如 (ℕ, +, 0) 是独异点但不是群（正整数无加法逆元）
3. 认为所有半群都含单位元：只有独异点才保证单位元的存在

## 例题

**例题1**：判断下列代数系统是否为半群 (a) (ℤ, -), (b) (ℤ⁺, max), (c) (M₂(R), ×)。

**解答**：
(a) 否。减法不满足结合律：(1-2)-3 = -4 ≠ 1-(2-3) = 2。
(b) 是。max(max(a,b),c) = max(a,max(b,c))。但无单位元（正整数无最小值），故为半群非独异点。
(c) 是。矩阵乘法满足结合律，有单位矩阵 I，故为独异点。

**例题2**：证明有限半群 S 中必存在幂等元。

**解答**：任取 a ∈ S。由鸽巢原理，无限序列 a, a², a³, ... 在有限集合 S 中必有重复，存在 i < j 使得 aⁱ = aʲ。令 k = j−i ≥ 1，则 aⁱ = aⁱ⁺ᵏ。进而 aⁱ⁺ᵏ = aⁱ⁺²ᵏ = ...。取充分大的 n 使 nk ≥ i，则 aⁿᵏ = aⁿᵏ * aⁿᵏ，即 aⁿᵏ 是幂等元。

## 代码示例

```python
from typing import Callable, TypeVar

T = TypeVar('T')

class Semigroup:
    """半群代数结构的Python表示"""
    def __init__(self, elements: set, op: Callable[[T, T], T]):
        self.elements = elements
        self.op = op
    
    def is_associative(self) -> bool:
        """验证结合律（穷举有限集）"""
        for a in self.elements:
            for b in self.elements:
                for c in self.elements:
                    if self.op(self.op(a, b), c) != self.op(a, self.op(b, c)):
                        return False
        return True
    
    def find_idempotent(self):
        """在有限半群中找幂等元"""
        for a in self.elements:
            x = a
            seen = set()
            while x not in seen:
                seen.add(x)
                x = self.op(x, x)
            if self.op(x, x) == x:
                return x
        return None

# 例：({0,1,2}, max) 是半群
s = Semigroup({0, 1, 2}, max)
print(f"结合律: {s.is_associative()}")  # True
```
