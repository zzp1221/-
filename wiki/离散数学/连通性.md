## 核心定义

连通性（Connectivity）是图的最基本的整体性质，衡量图的"整体性"和"抗毁性"。在无向图 G 中，若任意两个顶点之间存在通路，则称 G 是**连通图**（Connected Graph）；否则称为非连通图（Disconnected Graph）。图的**连通分量**（Connected Component）是极大连通子图——即不能再通过添加顶点或边来保持连通性的子图。在有向图中，连通性有三种级别：(1) **弱连通**（Weakly Connected）：忽略边的方向后得到的无向图是连通的；(2) **单连通/单向连通**（Unilaterally Connected）：对任意 u, v，存在从 u 到 v 或从 v 到 u 的有向通路；(3) **强连通**（Strongly Connected）：对任意 u, v，既存在从 u 到 v 又存在从 v 到 u 的有向通路。关键指标：**点连通度** κ(G) = 使图不连通所需删除的最少顶点数；**边连通度** λ(G) = 使图不连通所需删除的最少边数；Whitney不等式：κ(G) ≤ λ(G) ≤ δ(G)（δ 为最小度数）。**割点**（Articulation Point）是删除后使连通分量数目增加的顶点；**桥**（Bridge）是删除后使连通分量数目增加的边。

## 关键结论

- 一个连通图的生成树即是保持连通性的极小边子集，其边数为 n−1
- 树是边数最少的连通图（n 个顶点恰有 n−1 条边），也是不含圈的极大图
- Kosaraju和Tarjan算法在 O(V+E) 时间内找出有向图的所有强连通分量（SCC）
- Menger定理：图中两个不相邻顶点之间的顶点不交路径的最大数量等于分离这两个顶点所需删除的最少顶点数——这是连通性理论的基石
- 2-连通图（κ ≥ 2）无割点，2-边连通图（λ ≥ 2）无桥

## 易错点

1. 混淆"非连通"与"无边"：空图（无边图）是非连通的（若 |V| > 1），但存在有边也非连通的图
2. 认为割点删除后图一定变为两个连通分量：可能变为三个或更多
3. 有向图的弱连通不等于强连通：一条链式的有向图 v₁→v₂→v₃ 是弱连通的但不是强连通的（v₃ 无法到达 v₁）

## 例题

**例题1**：证明在有 n 个顶点的简单无向图中，若边数超过 (n−1)(n−2)/2，则图必定连通。

**解答**：反证法。假设 G 不连通，则 G 可分为两个非空子图 G₁（k 个顶点）和 G₂（n−k 个顶点，1≤k≤n−1），两者之间没有边。G 的最大可能边数为 C(k,2)+C(n−k,2)。该表达式在 k=1 或 k=n−1 时取得最大值 (n−1)(n−2)/2。故若边数超过此值，G 不可能不连通。证毕。

**例题2**：找出下图中所有的割点和桥：顶点 {A,B,C,D,E}，边 {AB, BC, CD, DE, BE}。

**解答**：该图是一个顶点 B 和 E 有两条路径的类链结构。割点：C（删除后 {A,B,E} 与 {D} 分离）、D（删除后 D... 其实 A,B,C,E 与空...）。桥：CD 和 DE。边 AB, BC, BE 不是桥（因 B-E 存在替代路径 B-C-D-E）。

## 代码示例

```python
def tarjan_scc(graph):
    """Tarjan算法求有向图的强连通分量"""
    index = 0
    stack = []
    on_stack = set()
    indices = {}
    lowlink = {}
    sccs = []
    
    def strongconnect(v):
        nonlocal index
        indices[v] = lowlink[v] = index
        index += 1
        stack.append(v)
        on_stack.add(v)
        
        for w in graph.get(v, []):
            if w not in indices:
                strongconnect(w)
                lowlink[v] = min(lowlink[v], lowlink[w])
            elif w in on_stack:
                lowlink[v] = min(lowlink[v], indices[w])
        
        if lowlink[v] == indices[v]:
            scc = []
            while True:
                w = stack.pop()
                on_stack.remove(w)
                scc.append(w)
                if w == v:
                    break
            sccs.append(scc)
    
    for v in graph:
        if v not in indices:
            strongconnect(v)
    return sccs

g = {0: [1], 1: [2], 2: [0, 3], 3: [4], 4: [3]}
print(f"强连通分量: {tarjan_scc(g)}")
```
