## 核心定义

基数（Cardinality）是对集合"大小"的度量，直观上表示集合中元素的"个数"。对有限集合 A，基数 |A| 是一个非负整数，表示 A 的元素计数。**无穷集合基数的理论**由康托（Georg Cantor）创立：两个集合 A 和 B 具有**相同基数**（等势 equinumerous），记作 |A| = |B| 或 A ≈ B，当且仅当存在从 A 到 B 的双射（一一对应）。若存在从 A 到 B 的单射但无满射，则 |A| < |B|。最小的无穷基数是 ℵ₀（阿列夫零），即自然数集 ℕ 的基数，具有该基数的集合称为**可数无穷集**（Countably Infinite）。实数集 ℝ 的基数为 ℏ（连续统），ℏ = 2^(ℵ₀) > ℵ₀ ——这是**康托定理**：|P(A)| > |A| 的特例。**康托-伯恩斯坦-施罗德定理**（Cantor-Bernstein-Schroeder Theorem）提供了一个证明基数相等的有力工具：若存在 A → B 和 B → A 的单射，则 |A| = |B|。连续统假设（Continuum Hypothesis）断言不存在介于 ℵ₀ 和 ℏ 之间的基数，此命题在 ZFC 公理系统中被证明是不可判定的。

## 关键结论

- 可数集包括：自然数 ℕ、整数 ℤ、有理数 ℚ；不可数集包括：实数 ℝ、[0,1] 区间、无理数集
- ℵ₀ + n = ℵ₀, ℵ₀ × 2 = ℵ₀, ℵ₀ × ℵ₀ = ℵ₀（可数无穷的算术性质）
- 有理数 ℚ 可数是反直觉的但却真实成立的结论（Cantor对角线构造证明）
- 实数集 ℝ 不可数的标准证明是康托对角线论证：假设 [0,1] 可数排列，构造一个新的实数其第 i 位与第 i 个实数的第 i 位不同，产生矛盾
- 康托定理：对任何集合 A，|P(A)| > |A|，这递归地产生了一个无穷递增的基数序列

## 易错点

1. 认为所有无穷集都有相同的基数：整数 ℤ 和实数 ℝ 都是无穷集，但 |ℤ| = ℵ₀ < ℏ = |ℝ|，两者的基数根本不同
2. 误认为可数无穷加可数无穷不等于可数无穷：ℕ ∪ ℤ = ℤ 仍可数，两个可数集的并可数
3. 混淆"基数比较"与"子集关系"：虽然 ℕ ⊂ ℤ，但 |ℕ| = |ℤ|（都存在双射），基数相等不要求"大小直观上一致"

## 例题

**例题1**：证明全体有理数集合 ℚ 是可数的。

**解答**：每个有理数可以唯一表示为最简分数 p/q（q > 0, gcd(p,q) = 1）。将所有有理数按 |p|+q 的大小排列，同等大小的按 p/q 的大小排列，得到一个序列将所有有理数不重复地排列出来，建立了 ℚ 与 ℕ 之间的一一对应，故 ℚ 可数。

**例题2**：利用康托定理证明不存在包含所有集合的"万有集合"。

**解答**（反证法）：假设存在万有集合 U 包含所有集合。则 P(U) ⊆ U（因 U 包含所有集合）。但由康托定理 |P(U)| > |U|，即 P(U) 的基数严格大于 U 的基数，与 P(U) ⊆ U 矛盾（子集基数不可能严格大于母集基数）。故万有集合不存在。

## 代码示例

```python
def cantor_pairing(x: int, y: int) -> int:
    """康托配对函数: ℕ×ℕ → ℕ 的双射，证明 |ℕ×ℕ| = ℵ₀"""
    return (x + y) * (x + y + 1) // 2 + y

# 验证可数性：所有非负整数对可一一对应到自然数
seen = set()
for x in range(100):
    for y in range(100):
        code = cantor_pairing(x, y)
        assert code not in seen  # 单射
        seen.add(code)
print(f"康托配对函数: ℕ×ℕ → ℕ 是单射，验证通过（100×100范围）")

# 对角线论证示意
def diagonal_proof_example():
    """康托对角线论证的直观演示"""
    # 假设 [0,1] 可数，列成表
    numbers = [
        "0.123456...",
        "0.789012...",
        "0.345678...",
    ]
    # 取对角线元素并改变每一位，构造的数不在表中
    print("对角线论证：构造的数不在任何可数排列中")
```
