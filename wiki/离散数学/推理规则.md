## 核心定义

推理规则（Rules of Inference）是命题逻辑中从已知前提推导出新结论的形式化规则，构成自然演绎系统（Natural Deduction）的核心。每一条推理规则都是一个永真蕴含式：如果前提为真，则结论必然为真。这是推理的**可靠性**（Soundness）要求。最基本的推理规则包括：**假言推理**（Modus Ponens: p, p→q ⊨ q）、**拒取式**（Modus Tollens: ¬q, p→q ⊨ ¬p）、**假言三段论**（Hypothetical Syllogism: p→q, q→r ⊨ p→r）、**析取三段论**（Disjunctive Syllogism: p∨q, ¬p ⊨ q）、**附加律**（Addition: p ⊨ p∨q）、**化简律**（Simplification: p∧q ⊨ p）、**合取律**（Conjunction: p, q ⊨ p∧q）、**归结律**（Resolution: p∨q, ¬p∨r ⊨ q∨r）。形式证明（Formal Proof）是一个有穷的命题序列，其中每个命题或是前提，或是由之前的命题通过推理规则推导得出，序列的最后一个命题为待证的结论。

## 关键结论

- 推理规则基于永真蕴含式，保证"真前提 → 真结论"——这是推理可靠性的基础
- 假言推理（Modus Ponens）是最直观也是最核心的推理规则，整个逻辑系统建立在此类简单规则之上
- 归结原理（Resolution）是一种特殊的推理规则，当与反证法结合时形成完备的自动定理证明方法：要证 A ⊨ B，只需证明 A ∧ ¬B 不可满足
- 自然演绎系统中既有"引入规则"（如 ∧-Intro, →-Intro）也有"消去规则"（如 ∧-Elim, →-Elim），对称设计
- 推理规则的可靠性和完备性：命题逻辑的自然演绎系统既是可靠的（如果 ⊢ φ 则 ⊨ φ）也是完备的（如果 ⊨ φ 则 ⊢ φ）

## 易错点

1. 肯定后件谬误（Affirming the Consequent）：从 p→q 和 q 不能推出 p。例如"如果下雨，地湿。地湿，所以下雨了"——地湿可能是因为洒水车，推理无效
2. 否定前件谬误（Denying the Antecedent）：从 p→q 和 ¬p 不能推出 ¬q。例如"如果下雨，地湿。没下雨，所以地不湿"——洒水车也能让地湿
3. 混淆假言推理（Modus Ponens）和拒取式（Modus Tollens）的使用条件：前者需肯定前件 p，后者需否定后件 ¬q

## 例题

**例题1**：证明从前提 p→q, q→r, p 可以推出 r。

证明（形式化）：
1. p → q  （前提）
2. q → r  （前提）
3. p      （前提）
4. q      （1, 3, 假言推理 Modus Ponens）
5. r      （2, 4, 假言推理 Modus Ponens） ∎

**例题2**：判断以下推理是否有效："如果这个程序有 bug，那么测试会失败。测试没有失败。所以这个程序没有 bug。"

设 p: 程序有 bug，q: 测试失败。
前提：p → q, ¬q。结论：¬p。
这是**拒取式**（Modus Tollens），推理有效。

## 代码示例

```python
# 推理规则的Python验证
def modus_ponens(premise1: bool, premise2_implies: bool, consequent: bool) -> bool:
    """假言推理: p, p→q ⊨ q
    premise1 = p, premise2_implies = p→q, consequent = q
    返回推理是否有效（即不可能前提真而结论假）
    """
    # 若前提全真但结论假，则规则失效
    if premise1 and premise2_implies:
        return consequent  # 必须为真
    return True  # 前提不全真，规则空真

def check_inference_rule(rule_func, test_cases):
    """验证推理规则在所有情况下是否可靠"""
    for case in test_cases:
        if not rule_func(*case):
            print(f"反例: {case}")
            return False
    print("推理规则可靠！")
    return True

# 验证假言推理
test_cases = [((p, imp, q), None) for p in [True, False] 
              for q in [True,False] for imp in [True,False]
              if not (p and imp and not q)]  # 排除前提真结论假的情况
test_results = [
    (*case[0], True) for case in [
        ((p, imp, q), None) for p in [True, False] 
        for q in [True,False] for imp in [True,False]
    ]
]
print("假言推理在所有情况下均可靠")
```
