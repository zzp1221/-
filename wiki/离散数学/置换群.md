## 核心定义

置换群（Permutation Group）是以双射映射（置换）作为元素、以函数复合作为运算的群，是群论中最具体也最重要的例子之一。n 个元素的全体置换在复合运算下构成**对称群**（Symmetric Group）Sₙ，其阶为 |Sₙ| = n!。每个置换 σ ∈ Sₙ 可以以下列方式表示：(1) **双行记号**：第一行列定义域，第二行列像；(2) **轮换记号**（Cycle Notation）：将置换分解为不相交轮换的乘积，如 (1 2 3)(4 5) 表示 1→2→3→1, 4→5→4；(3) **对换分解**：每个轮换可进一步分解为对换（长度为2的轮换）之积。置换的**奇偶性**（Parity）：一个置换可以表为偶数个对换之积时称为偶置换（Even Permutation），否则为奇置换（Odd Permutation）。所有偶置换构成**交代群**（Alternating Group）Aₙ ≤ Sₙ，|Aₙ| = n!/2。Cayley定理是群论的基石：任何群同构于某个置换群的子群——即每个抽象群都可以具体地实现为对称群的子群，这为所有有限群提供了统一的表示框架。

## 关键结论

- |Sₙ| = n!，是 n 个元素的所有可能排列数
- 轮换的阶等于轮换的长度：k-轮换的阶为 k
- Sn 中每个置换可以**唯一地**表示为不相交轮换的乘积（不计顺序）
- 偶置换的对换分解必含偶数个对换，奇置换的对换分解必含奇数个对换
- Aₙ 在 Sₙ 中的指数为 2，故 Aₙ ⊴ Sₙ（指数2子群必为正规子群）
- Cayley定理确立了置换群在群论中的核心地位：研究抽象群等价于研究置换群的子群
- n ≥ 3 时 Sₙ 非阿贝尔；n ≥ 5 时 Aₙ 是单群（无非平凡正规子群）

## 易错点

1. 轮换 (1 2 3) 不能写为 (1 2 3)(1 2 3) 这样的重复——一个置换的轮换分解中每个元素恰出现在一个轮换中
2. 混淆轮换的乘积顺序：不同文献采用不同的复合惯例（先右后左 vs 先左后右），需统一标准
3. 单位置换的轮换表示：单位置换（恒等映射）的轮换表示为 ()

## 例题

**例题1**：将置换 σ = (1→3, 2→1, 3→4, 4→2) 写成轮换的乘积并求其阶。

**解答**：追踪：1→3→4→2→1，构成一个 4-轮换 (1 3 4 2)。故此置换是一个 4-轮换，阶为 4。σ = (1 3 4 2)。

**例题2**：S₄ 中有多少个 3 阶元素？

**解答**：3 阶元素恰为 3-轮换。S₄ 中 3-轮换的数量：选 3 个元素的方法 C(4,3)=4，每个 3 元集上有 (3-1)!=2 个不同的 3-轮换。故共 4×2=8 个 3 阶元素。

## 代码示例

```python
from itertools import permutations

def symmetric_group(n):
    """生成对称群S_n的所有元素"""
    elements = list(range(n))
    return [list(p) for p in permutations(elements)]

def cycle_decomposition(perm):
    """将置换分解为不相交轮换"""
    n = len(perm)
    visited = [False] * n
    cycles = []
    for i in range(n):
        if not visited[i]:
            cycle = [i]
            visited[i] = True
            j = perm[i]
            while j != i:
                cycle.append(j)
                visited[j] = True
                j = perm[j]
            if len(cycle) > 1:
                cycles.append(tuple(cycle))
    return cycles

# S_3: 置换群
S3 = symmetric_group(3)
print(f"|S₃| = {len(S3)}")  # 6

for perm in S3:
    cycles = cycle_decomposition(perm)
    parity = sum(len(c)-1 for c in cycles) % 2
    print(f"{perm} -> {cycles}, {'偶' if parity==0 else '奇'}置换")
```
