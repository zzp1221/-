## 核心定义

子群（Subgroup）是群的子代数结构。设 (G, *) 是一个群，H ⊆ G 是 G 的非空子集。若 H 在 G 的运算 * 下自身构成一个群，则称 H 是 G 的子群，记作 H ≤ G。{e}（仅含单位元的平凡子群）和 G 自身是任何群 G 的**平凡子群**，其他所有子群称为**非平凡子群**（Proper Subgroup 记为 H < G）。子群判定定理提供了高效的检验方法：(1) **一步判定法**：H ≤ G 当且仅当 H 非空且 ∀a,b∈H, a*b⁻¹ ∈ H；(2) **两步判定法**：H ≤ G 当且仅当 H 非空、对运算封闭、且每个元素的逆元也在 H 中。对有限群还有简便的判定方法：有限群 G 的非空有限子集 H 是子群当且仅当 H 对 G 的运算封闭。**循环子群** ⟨a⟩ = {aⁿ | n ∈ ℤ} 是由元素 a 生成的子群，其阶等于 a 的阶。**生成子群** ⟨S⟩ 是由集合 S ⊆ G 生成的包含 S 的最小子群。

## 关键结论

- Lagrange定理是子群理论的基石：若 H ≤ G 且 |G| 有限，则 |H| 整除 |G|；商 |G|/|H| 称为 H 在 G 中的指数 [G:H]
- 素数阶群必为循环群且没有非平凡子群
- 子群的交仍是子群（任意多个子群的交集仍是子群）；子群的并不一定是子群（事实上 H ∪ K 是子群当且仅当 H ⊆ K 或 K ⊆ H）
- 循环群的每个子群都是循环的；n 阶循环群 Cₙ 中，对每个 d|n，恰好有一个 d 阶子群
- 对称群 S₄ 的子群格结构已相当复杂，体现了子群关系的层次性
- 群 G 上全体子群按包含关系构成完备格

## 易错点

1. 忘记验证子群非空：空集 ∅ 不是子群（没有单位元），"非空"是子群判定的必要条件
2. 误认为子群必须包含所有使群运算封闭的子集：对无限群，封闭子集不一定是子群——例如 (ℕ, +) ⊆ (ℤ, +) 对加法封闭但无逆元故非子群
3. 混淆真正规子群（Normal Subgroup）与普通子群：所有子群不一定都是正规子群

## 例题

**例题1**：求 (ℤ₁₂, +₁₂) 的所有子群。

**解答**：ℤ₁₂ 是 12 阶循环群。对每个 d|12，恰有一个 d 阶子群：
d=1: ⟨0⟩ = {0}
d=2: ⟨6⟩ = {0, 6}
d=3: ⟨4⟩ = {0, 4, 8}
d=4: ⟨3⟩ = {0, 3, 6, 9}
d=6: ⟨2⟩ = {0, 2, 4, 6, 8, 10}
d=12: ⟨1⟩ = ℤ₁₂

**例题2**：设 H 和 K 是群 G 的子群。证明 H ∩ K 也是 G 的子群。

**解答**：用一步判定法。(1) e ∈ H 且 e ∈ K，故 e ∈ H∩K，非空。(2) 对任意 a,b ∈ H∩K，a,b ∈ H 故 a*b⁻¹ ∈ H；a,b ∈ K 故 a*b⁻¹ ∈ K。因此 a*b⁻¹ ∈ H∩K。故 H∩K ≤ G。

## 代码示例

```python
def is_subgroup(G_elements, G_op, H_subset, G_identity):
    """有限群子群判定：检查封闭性"""
    if G_identity not in H_subset:
        return False
    for a in H_subset:
        for b in H_subset:
            if G_op(a, b) not in H_subset:
                return False
    return True

def cyclic_subgroup(a, G_op, G_identity, max_order=100):
    """生成循环子群 ⟨a⟩"""
    subgroup = {G_identity}
    current = a
    for _ in range(max_order):
        subgroup.add(current)
        if current == G_identity:
            break
        current = G_op(current, a)
    return subgroup

# ℤ₁₂ 中 ⟨3⟩ 的循环子群
Z12_op = lambda a, b: (a + b) % 12
sg = cyclic_subgroup(3, Z12_op, 0)
print(f"⟨3⟩ in ℤ₁₂ = {sorted(sg)}")  # {0, 3, 6, 9}
```
