## 核心定义

生成树（Spanning Tree）是连通图 G 的一个子图，包含 G 的所有顶点，且本身是一棵树。一个连通图至少有一棵生成树（可通过 BFS/DFS 构造）；生成树恰有 n−1 条边，是保持连通性的极小边子集。**生成森林**（Spanning Forest）是非连通图的类似概念——每个连通分量取一棵生成树。**最小生成树**（Minimum Spanning Tree, MST）使问题进一步：在边带有权重的加权连通图中，求一棵生成树使得其所有边的权重之和最小。经典算法包括：(1) **Kruskal 算法**：贪心法，从小到大考察每条边，若该边连接两个不同的连通分量则加入 MST，使用并查集（Union-Find）实现，复杂度 O(E log E)；(2) **Prim 算法**：从一个顶点出发，每次选择当前树与外部顶点之间的最小权边加入，复杂度 O(E log V)（二叉堆实现）；(3) **破圈法**：不断删除图中权最大的边（只要保持连通性），直到剩下 n−1 条边。MST 在通信网络设计、电路布线、聚类分析等领域有广泛应用。

## 关键结论

- 任何连通图至少有一棵生成树
- 若所有边权互不相同，则 MST 唯一；若有相同边权，则 MST 可能不唯一
- MST 的割性质（Cut Property）：任何割的最小权边必定属于某一棵 MST
- MST 的圈性质（Cycle Property）：任何圈的最大权边不属于任何 MST
- Kruskal 和 Prim 算法都正确且能输出一棵 MST——证明基于割性质和圈性质
- MST 问题的贪心选择性质是拟阵理论（Matroid Theory）的经典实例

## 易错点

1. 误用 Prim 算法于非连通图：Prim 和 Kruskal 都要求原图是连通的，对于非连通图，"最小生成树"无定义（应改为最小生成森林）
2. 混淆 Kruskal 和 Prim 的边选择过程：Kruskal 按全局边权排序，Prim 按当前树邻接的最小边扩展
3. MST 不保证两点间路径最短：MST 最小化总权，不是最小化任意两点间的距离（那是最短路径树的任务）

## 例题

**例题1**：加权图中边权如下：A-B:4, A-C:2, B-C:1, B-D:5, C-D:8, C-E:10, D-E:2, D-F:6, E-F:3。用 Kruskal 算法求 MST。

**解答**：按权排序边：1(B-C), 2(A-C), 2(D-E), 3(E-F), 4(A-B), 5(B-D), 6(D-F), 8(C-D), 10(C-E)。
- 选 B-C(1), 分量{BC}
- 选 A-C(2), {ABC}
- 选 D-E(2), {ABC}{DE}
- 选 E-F(3), {ABC}{DEF}
- 选 A-B(4) 构成圈 A-B-C-A，跳过
- 选 B-D(5), {ABCDEF}，6个顶点5条边，MST 完成。总权 = 1+2+2+3+5 = 13。

**例题2**：证明 MST 的唯一性条件：若加权连通图中所有边权互不相同，则 MST 唯一。

**解答**：反证法。设 T₁ 和 T₂ 是两棵不同的 MST，取 T₁ 中有但 T₂ 中无的最轻边 e。将 e 加入 T₂ 产生圈，该圈中必存在边 f 属于 T₂ 但不在 T₁ 中。由于所有边权互异，w(e) ≠ w(f)。若 w(e) < w(f)，则 T₂ − f + e 的权更小，与 T₂ 为 MST 矛盾。若 w(e) > w(f)，考虑从 T₁ 删除 e 并加入 f，可得更小权的生成树，与 T₁ 为 MST 矛盾。故 T₁ = T₂。

## 代码示例

```python
def kruskal(vertices, edges):
    """edges = [(w, u, v), ...], 返回MST的边列表"""
    parent = {v: v for v in vertices}
    rank = {v: 0 for v in vertices}
    
    def find(v):
        if parent[v] != v:
            parent[v] = find(parent[v])
        return parent[v]
    
    def union(u, v):
        ru, rv = find(u), find(v)
        if ru == rv:
            return False
        if rank[ru] < rank[rv]:
            parent[ru] = rv
        elif rank[ru] > rank[rv]:
            parent[rv] = ru
        else:
            parent[rv] = ru
            rank[ru] += 1
        return True
    
    mst = []
    for w, u, v in sorted(edges):
        if union(u, v):
            mst.append((w, u, v))
        if len(mst) == len(vertices) - 1:
            break
    return mst

V = {'A','B','C','D','E','F'}
E = [(4,'A','B'),(2,'A','C'),(1,'B','C'),(5,'B','D'),(8,'C','D'),(10,'C','E'),(2,'D','E'),(6,'D','F'),(3,'E','F')]
print(f"MST: {kruskal(V, E)}")
```
