## 核心定义

谓词（Predicate）是一阶逻辑（First-Order Logic）中的核心概念，是对命题逻辑中"原子命题"的推广。一个谓词是一个以个体变元（Individual Variable）为参数的命题函数 P(x₁, x₂, ..., xₙ)，当用具体的个体常元替代变元后，得到一个具有确定真值的命题。谓词刻画了个体的**性质**（一元谓词，如"x 是质数"）或个体之间的**关系**（多元谓词，如"x 大于 y"）。一个谓词的**元数**（Arity）是该谓词所含个体变元的个数：一元谓词表性质，二元谓词表二元关系，n 元谓词表 n 元关系。个体常元代表论域中具体的对象。谓词加上量词（全称量词 ∀、存在量词 ∃）构成一阶逻辑的表达骨架。谓词逻辑的表达能力远超命题逻辑：命题逻辑只能表达"苏格拉底是人""所有人都会死"为两个独立原子命题，而谓词逻辑能将两者关联起来并推导出"苏格拉底会死"。

## 关键结论

- 谓词本身没有真值——只有当代入具体个体或施加量词约束后才成为命题
- n 元谓词在给定论域 D 上的语义是一个从 Dⁿ 到 {T, F} 的函数，即 D 上的一个 n 元关系
- 特性谓词（Characteristic Predicate）用于在多元谓词逻辑中限定论域子集（如"对于所有人 x"需写作 ∀x(Man(x) → ...)）
- 谓词逻辑的表达能力足以形式化大部分数学命题（因此称为"一阶"——量词只作用于个体变元，不作用于谓词变元）
- 一阶逻辑中，谓词字母不能作为变元被量化；若允许量化谓词，则得到二阶逻辑

## 易错点

1. 将谓词本身当作命题使用：P(x) 不是命题（真值依赖于 x），∀x P(x) 或 P(a) 才是命题
2. 混淆谓词与函数：谓词返回真值（T/F），函数返回论域中的个体；谓词 P(a) 是有真值的陈述，函数 f(a) 是个体词项
3. 在全称量词后错误使用合取：∀x (Man(x) ∧ Mortal(x)) 表示"所有东西都是人且都会死"；正确的表达应为 ∀x (Man(x) → Mortal(x))

## 例题

**例题1**：将下列语句形式化为一阶逻辑公式。
论域：所有人。谓词：L(x): x 是学生; H(x): x 努力; P(x): x 通过考试。
(a) 所有努力的学生都通过了考试。
(b) 有些学生很努力但没有通过考试。
(c) 并非所有学生都通过了考试。

**解答**：
(a) ∀x ((L(x) ∧ H(x)) → P(x))
(b) ∃x (L(x) ∧ H(x) ∧ ¬P(x))
(c) ¬∀x (L(x) → P(x)) 或等价地 ∃x (L(x) ∧ ¬P(x))

**例题2**：分析一元谓词与二元谓词在表达力上的差异，举例说明。

**解答**：一元谓词只能描述个体自身的性质，二元谓词可以描述个体间的关系。例如，要表达"每个人都有一个朋友"，仅靠一元谓词 Friend(x) 无法表达"谁是谁的朋友"——需要二元谓词 FriendOf(x, y) 表示"x 是 y 的朋友"，然后写为 ∀x ∃y FriendOf(y, x)。

## 代码示例

```python
# 谓词的Python表示
from typing import Callable, TypeVar, List

T = TypeVar('T')

# 一元谓词: D -> bool
Predicate1 = Callable[[T], bool]
# 二元谓词: D x D -> bool
Predicate2 = Callable[[T, T], bool]

# 示例：论域为自然数
domain = range(1, 11)

# 一元谓词: "是质数"
def is_prime(x: int) -> bool:
    if x < 2:
        return False
    for i in range(2, int(x**0.5) + 1):
        if x % i == 0:
            return False
    return True

# 二元谓词: "x 整除 y"
def divides(x: int, y: int) -> bool:
    return y % x == 0

# 验证全称命题: ∀x (is_prime(x) → x >= 2) 在论域中为真
result = all(not is_prime(x) or x >= 2 for x in domain)
print(f"∀x (Prime(x) → x>=2) 为: {result}")
```
