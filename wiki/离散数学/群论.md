## 核心定义

群（Group）是抽象代数中最核心的结构。一个群 (G, *) 是一个配备了二元运算 * 的非空集合 G，满足四个公理：(1) **封闭性**：∀a,b∈G, a*b ∈ G；(2) **结合律**：∀a,b,c∈G, (a*b)*c = a*(b*c)；(3) **单位元**：∃e∈G, ∀a∈G, e*a = a*e = a；(4) **逆元**：∀a∈G, ∃a⁻¹∈G, a*a⁻¹ = a⁻¹*a = e。换言之，群是一个每个元素都可逆的独异点。若运算还满足交换律（a*b = b*a），则称为**阿贝尔群**（Abelian Group）或交换群。群的阶（Order）|G| 是其元素个数；有限群是元素个数有限的群。群的生成元是一组能通过运算生成整个群的元素。群论是数学的"语言"，统一处理对称性——任何对称都可以用一个群来描述。在计算机科学中，群论应用于密码学（如椭圆曲线密码基于有限群）、编码理论、计算复杂性（图同构问题与群论紧密相关）和自动机理论等。

## 关键结论

- 单位元唯一：若 e 和 e' 都是单位元，则 e = e*e' = e'
- 每个元素的逆元唯一：若 b 和 c 都是 a 的逆元，则 b = b*e = b*(a*c) = (b*a)*c = e*c = c
- (a⁻¹)⁻¹ = a 且 (a*b)⁻¹ = b⁻¹*a⁻¹（注意顺序反转）
- 群的消去律：若 a*b = a*c 则 b = c（左消去）；若 b*a = c*a 则 b = c（右消去）
- Lagrange定理是有限群论的基石：子群 H 的阶 |H| 整除母群 G 的阶 |G|
- 小阶群分类：1阶群（平凡群），2-5阶群中最简单的是循环群，最小非阿贝尔群为对称群 S₃（6阶）

## 易错点

1. "封闭性"有时被作为运算定义的一部分：若 * 是 G 上的二元运算，则封闭性自动满足；但有时运算定义在更大的集合上，则需要单独验证封闭性
2. 阿贝尔群中 (a*b)⁻¹ = a⁻¹*b⁻¹ 成立（因为交换），但非阿贝尔群中顺序必须反转
3. 混淆群的"阶"与元素的"阶"：群的阶 |G| 是元素总数，元素 a 的阶 ord(a) 是使 aⁿ = e 的最小正整数 n

## 例题

**例题1**：验证 (ℤ₆, +₆)（模 6 加法）是阿贝尔群，列出每个元素的阶。

**解答**：封闭：任何两数模6和仍为0-5。结合律：加法结合律遗传。单位元：0。逆元：0⁻¹=0, 1⁻¹=5, 2⁻¹=4, 3⁻¹=3, 4⁻¹=2, 5⁻¹=1。交换：加法交换律。元素阶：ord(0)=1, ord(1)=ord(5)=6, ord(2)=ord(4)=3, ord(3)=2。|G|=6为阿贝尔群（循环群，生成元为1或5）。

**例题2**：证明群 G 中方程 a*x = b 有唯一解 x = a⁻¹*b。

**解答**：存在性：代入 x = a⁻¹*b，a*(a⁻¹*b) = (a*a⁻¹)*b = e*b = b。唯一性：若 a*x₁ = a*x₂ = b，左乘 a⁻¹ 得 x₁ = x₂。故解唯一。

## 代码示例

```python
class FiniteGroup:
    def __init__(self, elements, op, identity):
        self.elements = set(elements)
        self.op = op
        self.e = identity
    
    def is_closed(self):
        return all(self.op(a, b) in self.elements 
                   for a in self.elements for b in self.elements)
    
    def is_associative(self):
        return all(self.op(self.op(a,b), c) == self.op(a, self.op(b,c))
                   for a in self.elements for b in self.elements for c in self.elements)
    
    def has_inverses(self):
        for a in self.elements:
            has_inv = any(self.op(a, b) == self.e and self.op(b, a) == self.e
                         for b in self.elements)
            if not has_inv:
                return False
        return True
    
    def is_group(self):
        return self.is_closed() and self.is_associative() and self.has_inverses()

# ℤ₄ 模4加法群
Z4 = FiniteGroup(
    elements=[0, 1, 2, 3],
    op=lambda a, b: (a + b) % 4,
    identity=0
)
print(f"ℤ₄是群: {Z4.is_group()}")  # True
```
