## 核心定义

函数（Function）是集合论中从定义域（Domain）到值域（Codomain）的一个特殊二元关系 f ⊆ A × B，满足：对于每个 a ∈ A，**恰好存在一个** b ∈ B 使得 (a, b) ∈ f——即每一个输入有唯一确定的输出，记作 f: A → B, f(a) = b。函数的三种重要性质定义了映射的不同"程度"：(1) **单射**（Injective / One-to-One）：不同输入映射到不同输出，即 ∀a₁,a₂∈A, f(a₁) = f(a₂) → a₁ = a₂；(2) **满射**（Surjective / Onto）：值域中的每一个元素都被映射到，即 ∀b∈B, ∃a∈A, f(a) = b，此时值域 = 陪域；(3) **双射**（Bijective）：既单且满——定义域与陪域之间存在完美的"一一对应"。双射是集合等势（具有相同基数）的形式化定义：|A| = |B| 当且仅当存在从 A 到 B 的双射。函数的像（Image）f(A) = {f(a) | a ∈ A} ⊆ B；原像（Preimage）f⁻¹(T) = {a ∈ A | f(a) ∈ T}。

## 关键结论

- 单射的条件 |A| ≤ |B|（对有限集），满射的条件 |A| ≥ |B|，双射的条件 |A| = |B|
- 单射 f: A → B 的逆关系（从 B 到 A 看）不一定满足函数定义（可能某个 b 没有原像），但若限制陪域为 f(A)（满射化），则导出双射
- 两个单射的复合是单射；两个满射的复合是满射；两个双射的复合是双射
- 双射存在逆函数 f⁻¹: B → A，满足 f⁻¹(f(a)) = a 且 f(f⁻¹(b)) = b
- 鸽巢原理（Pigeonhole Principle）的集合论表述：若 |A| > |B|，则不存在从 A 到 B 的单射

## 易错点

1. 混淆"分量唯一"与"像唯一"：函数的定义要求每个 a 恰有一个 b（而非每个 b 恰有一个 a），后者只在双射时对逆函数成立
2. 将满射条件与"每个 b 只有一个原像"混淆：满射只要求每个 b 至少有一个原像（不要求唯一）
3. 认为所有函数都有逆函数：仅双射才存在定义在全陪域上的逆函数；单射函数可以限制在像集上定义逆

## 例题

**例题1**：判断下列从 ℝ 到 ℝ 的函数类型：(a) f(x) = 2x+1, (b) f(x) = x², (c) f(x) = x³。

**解答**：
(a) 单射（若 2x₁+1 = 2x₂+1 → x₁ = x₂），满射（对任意 y，x = (y-1)/2 ∈ ℝ），故为双射。
(b) 既非单射（f(1) = f(-1) = 1）也非满射（y = -1 无原像）。
(c) 单射（x₁³ = x₂³ → x₁ = x₂），满射（对任意 y，x = ∛y ∈ ℝ），故为双射。

**例题2**：设 |A| = 5, |B| = 3。问：(a) 从 A 到 B 有多少个函数？(b) 其中有多少个满射？(c) 有多少个单射？

**解答**：
(a) 3^5 = 243（每个 A 中元素有 3 种选择）。
(b) 由容斥原理：满射数 = 3^5 − C(3,1)·2^5 + C(3,2)·1^5 = 243 − 96 + 3 = 150。
(c) 0（|A| > |B|，鸽巢原理，不可能有单射）。

## 代码示例

```python
def is_injective(f, domain):
    """判断函数f是否单射"""
    seen = set()
    for x in domain:
        y = f(x)
        if y in seen:
            return False
        seen.add(y)
    return True

def is_surjective(f, domain, codomain):
    """判断函数f是否满射"""
    image = {f(x) for x in domain}
    return image == set(codomain)

def is_bijective(f, domain, codomain):
    """判断函数f是否双射"""
    return is_injective(f, domain) and is_surjective(f, domain, codomain)

# 测试
def f1(x): return 2 * x + 1
domain = range(10)
codomain = [2*x+1 for x in domain]  # 恰好为像集
print(f"f(x)=2x+1 单射: {is_injective(f1, domain)}, 满射: {is_surjective(f1, domain, codomain)}")
```
