## 核心定义

**格**（Lattice）可以从两种等价视角定义：(1) **偏序格**：偏序集 (L, ≤) 中任意两个元素 a, b 都有**上确界**（最小上界，记作 a∨b 或 join）和**下确界**（最大下界，记作 a∧b 或 meet），称为格；(2) **代数格**：集合 L 上定义两个满足幂等律、交换律、结合律和吸收律的二元运算 ∨ 和 ∧，称为代数格。两种定义等价。常见的格包括：整除格 (ℕ⁺, |)、幂集格 (P(S), ⊆)、子群格等。格的几个重要子类：(a) **分配格**（Distributive Lattice）：∨ 和 ∧ 互相满足分配律；(b) **有界格**（Bounded Lattice）：存在最大元 1 和最小元 0；(c) **有补格**（Complemented Lattice）：有界格中每个元素有补元（a∧a'=0, a∨a'=1）；(d) **布尔代数**（Boolean Algebra）：有补分配格，元素与补元一一对应。布尔代数满足De Morgan律和其他经典逻辑代数定律。布尔代数 B 上的布尔函数是 Bⁿ → B 的映射，可以表示为析取范式（DNF）或合取范式（CNF）。

## 关键结论

- 有限格的 Hasse 图唯一确定格的结构，从 Hasse 图可直观验证格的性质
- 分配格的判定条件：不含 M₃（钻石格）或 N₅（五角格）子格（Birkhoff刻画定理）
- 布尔代数 B 中 a ≤ b 当且仅当 a∧b' = 0（或等价条件），布尔代数对应补分配格
- 有限布尔代数必同构于某个有限集合的幂集代数 (P(S), ∪, ∩, ᶜ)，因此 |B| = 2ⁿ
- 布尔函数可以用卡诺图（Karnaugh Map）进行简化，是数字电路逻辑设计的基础
- 布尔代数在逻辑电路和计算机硬件设计中有不可替代的地位——CPU 中所有组合逻辑都是布尔函数

## 易错点

1. 混淆格的上确界和极大元：上确界 a∨b 是 {a, b} 的最小上界，而极大元是全局概念——前者的上界需同时 ≥ a 和 ≥ b
2. 分配格中补元若存在则唯一：∀x∈L，若 ∃x' 使 x∧x'=0 且 x∨x'=1，则再有一个 y 同样满足条件，必有 x' = y（由分配律可证）
3. 不是所有格都是布尔代数：布尔代数 = 有补分配格，分配性是关键条件

## 例题

**例题1**：判断偏序集 (P({a,b,c}), ⊆) 的类型。

**解答**：该偏序集有 8 个元素。任两个子集的上确界是它们的并集，下确界是它们的交集，故为格。它有界（最大元{a,b,c}，最小元∅），每个子集 X 的补元为 Xᶜ，满足有补条件。∪ 和 ∩ 互相分配，故为分配格。综上，(P({a,b,c}), ⊆) 是**布尔代数**。

**例题2**：证明在任意格中，若 a ≤ b，则 a∨c ≤ b∨c 且 a∧c ≤ b∧c（单调性）。

**解答**：由 a ≤ b 知 a∨b = b。需证 a∨c ≤ b∨c。由于 b ≤ b∨c 且 c ≤ b∨c，只需证 a ≤ b∨c——而 a ≤ b ≤ b∨c，故成立。所以 a∨c ≤ b∨c。下确界部分类似：a∧c = a∧c，需 ≤ b∧c，由 a ≤ b 可推 a∧c ≤ b∧c。

## 代码示例

```python
class BooleanAlgebra:
    """幂集布尔代数的Python实现"""
    def __init__(self, universal_set):
        self.elements = universal_set
    
    def join(self, a, b):
        """上确界: 并集"""
        return a | b
    
    def meet(self, a, b):
        """下确界: 交集"""
        return a & b
    
    def complement(self, a):
        """补元: 差集"""
        return self.elements - a
    
    def zero(self):
        return set()
    
    def one(self):
        return self.elements
    
    def verify_laws(self):
        """验证布尔代数定律"""
        subsets = self._all_subsets()
        for a in subsets:
            for b in subsets:
                # 验证 De Morgan: (a∪b)^c = a^c ∩ b^c
                assert self.complement(self.join(a,b)) == \
                       self.meet(self.complement(a), self.complement(b))
        return True
    
    def _all_subsets(self):
        from itertools import combinations
        all_subs = []
        for r in range(len(self.elements)+1):
            all_subs.extend(set(c) for c in combinations(self.elements, r))
        return all_subs

ba = BooleanAlgebra({1, 2, 3})
print(f"布尔代数定律验证通过: {ba.verify_laws()}")
```
