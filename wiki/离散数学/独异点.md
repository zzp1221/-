## 核心定义

独异点（Monoid），又称幺半群，是一个带有单位元（幺元，Identity Element）的半群。形式化地，独异点 (M, *, e) 由一个非空集合 M、M 上的一个满足结合律的二元运算 *、以及一个特殊元素 e ∈ M（单位元）组成，满足对任意 a ∈ M 有 e * a = a * e = a。独异点在半群的基础上增加了"空的运算"或"零步操作"的概念。独异点在计算机科学中处处可见：(1) 字符串的连接运算和空串 ε 构成**自由独异点** Σ*（Kleene星号）；(2) 自然数关于加法 (ℕ, +, 0) 是独异点；(3) 某个集合上的全体自映射关于函数复合和恒等映射构成独异点；(4) 布尔值关于合取 ∧ 和 True 构成独异点，关于析取 ∨ 和 False 也构成独异点。独异点的子代数称为**子独异点**（必须包含原来的单位元）。独异点的同态保持单位元：若 φ: M→N 是独异点同态，则 φ(e_M) = e_N。

## 关键结论

- 独异点 = 半群 + 单位元：单位元是唯一确定的存在（若存在单位元则必然唯一）
- 独异点中可定义元素的**非负整数次幂**：a⁰ = e, aⁿ⁺¹ = aⁿ * a
- 独异点的可逆元素（有逆元的元素）组成该独异点中的最大子群，称为独异点的**单位群**（Group of Units）
- 自由独异点 Σ*（字母表 Σ 上所有有限字符串在连接运算下）具有通用性质：任何从 Σ 到独异点 M 的映射可唯一扩张为从 Σ* 到 M 的同态
- BNF（Backus-Naur Form）文法产生的语言恰为自由独异点的特定子集
- 独异点在范畴论中定义为只有一个对象的范畴，独异点的元素对应范畴中的态射

## 易错点

1. 混淆独异点与群：独异点的元素不一定有逆元——例如 (ℕ, +, 0) 中的正整数 5，不存在 x 使 5+x = 0
2. 在子独异点中遗漏单位元：子独异点必须包含原独异点的单位元，这与子半群的定义不同
3. 混淆空集上定义的独异点：空集上不能定义独异点（因为单位元必须存在），但可以定义空半群（trivial的情况需注意）

## 例题

**例题1**：证明在独异点 (M, *, e) 中，单位元是唯一的。

**解答**：设 e₁ 和 e₂ 都是单位元。由 e₁ 是单位元：e₁ * e₂ = e₂。由 e₂ 是单位元：e₁ * e₂ = e₁。故 e₁ = e₂，证毕。

**例题2**：设 Σ = {a, b}。描述自由独异点 Σ* 的结构，并说明它不是群。

**解答**：Σ* = {ε, a, b, aa, ab, ba, bb, aaa, ...}（所有有限字符串，ε 为空串）。运算为字符串连接。单位元为 ε。Σ* 不是群因为非空字符串没有逆元：不存在串 x 使 a·x = ε（连接不可"撤销"）。

## 代码示例

```python
class Monoid:
    """独异点的Python表示"""
    def __init__(self, elements, op, identity):
        self.elements = elements
        self.op = op
        self.identity = identity
    
    def power(self, a, n):
        """a的n次幂 (n >= 0)"""
        result = self.identity
        for _ in range(n):
            result = self.op(result, a)
        return result
    
    def is_identity(self, e):
        return all(self.op(e, a) == a and self.op(a, e) == a 
                   for a in self.elements)

# 字符串独异点 (Σ* over {0,1})
def concat(s, t): return s + t
sigma_star = {"", "0", "1", "00", "01", "10", "11", "000",
              "001", "010", "011", "100", "101", "110", "111"}

print(f"空串是单位元: {all(concat('', s) == s and concat(s, '') == s for s in sigma_star)}")
# True
```
