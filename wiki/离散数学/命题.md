## 核心定义

命题（Proposition）是命题逻辑中最基本的语义单元，指一个能够唯一判断真假的**陈述句**。一个命题必须是**非真即假**的，不能同时为真又为假（矛盾律），也不能既不为真又不为假（排中律）。命题的真值（Truth Value）只有两种可能：真（True，记作 T 或 1）或假（False，记作 F 或 0）。原子命题（Atomic Proposition）是不可再分解为更简单命题的命题，通常用小写字母 p, q, r 等表示。复合命题（Compound Proposition）则是由原子命题通过逻辑联结词组合而成的命题。需要特别注意，疑问句、祈使句、感叹句以及含有自由变量的陈述句（如"x > 5"）都不是命题。含有代词的、在不同语境下真值不同的句子（如"今天下雨"）在被给定具体语境后可以成为命题。

## 关键结论

- 命题必须满足二值原则：任何命题恰有一个真值，要么为真要么为假
- 命题逻辑讨论的基本对象是陈述句，研究的是命题之间的真值函数关系而非命题的内容
- "这句话是假的"这类自指语句导致悖论，不是合法的命题
- 原子命题是构建复杂逻辑表达式的最小单元，类似于自然语言中的简单句
- 在日常语言中，同一句话在不同语境下可能表达不同的命题，但在形式逻辑中我们要求命题的真值是确定的

## 易错点

1. 将祈使句或疑问句误认为命题：例如"请关门"和"你吃饭了吗？"都不是命题，因为它们没有真值
2. 将含有变量的陈述句当作命题："x + 3 = 7"不是命题，因为 x 是自由变量，真值不唯一确定；但"存在 x 使得 x + 3 = 7"是命题（真）
3. 混淆命题与命题的真值：命题是一个语句，真值是该语句的语义属性，两者是不同的概念
4. 认为所有陈述句都是命题：含模糊词（如"很多人"）或依赖主观判断的陈述句可能不具备客观真值

## 例题

**例题1**：判断下列哪些是命题，若是命题请指出其真值。
(a) 北京是中国的首都。
(b) 2 + 2 = 5。
(c) x > 10。
(d) 请保持安静。
(e) 火星上有生命存在。

**解答**：
(a) 是命题，真值为真。
(b) 是命题，真值为假。
(c) 不是命题，x 是自由变量，真值不确定。
(d) 不是命题，是祈使句。
(e) 是命题。尽管目前人类不知道其真值，但客观上它要么为真要么为假，满足命题的定义。

**例题2**：将下列自然语言语句形式化为原子命题。
"如果明天不下雨，我们就去公园，并且如果公园开放，我们会在那里野餐。"

**解答**：设原子命题：
p: 明天下雨
q: 我们去公园
r: 公园开放
s: 我们在公园野餐

则该复合命题为：(¬p → q) ∧ (r → s)

## 代码示例

```python
# 命题的真值表示与基本操作
class Proposition:
    """命题类，封装真值"""
    def __init__(self, name: str, value: bool):
        self.name = name
        self.value = value
    
    def __repr__(self):
        return f"{self.name} = {self.value}"

# 定义原子命题
p = Proposition("p", True)
q = Proposition("q", False)

# 真值表：复合命题的真值由原子命题真值唯一确定
print(f"命题 {p.name} 的真值为: {p.value}")
print(f"命题 {q.name} 的真值为: {q.value}")
```
