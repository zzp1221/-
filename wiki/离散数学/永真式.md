## 核心定义

永真式（Tautology），也称重言式或恒真式，是指对于其所有原子命题的任何真值指派，其真值恒为真的复合命题。与永真式相对的是**永假式**（Contradiction，矛盾式），即所有真值指派下均为假的命题。介于两者之间的称为**可满足式**（Contingency / Satisfiable Formula），即至少存在一种真值指派使其为真、至少存在一种使其为假的命题。永真式是命题逻辑中的定理（Theorem），不依赖任何前提即为真，构成逻辑推理的基石。重要的永真式包括：排中律 p ∨ ¬p、同一律 p → p、De Morgan律 ¬(p ∧ q) ↔ ¬p ∨ ¬q、双重否定律 ¬¬p ↔ p、假言推理 (p ∧ (p → q)) → q、归谬律 (¬p → F) → p 等。判断一个命题是否为永真式的基本方法是构造其真值表并验证其最终列全为 T。

## 关键结论

- 永真式是语义有效的（semantically valid）：在所有可能模型中均为真，即 ⊨ φ（φ 是永真式）
- 永真是逻辑真理（Logical Truth）的体现——它的真不依赖于世界的任何特定事实
- A ≡ B（A 与 B 逻辑等价）当且仅当 A ↔ B 是永真式
- 推理 A ⊨ B（从 A 语义蕴含 B）当且仅当 A → B 是永真式
- 永真式集合是可判定的：对于命题逻辑，存在算法（如真值表法、归结法、DPLL算法）在有限步骤内判定任意命题是否为永真式
- 公理化方法：从少数永真式（公理）出发，通过推理规则推导出所有永真式

## 易错点

1. 混淆"永真"与"可满足"：p ∨ q 在大多数指派下为真，但不是永真（p=F, q=F 时为假），它只是可满足式
2. 误认为日常语言中"显然为真"的命题都是永真式："1+1=2"为真但不是永真式，因为其真依赖于算术事实而非纯逻辑形式
3. 混淆 p → q 本身与 p → q 为永真式：p → q 不是永真式（当 p=T, q=F 时为假），但 (p ∧ (p → q)) → q 是永真式

## 例题

**例题1**：判断 (p → q) ∧ (q → r) → (p → r) 是否为永真式。

**解答**：构造真值表（见[[真值表]]例题），发现在所有 8 种真值指派下该命题均为真，故为永真式。此为"传递律"（或"三段论"），是经典逻辑的基本定理之一。

**例题2**：证明 ¬(p ∧ q) ↔ (¬p ∨ ¬q) 是永真式。

| p | q | p∧q | ¬(p∧q) | ¬p∨¬q | ¬(p∧q)↔(¬p∨¬q) |
|---|---|-----|--------|-------|---------------|
| T | T | T   | F      | F     | T             |
| T | F | F   | T      | T     | T             |
| F | T | F   | T      | T     | T             |
| F | F | F   | T      | T     | T             |

最终列全为 T，故该命题是永真式。这证明了 De Morgan 律的正确性。

## 代码示例

```python
import itertools

def is_tautology(formula, variables):
    """判定一个命题公式是否为永真式"""
    for values in itertools.product([True, False], repeat=len(variables)):
        assignment = dict(zip(variables, values))
        if not formula(assignment):
            return False, assignment  # 找到反例
    return True, None

# 测试排中律 p ∨ ¬p
def excluded_middle(assign):
    return assign['p'] or (not assign['p'])

result, counterexample = is_tautology(excluded_middle, ['p'])
print(f"排中律是永真式: {result}")

# 测试假言推理 (p ∧ (p → q)) → q
def modus_ponens_formula(assign):
    p, q = assign['p'], assign['q']
    return (p and (p <= q)) <= q

result, _ = is_tautology(modus_ponens_formula, ['p', 'q'])
print(f"假言推理公式是永真式: {result}")
```
