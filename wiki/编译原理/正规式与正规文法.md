## 核心定义

**正规式**（Regular Expression，正则表达式）是描述正则语言（Regular Language）的代数表示工具，由原子模式通过三种基本运算构造而成：连接（Concatenation）、选择（Union，以 | 表示）、Kleene闭包（*，零次或多次重复）。形式化定义：(1) ∅ 和 ε 是正规式，分别表示空语言和仅含空串的语言；(2) 对每个 a∈Σ，a 是正规式，表示语言 {a}；(3) 若 r 和 s 是正规式，则 (r|s)、rs、r* 也是正规式，分别表示 L(r) ∪ L(s)、L(r)L(s)、L(r)*。正规式的扩展运算包括：正闭包 r⁺（一次或多次）、可选 r?（零次或一次）、字符类 [a-z]（区间内任一字符）。

**正规文法**（Regular Grammar / Type-3 Grammar）是 Chomsky 文法谱系中限制最严的文法类型，生成式形式仅能为：(1) 右线性文法：A → aB 或 A → a 或 A → ε；(2) 左线性文法：A → Ba 或 A → a 或 A → ε。其中 A, B 是非终结符，a 是终结符。正规文法是**唯一一类与正则表达式和有限自动机等价的文法**，三者都恰好描述正则语言类。

## 关键结论

- Chomsky文法层次：0型（无限制）⊃ 1型（上下文有关）⊃ 2型（上下文无关）⊃ 3型（正规）。越下层表达能力越弱但解析效率越高
- 正规式 ⇔ NFA ⇔ DFA ⇔ 正规文法，四者等价，都可以相互转化
- 正规文法到正规式的转换：对右线性文法建立方程组，用 Arden 引理（X = Xα|β ⇒ X = βα*）求解
- 正规式到正规文法的转换：为每个子正规式分配一个非终结符，按正规式的结构递归构造文法
- 正规语言的判定性质：正则语言对交、并、补、差、反转、连接、Kleene闭包均封闭
- 证明某语言不是正规的强有力工具：正则语言的泵引理（Pumping Lemma for Regular Languages）

## 易错点

1. 混淆正规式与正规文法：正规式是语言的代数表示（表达简洁），正规文法是生成语言的文法（生成式形式受限）。两者等价但不能简单替换
2. 左线性文法到右线性文法的转换：简单反转生成式会导致终结符顺序错误，需引入新非终结符或先转为FA再转回
3. Kleene闭包 r* 包含空串 ε：初学者容易忘记 r* 匹配零次重复

## 例题

**例题1**：构造正规式 (a|b)*a(a|b) 的正规文法。

**解答**：用右线性文法。设 S 为开始符号。
S → aS | bS | A
A → aB
B → a | b
其中 S 对应 (a|b)*，A 对应读完最后一个 a，B 对应最后的 (a|b)。

验证：例如串 "aba"：S→a→S→b→S→A→aB→a ✔

**例题2**：将正规文法 G = ({S, A}, {a, b}, P, S)，P = {S → aS | bA, A → aA | bS | ε}，转换为正规式。

**解答**：列方程组（X = ... X | ... 形式）：
S = aS | bA
A = aA | bS | ε

由 Arden 引理解 A：A = aA | (bS | ε) ⇒ A = a*(bS | ε) = a*bS | a*

代入 S：S = aS | b(a*bS | a*) = aS | ba*bS | ba* = (a | ba*b)S | ba*

再由 Arden 引理：S = (a | ba*b)* ba*

此正规式描述的语言为：任意以 b 开头和间隔的字符串最终以 a* 结尾，但更准确的理解是：所有至少含一个 b 且以任意 a 序列分隔的串。验证："aba" 匹配：a|b=a, ba*b=b, ... 实际上该文法描述的语言是所有包含偶数个 b 的字符串。

## 代码示例

```python
import re

def test_regular_expression():
    """验证正规式 (a|b)*a(a|b)"""
    pattern = re.compile(r'^[ab]*a[ab]$')
    tests = ['aa', 'ab', 'ba', 'bb', 'aaa', 'aba', 'bba', 'bab']
    for s in tests:
        print(f"'{s}' -> {'匹配' if pattern.match(s) else '不匹配'}")

# 正规文法 → 简单解析器
def parse_regular_grammar(s):
    """模拟右线性文法 S→aS|bS|A, A→aB, B→a|b"""
    i = 0
    n = len(s)
    # (a|b)* 部分
    while i < n - 2:
        if s[i] in 'ab':
            i += 1
        else:
            return False
    # 倒数第2个字符应为 a
    if i + 1 < n and s[i] == 'a' and s[i+1] in 'ab':
        return True
    return False

test_regular_expression()
```
