## 核心定义

符号表（Symbol Table）是编译器中用于存储程序中每个标识符（变量名、函数名、类型名等）相关信息的核心数据结构。符号表的基本操作包括：(1) **插入**（Insert）：在当前作用域中添加一个新的标识符及其属性；(2) **查找**（Lookup）：在符号表中搜索某标识符，通常从最内层作用域向外查找；(3) **删除**（Delete）：当离开某个作用域时删除该作用域中的所有声明（或标记为不可见）。

符号表记录的信息随标识符类型不同而异。对于变量：类型（type）、存储类别（storage class）、作用域层级（scope level）、偏移量（offset in activation record）等。对于函数：返回类型、参数列表（类型和名称）、是否为外部函数等。对于类型名：类型的结构描述。符号表的实现技术主要包括：(a) **哈希表**（Hash Table）：提供 O(1) 平均查找时间，是最常用的实现；(b) **二叉树/平衡树**：O(log n) 查找，可支持按名称排序遍历；(c) **线性表**：简单但 O(n) 查找，仅适用于极小的作用域。

对于嵌套作用域的语言（如 C, Java, Python），符号表通常组织为**作用域栈**（Scope Stack / Symbol Table Stack），每个作用域一个独立的哈希表（或符号表），查找时从最内层向外搜索。进入新作用域时压入新表，离开时弹出。

## 关键结论

- 符号表是编译器中**贯穿所有阶段**的全局数据结构
- 作用域规则的实现：最近嵌套作用域优先（Most Closely Nested Rule）——内层声明遮蔽外层同名声明
- 前向引用（Forward Reference）的处理：某些语言（如 C）要求先声明后使用，符号表需支持"前瞻声明"
- 重载（Overloading）的处理：C++/Java 允许同名不同参函数——符号表需支持按签名（名称+参数类型）的多维查找
- 分程序结构（Block Structure）的符号表管理通过作用域栈天然支持
- 编译期结束后符号表可丢弃（仅运行期需要调试信息时保留）

## 易错点

1. LOOKUP 时忘记从内层开始搜索：直接返回外层声明，导致内层变量"被忽略"，产生错误的类型信息
2. 作用域离开时忘记删除符号表条目：导致后续的查找找到已离开作用域的过时声明——可通过栈式管理自动避免
3. 不同命名空间（标签名、结构名、变量名）的标识符共享同一符号表的处理不当——C 语言中标签和变量有独立的命名空间

## 例题

**例题1**：设计支持嵌套作用域的符号表结构。说明 `int x; { int x; x = 1; } x = 2;` 的处理。

**解答**：作用域栈：[全局表: {x→(int, offset=0)}]。进入块：压入新表。声明 `int x`：在块表中插入 {x→(int, offset=block)}。`x=1`：查找 x 从块表开始找到块内 x。离开块：弹出块表。`x=2`：查找 x 找到全局 x。通过栈式管理自然实现了遮蔽和恢复语义。

**例题2**：哈希表实现符号表时如何解决哈希冲突？对符号表性能的影响如何？

**解答**：常用链地址法（Separate Chaining）——每个桶维护一个链表。在最坏情况下（全冲突）查找退化到 O(n)，但通过合理设计哈希函数和初始桶数，实践中接近 O(1)。也可用开放地址法（Open Addressing）如线性探测，但删除操作较复杂。符号表查找是编译器中执行最频繁的操作之一（关键字除外），其性能直接影响编译器速度。

## 代码示例

```python
class SymbolTable:
    def __init__(self, parent=None):
        self.table = {}  # name -> {type, offset, ...}
        self.parent = parent
    
    def insert(self, name, **attrs):
        self.table[name] = attrs
    
    def lookup(self, name):
        """从当前作用域向外查找"""
        if name in self.table:
            return self.table[name]
        if self.parent:
            return self.parent.lookup(name)
        return None

class ScopeStack:
    def __init__(self):
        self.stack = [SymbolTable()]
    
    def enter_scope(self):
        self.stack.append(SymbolTable(parent=self.stack[-1]))
    
    def exit_scope(self):
        if len(self.stack) > 1:
            self.stack.pop()
    
    def insert(self, name, **attrs):
        self.stack[-1].insert(name, **attrs)
    
    def lookup(self, name):
        return self.stack[-1].lookup(name)

# 示例
scopes = ScopeStack()
scopes.insert('x', type='int')
scopes.enter_scope()
scopes.insert('x', type='float')  # 遮蔽外层x
print(f"内层x类型: {scopes.lookup('x')['type']}")  # float
scopes.exit_scope()
print(f"外层x类型: {scopes.lookup('x')['type']}")  # int
```
