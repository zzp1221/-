## 核心定义

中间代码（Intermediate Code / Intermediate Representation, IR）是源程序与目标机器代码之间的桥梁表示形式。它比源程序更接近机器，但仍保持与机器无关的特性，使得编译器的前端和后端可以独立开发和移植。最常见的中间代码形式是**三地址码**（Three-Address Code, TAC）。

**三地址码**是一种线性化的中间表示，每条指令最多有三个操作数（两个源操作数和一个目标），形如 x = y op z。三地址码的指令类型包括：赋值 x=y op z、一元运算 x=op y、复制 x=y、无条件跳转 goto L、条件跳转 if x goto L / ifFalse x goto L、过程调用 param x / call p,n、以及下标和地址操作 x=y[i] / x[i]=y、地址和指针操作 x=&y / x=*y / *x=y。三地址码可以通过不同的数据结构表示：

(1) **四元式**（Quadruple）：(op, arg1, arg2, result) 四个字段，不依赖临时变量名的顺序——所有临时变量名明确写出。占用空间较大但操作灵活，适合优化和重排。

(2) **三元式**（Triple）：(op, arg1, arg2) 三个字段，用三元式自身的**编号**（位置）代表中间结果。节省空间但一旦指令重排就需要更新所有引用编号，不利于代码移动。

(3) **间接三元式**（Indirect Triple）：在三元式基础上增加一个间接表（指针数组），重排指令时只需修改间接表，不修改三元式本身。兼具三元式的紧凑和四元式的灵活。

## 关键结论

- 三地址码是编译器 IR 的主流基础，LLVM IR 和 Java Bytecode 均可视为三地址码的扩展
- 四元式便于优化（指令可自由移动），三元式紧凑但不利于优化
- 静态单赋值形式（SSA, Static Single Assignment）是三地址码的重要变体：每个变量仅被赋值一次，大幅简化数据流分析和优化
- 中间代码的设计需平衡表达力、紧凑性和可优化性
- 生成三地址码的常见方法是基于语法制导翻译（SDT）——在归约或推导过程中即时生成中间代码指令
- 回填技术（Backpatching）用于处理控制流中的前向跳转：先生成占位跳转指令，待目标确定后再回填

## 易错点

1. 临时变量名冲突：同名临时变量应在不同作用域内不冲突——通常按序号递增确保唯一性（t1, t2, ...）
2. 三地址码中跳转目标的表示：在生成阶段目标地址未知时需用符号标号占位，最后再替换为实际地址
3. 四元式与三元式结果表示的区别：四元式中 result 显式命名；三元式中结果由三元式编号隐式表示

## 例题

**例题1**：将语句 `if (a > b) x = a + b; else x = a - b;` 翻译为三地址码，使用回填技术。

**解答**：
```
    if a > b goto L1
    goto L2
L1: t1 = a + b
    x = t1
    goto L3
L2: t2 = a - b
    x = t2
L3: ...
```
或用回填技术：先将跳转目标留空标记，后续回填。`if a > b goto _` (待回填) `goto _` (待回填) ... 等确定目标后回填标号。

**例题2**：将三地址码序列表示为四元式和三元式。

设有 `t1 = a + b; t2 = t1 * c;` 用四元式和三元式分别表示。

**四元式**：
(0) (+, a, b, t1)
(1) (*, t1, c, t2)

**三元式**：
(0) (+, a, b)
(1) (*, (0), c)
其中 (0) 表示引用第 0 条三元式的结果。

## 代码示例

```python
class ThreeAddressCode:
    def __init__(self):
        self.code = []
        self.next_temp = 1
    
    def new_temp(self):
        t = f't{self.next_temp}'
        self.next_temp += 1
        return t
    
    def emit(self, op, arg1, arg2=None, result=None):
        self.code.append((op, arg1, arg2, result))
    
    def gen_expr(self, expr):
        """简化: 为二元表达式生成TAC"""
        if expr[0] == 'binop':
            left = self.gen_expr(expr[1])
            right = self.gen_expr(expr[2])
            t = self.new_temp()
            self.emit(expr[3], left, right, t)
            return t
        elif expr[0] == 'num':
            return str(expr[1])
        elif expr[0] == 'id':
            return expr[1]

tac = ThreeAddressCode()
tac.gen_expr(('binop', ('num', 3), ('binop', ('num', 5), ('num', 2), '+'), '*'))
for instr in tac.code:
    print(instr)
```
