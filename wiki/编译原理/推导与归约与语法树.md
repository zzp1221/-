## 核心定义

**推导**（Derivation）是从文法的开始符号出发，反复使用产生式替换句型中的非终结符，最终得到仅含终结符的句子的过程。形式化：若 A → β 是一个产生式，则 αAγ ⇒ αβγ 称为一个推导步骤。最左推导（Leftmost Derivation）每次替换句型中**最左边**的非终结符；最右推导（Rightmost Derivation，又称**规范推导** Canonical Derivation）每次替换最右边的非终结符。

**归约**（Reduction）是推导的逆过程——从句子出发，反向应用产生式将句型中的子串替换为对应的非终结符，逐步回归到开始符号。归约是自底向上语法分析（LR 分析）的核心操作。

**语法树**（Parse Tree）是推导的图形表示：根为开始符号，叶子从左到右构成推导出的句子，内部节点为非终结符，其孩子对应产生式右边。语法树的一个关键性质：它抽象掉推导顺序——同一个语法树可对应多个推导序列（例如最左和最右推导序列不同但语法树相同）。若某文法允许同一个句子有不同的语法树，则该文法是**二义的**。

核心概念层级：短语（Phrase）是语法树中某个子树的叶子序列；直接短语（Simple Phrase）是单步推导对应的短语；句柄（Handle）是最左直接短语，是自底向上分析中归约的触发条件。

## 关键结论

- 对给定语法树，恰有一个最左推导和一个最右推导与之对应
- 句型是推导过程中出现的任意符号串（可含非终结符），句子是不含非终结符的句型
- 语法树的结构决定了句子的层次语义和运算优先级——这是结构化解释的基础
- 句柄是自底向上分析的核心概念：LR 分析器通过识别句柄来决策何时进行归约
- 规范归约（Canonical Reduction）是最右推导的逆过程，每一步归约的都是当前句型的句柄
- 语法树与抽象语法树（AST）的区别：前者包含所有语法细节（如分号、括号等），后者只保留语义相关的结构

## 易错点

1. 最左推导与最右推导得到的是否同一语法树：对于无二义文法，最左和最右推导得到同一个语法树，只是推导步骤顺序不同
2. "短语"的定义：α 是句型 αβδ 关于非终结符 A 的短语，当且仅当 S ⇒* αAδ ⇒+ αβδ——短语是相对于特定非终结符而言的
3. 混淆语法树与 AST：语法树（Parse Tree）包含所有语法细节；AST 去掉冗余信息（如关键字、分隔符），更紧凑

## 例题

**例题1**：文法 E → E+E | E*E | (E) | id。给出句子 id+id*id 的最左推导和最右推导，并画出语法树。

**解答**：
最左推导：E ⇒ E+E ⇒ id+E ⇒ id+E*E ⇒ id+id*E ⇒ id+id*id
最右推导：E ⇒ E+E ⇒ E+E*E ⇒ E+E*id ⇒ E+id*id ⇒ id+id*id

语法树（对应最左推导）：
```
        E
      / | \
     E  +  E
     |    /|\
    id   E * E
         |   |
        id  id
```
注意：此文法二义——句子 id+id*id 还有另一个语法树（先算 + 再算 *），读者可自行画出对比。

**例题2**：对于上述最右推导的每一步，找出句柄（Handle）。

**解答**：
句型序列：E+E*id, E+id*id, id+id*id。最后一步的句柄是 id（最左边的 id），倒数第二步的句柄是 id（E+id*id 中 E+ 后面的 id），...。句柄的识别是 LR 分析器的核心。

## 代码示例

```python
def leftmost_derivation(productions, target, max_steps=20):
    """模拟最左推导，展示步骤"""
    sentential = ['S']
    steps = [''.join(sentential)]
    for _ in range(max_steps):
        # 找最左非终结符
        for i, sym in enumerate(sentential):
            if sym in productions:
                # 替换
                rhs = productions[sym][0]  # 取第一个产生式
                sentential = sentential[:i] + list(rhs) + sentential[i+1:]
                steps.append(''.join(sentential))
                if ''.join(sentential) == target:
                    return steps
                break
    return steps

# 简化文法: S → aSb | ε  (生成 aⁿbⁿ)
prod = {'S': [['a','S','b'], ['ε']]}
steps = leftmost_derivation(prod, 'aabb')
print("推导序列:", " ⇒ ".join(steps))
# S ⇒ aSb ⇒ aaSbb ⇒ aabb
```
