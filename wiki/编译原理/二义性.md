## 核心定义

二义性（Ambiguity）是指一个上下文无关文法允许同一个句子拥有两棵（或多棵）不同的语法树（Parse Tree），即存在多种不同的推导方式得到不同语法结构的同一个句子。具有该性质的文法称为**二义文法**（Ambiguous Grammar）。二义性是语法分析中的核心问题：编程语言的文法必须是**无二义的**，因为编译器需要为每个程序确定唯一的语法结构以便进行后续翻译。常见的二义性场景包括：(1) **运算优先级未指定**：如 `id+id*id` 在文法 E→E+E|E*E|id 下有两种解释（先加后乘 vs 先乘后加）；(2) **结合性未指定**：如 `id-id-id` 可以左结合 `(id-id)-id` 或右结合 `id-(id-id)`；(3) **悬空 else 问题**（Dangling Else）：`if C1 then if C2 then S1 else S2` 中 else 可以与内层或外层 if 匹配。

消除二义性的方法：(a) 重写文法（引入额外非终结符强制优先级和结合性）；(b) 在语法分析器中添加消歧规则（如 YACC 中使用 %left、%right 声明结合性和优先级）；(c) 使用上下文条件（如"else 与最近未匹配的 then 配对"）。

注意：**判定任意 CFG 是否二义是不可判定的**——不存在通用算法判断任意 CFG 有无二义。这是形式语言理论的一个经典结论。

## 关键结论

- 二义性是文法的性质，不是语言的本质属性：同一个语言可以有二义的文法，也可以有无二义的文法
- 编程语言规范一般给出无二义文法或附加消歧规则
- 通过引入新的非终结符来层级化文法（如 expr → expr + term | term; term → term * factor | factor）是消除运算符优先级二义的标准方法
- YACC/Bison 通过 %left、%right、%nonassoc 声明和优先级比较来解决移进-归约冲突
- 悬空 else 的标准解决是"else 匹配最近的未匹配 then"
- 固有二义语言（Inherently Ambiguous Language）：存在某些 CFL，其所有生成它的 CFG 都是二义的

## 易错点

1. 仅用部分推导证明无二义——仅展示最左和最右推导可产生相同语法树不够；需证明对任意句子只有一棵语法树
2. 以为引入运算符优先级后文法自动处理了结合性——需同时指定结合性方向（左结合 or 右结合）
3. 混淆"消除二义"与"消除文法二义"：有时保留二义文法但配合消歧规则（如 YACC）使用是更实用的做法

## 例题

**例题1**：证明文法 S → aS | Sa | a 是二义的。

**解答**：构造句子 "aaa"。语法树1: S→aS→aaS→aaa（连续左扩展）。语法树2: S→Sa→aSa→aaa（先右扩展后左扩展）。两棵语法树结构不同：树1呈左偏斜，树2呈右偏斜后左偏斜。句子相同但结构不同，故文法二义。

**例题2**：将二义文法 E → E+E | E*E | (E) | id 改写为无二义文法（指定 + 左结合且优先级低于 *）。

**解答**：
```
E → E + T | T
T → T * F | F
F → ( E ) | id
```
其中三个非终结符层级反映了优先级（F > T > E），+ 的产生式 E → E+T 体现了左结合性。

## 代码示例

```python
def check_leftmost_derivations(grammar, sentence, max_depth=10):
    """检查句子是否有多种最左推导（简单情况下的二义性检测）"""
    def derive_all(sf, depth):
        if depth == 0:
            return set()
        results = set()
        for i, sym in enumerate(sf):
            if sym in grammar:
                for rhs in grammar[sym]:
                    new_sf = sf[:i] + list(rhs) + sf[i+1:]
                    new_str = ''.join(new_sf)
                    if new_str == sentence:
                        results.add(tuple(sf))  # 记录推导路径
                    else:
                        results |= derive_all(new_sf, depth-1)
        return results
    return len(derive_all(['S'], max_depth)) > 0

# 二义文法示例
g_ambig = {'S': [['a','S'], ['S','a'], ['a']]}
print(f"存在多推导: {check_leftmost_derivations(g_ambig, 'aaa')}")
```
