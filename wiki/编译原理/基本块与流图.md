## 核心定义

基本块（Basic Block）是程序中一个最大的**顺序执行**的指令序列——块内不含任何跳转指令（除最后一条可能含跳转），也没有其他跳转指令的目标落在块中间（除第一条指令）。换句话说，基本块的执行必然是流式进行的：一旦进入块的第一条指令，块内所有指令都会依次执行完毕。

基本块的划分算法：(1) 确定**首指令**（Leader）：(a) 第一条指令是首指令；(b) 任何跳转指令的目标指令是首指令；(c) 紧跟在条件跳转或无条件跳转之后的指令是首指令；(2) 每个首指令开始的直到下一个首指令之前（或程序末尾）的指令序列构成一个基本块。

**流图**（Flow Graph）或**控制流图**（Control Flow Graph, CFG）是以基本块为节点、以跳转关系为有向边的图：基本块 B₁ 到 B₂ 有一条边当且仅当 B₁ 的最后一条指令跳转到 B₂ 的第一条指令（条件跳转有两类出边：跳转目标块和顺序后续块）。CFG 显示了程序所有可能的执行路径，是后续所有代码优化（数据流分析、死代码消除、循环检测等）的基础。

## 关键结论

- 基本块是代码优化的最小"原子"单元——块内的优化相对简单（局部优化），跨块的优化更为复杂（全局优化）
- 确定首指令的三个规则覆盖了所有基本情况，算法复杂度 O(n)（n 为指令数）
- 流图节点代表基本块，边代表可能的控制转移——条件跳转产生两条出边
- 首块（Entry Block）是包含程序第一条指令的基本块；出口块（Exit Blocks）是无出边的基本块
- 流图的计算机表示：邻接表（每个块记录前驱和后继列表）
- 支配关系（Dominance）：块 D 支配块 B 当任何从入口到 B 的路径都必须经过 D——是循环分析的基础

## 易错点

1. 首指令判断时忘记第三条规则：条件跳转后的语句也是首指令（因为条件跳转若未跳转，会顺序执行到下一条指令）
2. 将 call 指令视为跳转：过程调用后通常返回原处继续执行，**不是**跳转——除非是尾调用（Tail Call）
3. 基本块不等于函数：一个函数通常包含多个基本块

## 例题

**例题1**：将以下三地址码划分为基本块并画流图。
```
(1) a = 1
(2) b = 2
(3) if a < b goto (6)
(4) c = 3
(5) goto (7)
(6) c = 4
(7) d = a + b
(8) return d
```

**解答**：首指令：(1)第一条、(4)跳转目标、(6)紧接条件跳转、(7)跳转目标。
基本块：B1={(1),(2),(3)}、B2={(4),(5)}、B3={(6)}、B4={(7),(8)}。
流图边：B1→B2（条件不满足）、B1→B3（条件满足跳转）、B2→B4（goto）、B3→B4（顺序）。

**例题2**：流图中循环的自然循环（Natural Loop）的定义是什么？

**解答**：自然循环有两个条件：(1) 循环必须有唯一的入口节点（Header），该节点支配循环中所有其他节点；(2) 循环中至少有一条返回边（Back Edge）指向 Header。形式上，边 n→d 是返回边当 d 支配 n。自然循环是返回边 d→n 对应的循环：包含 d 和所有到 n 无需经过 d 到达 n 的节点。

## 代码示例

```python
def divide_basic_blocks(instructions):
    """将三地址码指令序列划分为基本块"""
    # 第一步: 找到首指令
    leaders = {0}  # 第一条指令始终是首指令
    for i, instr in enumerate(instructions):
        if instr.startswith('goto') or instr.startswith('if'):
            if i + 1 < len(instructions):
                leaders.add(i + 1)  # 跳转后的指令
            # 解析跳转目标
            target = int(instr.split()[-1].replace('(', '').replace(')', ''))
            leaders.add(target - 1 if target > 0 else target)
    
    # 第二步: 按首指令分组
    leaders = sorted(leaders)
    blocks = []
    for i in range(len(leaders)):
        start = leaders[i]
        end = leaders[i+1] if i + 1 < len(leaders) else len(instructions)
        blocks.append(instructions[start:end])
    return blocks

code = ['a=1', 'b=2', 'if a<b goto 6', 'c=3', 'goto 7', 'c=4', 'd=a+b', 'return d']
blocks = divide_basic_blocks(code)
for i, b in enumerate(blocks): print(f"B{i+1}: {b}")
```
