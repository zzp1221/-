## 核心定义

翻译模式（Translation Scheme）是将高级语言的控制结构和表达式翻译为中间代码（特别是三地址码）的系统化方法。关键在于将结构化控制流（if-else, while, for, switch）转换为包含条件跳转和无条件跳转的线性指令序列。

**布尔表达式的短路翻译**是最具代表性的翻译模式。在 C/Java 等语言中，布尔表达式使用短路求值（Short-Circuit Evaluation）：`A && B` 中若 A 为假则不计算 B；`A || B` 中若 A 为真则不计算 B。对应的三地址码模式：
- `E1 && E2`：翻译 E1 → 若假跳转到 `E.false`；翻译 E2 → 若假跳转到 `E.false`；E.true 为后继代码
- `E1 || E2`：翻译 E1 → 若真跳转到 `E.true`；翻译 E2 → 若真跳转到 `E.true`；E.false 为后继位置

**回填技术**（Backpatching）是核心实现策略：在单遍翻译中，当前向跳转的目标地址尚未确定时，先生成占位跳转指令并将其加入待回填列表（list of incomplete jumps）。待到目标地址确定后，统一回填（用实际地址替换占位符）。回填技术避免了额外的遍扫描来解析跳转地址。

**控制流语句的翻译模式**：(1) if (E) S → E.true 跳到 S 开始，E.false 跳到 S 结束后；(2) if (E) S1 else S2 → E.true 跳到 S1, E.false 跳到 S2；(3) while (E) S → 生成循环开始标号，E.true 跳到 S，E.false 跳出循环，S 结束后无条件跳回循环开始。

## 关键结论

- 布尔表达式的翻译利用了短路求值，将 && 和 || 映射为条件跳转序列
- 回填技术由三个核心函数支撑：makelist(i) 创建包含指令 i 的列表、merge(p1,p2) 合并两个列表、backpatch(p,i) 将列表 p 中所有指令的目标回填为标号 i
- 控制流翻译生成了基本块（基本块内部不含跳转），为后续优化奠定基础
- 布尔表达式的真假出口合成了控制流的"双出口"结构
- switch 语句的翻译可用跳转表（Jump Table）实现 O(1) 选择，或用条件跳转链实现
- 现代编译器多采用 IR 中的 φ 函数（SSA形式）处理控制流汇聚，而非纯粹的回填

## 易错点

1. 回填中忘记处理空列表：backpatch 需检查列表非空，否则对空列表回填无意义
2. 短路求值的语义必须正确：`A && B` 中 A 为假时不评估 B——不能简单翻译为 `t1=A; t2=B; result=t1&&t2`（因为 B 中的副作用被错误执行）
3. 多级嵌套控制流中回填目标的归属关系：需准确跟踪每条产生式关联的真/假出口列表

## 例题

**例题1**：用回填法翻译 `if (a < b) x = a; else x = b;`。

**解答**：
先生成 `if a < b goto _`（占位，加入 E.true 列表），`goto _`（E.false 列表）。然后翻译 S1 (x=a)，生成 `goto _`（跳出整个 if-else 的列表）。回填 E.true 到 S1 的开始，回填 E.false 到 S2 的开始。最终回填"跳出"到 if-else 后的下一条指令。

**例题2**：翻译 `while (i < n) { sum = sum + i; i = i + 1; }` 为三地址码。

**解答**：
```
L1: t1 = i < n
    ifFalse t1 goto L2
    t2 = sum + i
    sum = t2
    t3 = i + 1
    i = t3
    goto L1
L2: ...
```

## 代码示例

```python
class Backpatcher:
    def __init__(self):
        self.code = []
        self.next_instr = 100
    
    def new_label(self):
        return self.next_instr
    
    def emit(self, instr):
        self.code.append(instr)
    
    def backpatch(self, instr_indices, target):
        """回填跳转指令的目标地址"""
        for idx in instr_indices:
            self.code[idx] = self.code[idx].replace('_', str(target))
    
    def translate_if(self, cond, then_code, else_code=None):
        """回填法翻译if语句"""
        if_idx = len(self.code)
        self.code.append(f"if {cond} goto _")      # 待回填E.true
        
        false_jump = len(self.code)
        self.code.append("goto _")                   # 待回填E.false
        
        then_start = len(self.code)
        for instr in then_code:
            self.code.append(instr)
        
        if else_code:
            skip_else = len(self.code)
            self.code.append("goto _")               # 待回填跳过else
            self.backpatch([false_jump], len(self.code))
            for instr in else_code:
                self.code.append(instr)
            self.backpatch([if_idx], then_start)
            self.backpatch([skip_else], len(self.code))
        else:
            self.backpatch([if_idx], then_start)
            self.backpatch([false_jump], len(self.code))

bp = Backpatcher()
bp.translate_if('a < b', ['x = a'], ['x = b'])
for i, instr in enumerate(bp.code): print(f"{i+100}: {instr}")
```
