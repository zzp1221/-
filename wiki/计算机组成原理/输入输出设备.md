## 核心定义

输入输出设备（I/O Devices / Peripherals）是计算机与外界环境进行信息交互的物理设备，是计算机系统的重要组成部分。输入设备将外部世界的模拟或数字信息转换为计算机可以处理的二进制数据：键盘将按键位置转换为扫描码（Scancode），经由键盘控制器翻译为 ASCII/Unicode 字符码；鼠标将机械移动/光学反射转换为相对坐标增量（dx, dy），通过 USB 或 PS/2 协议报告给主机；触摸屏、麦克风、摄像头、扫描仪等将不同形式的信息数字化。输出设备将计算机内部处理的二进制数据转换为人类可感知的信号：显示器通过显卡（GPU）将帧缓冲（Framebuffer）中的像素数据转换为屏幕上的光信号，分 CRT、LCD/LED、OLED 等技术；打印机将页面描述语言（PCL/PostScript）转换为纸张上的墨点，主要有激光打印（静电成像）和喷墨打印技术。磁盘驱动器（HDD/SSD）既是输入设备也是输出设备（存储设备）。

## 关键结论

- 键盘工作原理：矩阵式按键扫描 -> 检测行/列交叉 -> 键盘控制器发送扫描码 -> BIOS/OS 转码为 ASCII
- 鼠标工作原理：光电传感器或激光传感器周期性采样表面图像 -> DSP 计算位移增量 -> 报告给主机
- 显示器（LCD）：液晶分子在电场下偏转 -> 调节背光通过偏光片的透光量 -> 红绿蓝三色子像素合成颜色
- 显卡（GPU）是连接 CPU 和显示器的关键组件：CPU 将绘制指令发给 GPU -> GPU 执行渲染管线生成像素 -> 帧缓冲 -> 显示器刷新
- 打印机页描述语言（PDL）：PostScript（Adobe，图灵完备的栈式语言）、PCL（HP，较简单）、PDF（直接传递）

## 易错点

1. 显示器刷新率和显卡帧率是不同的概念：显示器刷新率（60Hz/120Hz/144Hz）是屏幕物理刷新频率；帧率是 GPU 生成画面的速度（FPS）。两者通过 V-Sync/Adaptive Sync 协调。
2. 键盘扫描码不是 ASCII：按下一个键产生"扫描码"（键的物理位置编号），键盘驱动程序负责转换为对应字符（考虑 Shift/CapsLock/键盘布局等）。
3. I/O 设备的性能指标：分辨率（DPI）、响应时间（ms）、刷新率（Hz）、色彩深度（bit/色域）等。

## 例题

**例题1：** 以键盘输入一个字符'A'为例，描述从按键到屏幕显示的数据流。

**解答：** (1) 键盘矩阵扫描检测到'A'键闭合；(2) 键盘控制器将扫描码（如 USB HID 协议的 usage code 0x04）发送给主机 USB 控制器；(3) USB 驱动→HID 驱动→键盘驱动解析扫描码，结合 Shift 状态映射为 ASCII 0x41；(4) 操作系统将字符放入前台应用程序的消息队列；(5) 应用程序（如文本编辑器）将字符'A'的字形光栅化写入显存/帧缓冲；(6) 显示器刷新周期读取帧缓冲，LCD 阵列显示'A'的像素矩阵。

**例题2：** LCD 显示器的子像素排列和颜色合成原理。

**解答：** 每个像素由红(R)、绿(G)、蓝(B)三个子像素紧密排列组成。背光层发出白光，经液晶层和彩色滤光片（Color Filter）选择性地透光各颜色的强度。调节液晶偏转电压控制透光量，不同强度 R/G/B 的叠加产生任意颜色（加色混色）。例如黄色 = R 全开 + G 全开 + B 关 = 最大红光+最大绿光=人眼感知黄色。
