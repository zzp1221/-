## 核心定义

图的存储结构主要有两种：邻接矩阵和邻接表。(1) 邻接矩阵（Adjacency Matrix）：用一个 n*n 的二维数组存储图中顶点间的邻接关系。无权图中 g[i][j]=1 表示有边，=0 无边；带权图中 g[i][j]=w（权重）或 INF（无连接）。无向图的邻接矩阵是对称的。(2) 邻接表（Adjacency List）：为每个顶点建立一个单链表，链表中存储与该顶点邻接的所有顶点（或边信息）。对于有向图，还有逆邻接表（存储入边）和十字链表（同时存储出边和入边）；对于无向图，有邻接多重表（每条边用一个结点表示，方便删除）。选择哪种存储结构取决于图是稠密还是稀疏以及需要何种操作。

## 关键结论

- 邻接矩阵空间复杂度 O(n^2)，适合稠密图（|E| 接近 n^2）
- 邻接表空间复杂度 O(n+e)，适合稀疏图（|E| 远小于 n^2）
- 邻接矩阵判断 i-j 是否有边只需 O(1)；邻接表判断需遍历链表，O(degree(v))
- 邻接矩阵求所有邻接点需 O(n)；邻接表求所有邻接点只需 O(degree(v))
- 无向图邻接矩阵是对称矩阵，可压缩存储
- 十字链表（有向图）和邻接多重表（无向图）是邻接表的进一步改进

## 易错点

1. 入度和出度的计算：邻接矩阵中，第 i 行非零元个数是顶点 i 的出度，第 i 列非零元个数是顶点 i 的入度；邻接表中查入度需要遍历所有顶点的链表（或维护逆邻接表）
2. 带权图用 INF 表示无边时的选择：INF 不能太大（避免溢出），不能太小（不能和真实权重混淆），常用 INT_MAX/2 或定义一个较大的数
3. 邻接表结点的结构设计：对于带权图，链表结点需要包含邻接点编号和权值两个域；对于无向图，每条边在邻接表中出现两次

## 例题

**例1：** n 个顶点的无向图采用邻接矩阵存储，问该矩阵有多少个有效元素？若采用邻接表，最多有多少个边表结点？

**解答：** 邻接矩阵有 n^2 个元素，若存储整个矩阵（非压缩）。邻接表边表结点最多为 n(n-1)（完全无向图，每条边存储 2 次）。

## 代码示例

```cpp
#include <vector>
#include <iostream>
using namespace std;

// 邻接矩阵存储
class Graph_AM {
    vector<vector<int>> adj;
    int n;
public:
    Graph_AM(int vertices) : n(vertices), adj(vertices, vector<int>(vertices, 0)) {}
    
    void addEdge(int u, int v, int w = 1) {
        adj[u][v] = w;
        adj[v][u] = w;  // 无向图
    }
    
    bool isAdjacent(int u, int v) { return adj[u][v] != 0; }
};

// 邻接表存储
class Graph_AL {
    struct EdgeNode {
        int adjvex;  // 邻接点编号
        int weight;  // 权值
        EdgeNode* next;
        EdgeNode(int v, int w) : adjvex(v), weight(w), next(nullptr) {}
    };
    vector<EdgeNode*> vertices;  // 顶点表
    int n;
public:
    Graph_AL(int v) : n(v), vertices(v, nullptr) {}
    
    void addEdge(int u, int v, int w = 1) {
        EdgeNode* e = new EdgeNode(v, w);
        e->next = vertices[u];
        vertices[u] = e;
        // 无向图：加上反向边
        e = new EdgeNode(u, w);
        e->next = vertices[v];
        vertices[v] = e;
    }
    
    ~Graph_AL() {
        for (int i = 0; i < n; i++) {
            EdgeNode* p = vertices[i];
            while (p) {
                EdgeNode* t = p;
                p = p->next;
                delete t;
            }
        }
    }
};
```
