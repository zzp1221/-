## 核心定义

链地址法（Separate Chaining）是哈希冲突解决的另一种主流方法。其基本思想是：将哈希表设计成一个由 m 个链表头指针组成的数组，每个映射到同一地址 i 的关键字都存储在哈希表第 i 个槽位对应的一条链表中。当发生冲突时，新元素直接插入对应链表的末尾（或头部）。查找时先计算哈希值确定链表，再在链表中顺序查找目标元素。与开放定址法相比，链地址法的优点是不受负载因子限制（可以通过链表延伸），不会产生"堆积"，删除操作也更加自然（直接删除链表结点即可，无需懒惰标记）。链地址法的缺点是每个元素需要额外存储链表指针，且不同链表的长度可能不均匀。

## 关键结论

- 同义词（映射到同一地址的元素）存储在同一条链表中
- 负载因子可以 > 1（表中总元素数可超过表长 m）
- 插入通常在链表头部（O(1)）或尾部（需 O(k) 遍历）
- 查找时间复杂度取决于链表平均长度 alpha（负载因子）；当 alpha 不大时接近 O(1)
- 删除操作简单直接，不需要特殊标记
- 负载因子过大时，同样需要 rehash（扩大表并重新散列所有元素）

## 易错点

1. 链地址法的"空槽"：空槽对应的链表头指针为 NULL/空链表，与开放定址的 EMPTY 槽概念不同
2. 查找失败的比较次数：在链地址法中，查找失败时需要遍历完整条链表（而非遇到空槽即停），每条链表的平均长度为 alpha
3. rehash 的必要性：虽然链地址法可以无限加长链表，但链表过长会导致性能退化到 O(n)，需要在负载因子过大时做 rehash

## 例题

**例1：** 哈希表长 m=7, h(k)=k%7，用链地址法依次插入 {8, 15, 22, 10, 3, 17}，求查找成功时的平均查找长度。

**解答：** 8%7=1；15%7=1（链表中第二个）；22%7=1（链表中第三个）；10%7=3；3%7=3（链表中第二个）；17%7=3（链表中第三个）。比较次数：1,2,3,1,2,3。ASL = (1+2+3+1+2+3)/6 = 12/6 = 2。

## 代码示例

```cpp
#include <vector>
#include <list>
using namespace std;

class ChainingHash {
    vector<list<pair<int,int>>> table;  // (key, value)
    int m;
    int hash(int key) { return key % m; }
    
public:
    ChainingHash(int size) : m(size), table(size) {}
    
    void insert(int key, int value) {
        int idx = hash(key);
        for (auto& p : table[idx]) {
            if (p.first == key) { p.second = value; return; }
        }
        table[idx].push_front({key, value});  // 头插
    }
    
    bool search(int key, int& value) {
        int idx = hash(key);
        for (auto& p : table[idx]) {
            if (p.first == key) { value = p.second; return true; }
        }
        return false;
    }
    
    bool remove(int key) {
        int idx = hash(key);
        for (auto it = table[idx].begin(); it != table[idx].end(); ++it) {
            if (it->first == key) { table[idx].erase(it); return true; }
        }
        return false;
    }
};
```

```java
public class ChainingHash<K, V> {
    class Node { K key; V value; Node next; }
    Node[] table;
    
    public ChainingHash(int size) { table = new Node[size]; }
    
    int hash(K key) { return (key.hashCode() & 0x7FFFFFFF) % table.length; }
    
    public void put(K key, V value) {
        int idx = hash(key);
        Node cur = table[idx];
        while (cur != null) {
            if (cur.key.equals(key)) { cur.value = value; return; }
            cur = cur.next;
        }
        Node newNode = new Node(); newNode.key = key; newNode.value = value;
        newNode.next = table[idx];
        table[idx] = newNode;
    }
    
    public V get(K key) {
        Node cur = table[hash(key)];
        while (cur != null) {
            if (cur.key.equals(key)) return cur.value;
            cur = cur.next;
        }
        return null;
    }
}
```
