## 核心定义

快速排序（Quick Sort）由 Tony Hoare 于 1961 年提出，是最著名的分治排序算法，也是实践中效率最高的通用排序算法之一。算法的基本思想：(1) 从序列中选择一个元素作为枢轴（pivot）；(2) 通过一趟划分（partition）将序列分为两部分，左边所有元素 <= pivot，右边所有元素 >= pivot，pivot 被放到其最终位置；(3) 对左右两部分递归执行快速排序。快排的平均时间复杂度为 O(n log n)，最坏情况（每次选到极值）退化为 O(n^2)；空间复杂度取决于递归深度，平均 O(log n)。通过随机选择 pivot 或三数取中法可有效避免最坏情况。快排是不稳定排序，因为交换操作可能跨越相等的元素。

## 关键结论

- 平均 O(n log n)，最坏 O(n^2)，最好 O(n log n)
- 空间复杂度 O(log n) 递归栈
- 不稳定排序
- 性能关键：pivot 的选择影响划分均衡性，三数取中或随机 pivot 可优化
- 对小规模子数组通常切换到插入排序（阈值约 10-20）
- 实际性能通常优于归并排序和堆排序（常数因子小 + 缓存友好）

## 易错点

1. 划分时的指针条件和停止条件：两个指针 while 循环中必须保证 i < j 且不越界；遇到等于 pivot 的元素时是否交换影响稳定性但不影响正确性
2. 递归终止条件：当子数组长度 <= 1 时终止（low >= high）；使用插排优化时阈值处理要小心
3. pivot 选择的坑：不能总是选第一个/最后一个元素（有序输入会退化），应使用随机 pivot 或三数取中法

## 例题

**例1：** 对序列 {46, 79, 56, 38, 40, 84} 做一趟划分，以 46 为 pivot，写出划分后的结果。

**解答：** low=0, high=5, pivot=46。j 从右向左找到 40 < 46，i 从左向右找到 79 > 46，交换 -> {46,40,56,38,79,84}。j 继续向左找 38 < 46，i 向右找 56 > 46，交换 -> {46,40,38,56,79,84}。j 继续到 38，ij 相遇。交换 pivot 与 a[j] -> {38,40,46,56,79,84}。划分后：左 [38,40] <= 46，右 [56,79,84] >= 46。

## 代码示例

```cpp
int partition(int arr[], int low, int high) {
    int pivot = arr[low];  // 或使用三数取中
    int i = low, j = high;
    while (i < j) {
        while (i < j && arr[j] >= pivot) j--;
        arr[i] = arr[j];
        while (i < j && arr[i] <= pivot) i++;
        arr[j] = arr[i];
    }
    arr[i] = pivot;
    return i;
}

void quickSort(int arr[], int low, int high) {
    if (low >= high) return;
    // if (high - low <= 10) { insertionSort(arr, low, high); return; }
    int pivotIdx = partition(arr, low, high);
    quickSort(arr, low, pivotIdx - 1);
    quickSort(arr, pivotIdx + 1, high);
}
```

```java
public void quickSort(int[] arr, int low, int high) {
    if (low >= high) return;
    int pivotIdx = partition(arr, low, high);
    quickSort(arr, low, pivotIdx - 1);
    quickSort(arr, pivotIdx + 1, high);
}

int partition(int[] arr, int low, int high) {
    int pivot = arr[low], i = low, j = high;
    while (i < j) {
        while (i < j && arr[j] >= pivot) j--;
        if (i < j) arr[i++] = arr[j];
        while (i < j && arr[i] <= pivot) i++;
        if (i < j) arr[j--] = arr[i];
    }
    arr[i] = pivot;
    return i;
}
```
