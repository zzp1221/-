## 核心定义

归并排序（Merge Sort）是采用分治策略的经典排序算法。其核心思想是将两个（或多个）有序序列合并成一个更大的有序序列。二路归并排序的具体步骤：(1) 将待排序序列逐层二分直到每个子序列只有一个元素（天然有序）；(2) 递归返回时，将两个有序子序列合并（merge）为一个有序序列。归并排序的时间复杂度始终保持 O(n log n)，无论输入序列的初始状态如何。空间复杂度为 O(n)，需要辅助数组来暂存合并结果。归并排序是稳定排序（合并时相等元素保持原有顺序）。其天然的 O(n log n) 时间和稳定性使其成为许多编程语言标准库的默认稳定排序算法（如 Java 的 TimSort 和 Python 的 sorted/libc）。

## 关键结论

- 任何情况下时间复杂度均为 O(n log n)
- 稳定排序——两个等值元素的相对顺序在合并时被保留
- 空间复杂度 O(n)（需要辅助数组），不是原地排序
- 适合链表排序：链表的归并 O(1) 额外空间 O(n log n) 时间
- 是外部排序的核心基础（多路归并）

## 易错点

1. 合并时两个子序列的长度计算：mid = (low+high)/2，左子序列 [low, mid]，右子序列 [mid+1, high]。左右子序列可能长度不等（差 1）
2. 合并时等值元素的处理：`if (arr[i] <= arr[j]) temp[k++] = arr[i++]` 使用 <=（而非 <）可保证稳定性
3. 递归版与迭代版的区别：非递归（自底向上）归并通过控制子序列长度 gap 循环进行，空间 O(n) 时间 O(n log n)

## 例题

**例1：** 二路归并排序对 8 个元素的序列进行排序，共需多少趟归并？

**解答：** 8 = 2^3，共需 3 趟归并。第1趟两两归并为 4 组长度为 2 的有序子序列；第2趟归并为 2 组长度为 4；第3趟归并为 1 组长度为 8。

## 代码示例

```cpp
void merge(int arr[], int temp[], int low, int mid, int high) {
    int i = low, j = mid + 1, k = low;
    while (i <= mid && j <= high) {
        if (arr[i] <= arr[j]) temp[k++] = arr[i++];
        else temp[k++] = arr[j++];
    }
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= high) temp[k++] = arr[j++];
    for (i = low; i <= high; i++) arr[i] = temp[i];
}

void mergeSort(int arr[], int temp[], int low, int high) {
    if (low >= high) return;
    int mid = low + (high - low) / 2;
    mergeSort(arr, temp, low, mid);
    mergeSort(arr, temp, mid + 1, high);
    merge(arr, temp, low, mid, high);
}

// 非递归版（自底向上）
void mergeSortIterative(int arr[], int n) {
    int* temp = new int[n];
    for (int gap = 1; gap < n; gap *= 2) {
        for (int i = 0; i < n; i += 2 * gap) {
            int low = i;
            int mid = min(i + gap - 1, n - 1);
            int high = min(i + 2 * gap - 1, n - 1);
            if (mid < high) merge(arr, temp, low, mid, high);
        }
    }
    delete[] temp;
}
```

```java
public void mergeSort(int[] arr, int[] temp, int low, int high) {
    if (low >= high) return;
    int mid = low + (high - low) / 2;
    mergeSort(arr, temp, low, mid);
    mergeSort(arr, temp, mid + 1, high);
    merge(arr, temp, low, mid, high);
}

void merge(int[] arr, int[] temp, int low, int mid, int high) {
    int i = low, j = mid + 1, k = low;
    while (i <= mid && j <= high)
        temp[k++] = arr[i] <= arr[j] ? arr[i++] : arr[j++];
    while (i <= mid) temp[k++] = arr[i++];
    while (j <= high) temp[k++] = arr[j++];
    System.arraycopy(temp, low, arr, low, high - low + 1);
}
```
