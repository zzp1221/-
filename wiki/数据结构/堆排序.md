## 核心定义

堆排序（Heap Sort）是利用堆（完全二叉树的顺序存储）数据结构设计的选择排序改进算法。堆分为大顶堆（每个结点 >= 它的孩子，根为最大值）和小顶堆（每个结点 <= 它的孩子，根为最小值）。堆排序的基本思想：(1) 建堆——将待排序序列构建成一个大顶堆；(2) 反复取堆顶（最大元素）与堆最后一个元素交换，然后对减少了一个元素的堆重新调整为大顶堆（"下沉"操作）。重复直到堆只剩一个元素。建堆可以通过从最后一个非叶子结点（下标 n/2-1）开始向前逐个下沉来在线性时间 O(n) 内完成。堆排序的时间复杂度为 O(n log n)（建堆 O(n) + n次调整 O(log n)）。空间复杂度 O(1)。堆排序是不稳定排序。

## 关键结论

- 时间复杂度 O(n log n)，原地排序
- 建堆 O(n)，每次删除堆顶和堆调整 O(log n)
- 空间复杂度 O(1)，不需要额外数组
- 不稳定排序（交换可能跨越大范围）
- 缺点是缓存不友好（父子下标不连续跳跃大），实际速度常不如快排

## 易错点

1. 建堆的方向：必须从最后一个非叶子结点（(n/2)-1）开始向前遍历到0，逐个调用 heapify。从0开始向下调整不能保证整体是堆
2. heapify 的递归/迭代实现：下沉过程中需要与最大的孩子比较交换（而非只与左孩子），必须找到两个孩子中较大者
3. 大顶堆/小顶堆选择：升序排序使用大顶堆（每次取最大放到末尾），降序排序使用小顶堆

## 例题

**例1：** 对序列 {4, 10, 3, 5, 1, 2} 进行堆排序的建堆过程。

**解答：** n=6，从下标 n/2-1 = 2 开始。下标2值为3，孩子为结点2（无孩子）。下标1值为10，孩子为5和1，10最大不需调整。下标0值为4，孩子为10和3，与10交换。继续下沉4到下标1，孩子5和1，与5交换。建堆后序列：[10, 5, 3, 4, 1, 2]。

## 代码示例

```cpp
void heapify(int arr[], int n, int i) {
    int largest = i;
    int left = 2 * i + 1, right = 2 * i + 2;
    
    if (left < n && arr[left] > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;
    
    if (largest != i) {
        swap(arr[i], arr[largest]);
        heapify(arr, n, largest);
    }
}

void heapSort(int arr[], int n) {
    // 建堆：O(n)
    for (int i = n / 2 - 1; i >= 0; i--)
        heapify(arr, n, i);
    
    // 逐个取堆顶
    for (int i = n - 1; i > 0; i--) {
        swap(arr[0], arr[i]);
        heapify(arr, i, 0);
    }
}
```

```java
public void heapSort(int[] arr) {
    int n = arr.length;
    for (int i = n / 2 - 1; i >= 0; i--)
        heapify(arr, n, i);
    for (int i = n - 1; i > 0; i--) {
        int t = arr[0]; arr[0] = arr[i]; arr[i] = t;
        heapify(arr, i, 0);
    }
}

void heapify(int[] arr, int n, int i) {
    int largest = i, l = 2*i+1, r = 2*i+2;
    if (l < n && arr[l] > arr[largest]) largest = l;
    if (r < n && arr[r] > arr[largest]) largest = r;
    if (largest != i) {
        int t = arr[i]; arr[i] = arr[largest]; arr[largest] = t;
        heapify(arr, n, largest);
    }
}
```
