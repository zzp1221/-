## 核心定义

基数排序（Radix Sort）是一种非比较型排序算法，它不通过元素间的比较来确定顺序，而是利用关键字的分配和收集来完成排序。基数排序的思想源于多关键字排序：对于十进制整数，可以按个位、十位、百位...依次排序。有两种实现方式：(1) LSD（Least Significant Digit）从最低位到最高位——先按个位分配和收集，再按十位，依此类推，最终得到有序序列。要求每次分配收集必须是稳定的；(2) MSD（Most Significant Digit）从最高位到最低位——先按最高位分成若干桶，桶内递归排序。基数排序的时间复杂度为 O(d*(n+r))，其中 d 是关键字的位数（趟数），r 是基数（十进制 r=10），n 是元素个数。空间复杂度为 O(n+r)。

## 关键结论

- 非比较排序，依赖分配和收集操作，突破 O(n log n) 下限（前提是关键码可分解）
- 时间复杂度 O(d*(n+r))，当 d 为常数时接近 O(n)
- LSD 基数排序要求每一趟分配收集都必须稳定
- 适用于整数、字符串、日期等多位关键码排序
- 缺点：需额外空间，对数据格式敏感，浮点数处理复杂

## 易错点

1. LSD 每趟稳定性的至关重要性：若某趟收集不稳定（改变了同桶中元素的顺序），之前趟的排序效果被破坏，最终结果错误
2. 分配时的桶计数与定位：需要先统计各桶的元素个数，再计算各桶的起始位置（前缀和），然后从后往前将元素放入临时数组（从后往前保持稳定性）
3. 支持负数：需先将负数与正数分开处理，或者将桶偏移（如额外偏移 0..9 个桶处理负数）

## 例题

**例1：** 对序列 {329, 457, 657, 839, 436, 720, 355} 进行 LSD 基数排序，写出每趟结果。

**解答：** 趟1（个位）：桶分配 720(0), 436(6), 457(7), 657(7), 329(9), 839(9), 355(5)。收集：[720, 436, 457, 657, 329, 839, 355]。趟2（十位）：[720, 329, 436, 839, 355, 457, 657]。趟3（百位）：[329, 355, 436, 457, 657, 720, 839]。

## 代码示例

```cpp
void countingSort(int arr[], int n, int exp) {
    vector<int> output(n);
    int count[10] = {0};
    
    // 统计各桶数量
    for (int i = 0; i < n; i++)
        count[(arr[i] / exp) % 10]++;
    
    // 前缀和：计算各桶起始位置
    for (int i = 1; i < 10; i++)
        count[i] += count[i - 1];
    
    // 从后往前放入（保证稳定性）
    for (int i = n - 1; i >= 0; i--) {
        int digit = (arr[i] / exp) % 10;
        output[--count[digit]] = arr[i];
    }
    
    for (int i = 0; i < n; i++)
        arr[i] = output[i];
}

void radixSort(int arr[], int n) {
    int maxVal = *max_element(arr, arr + n);
    for (int exp = 1; maxVal / exp > 0; exp *= 10)
        countingSort(arr, n, exp);  // 对每位做计数排序
}
```

```java
public void radixSort(int[] arr) {
    int maxVal = Arrays.stream(arr).max().getAsInt();
    for (int exp = 1; maxVal / exp > 0; exp *= 10)
        countingSortByDigit(arr, exp);
}

void countingSortByDigit(int[] arr, int exp) {
    int n = arr.length;
    int[] output = new int[n];
    int[] count = new int[10];
    for (int x : arr) count[(x / exp) % 10]++;
    for (int i = 1; i < 10; i++) count[i] += count[i-1];
    for (int i = n-1; i >= 0; i--)
        output[--count[(arr[i]/exp)%10]] = arr[i];
    System.arraycopy(output, 0, arr, 0, n);
}
```
