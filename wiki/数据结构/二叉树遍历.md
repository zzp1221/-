## 核心定义

二叉树遍历（Binary Tree Traversal）是指按照某种规则依次访问二叉树中的每个结点，使得每个结点被访问一次且仅一次。二叉树的遍历分为两种策略：(1) 深度优先遍历（DFS）——包括前序遍历（NLR：根-左-右）、中序遍历（LNR：左-根-右）、后序遍历（LRN：左-右-根）。三种遍历的命名由访问根结点的顺序决定。(2) 广度优先遍历（BFS）——即层序遍历，从上到下、从左到右逐层访问。递归实现简单优雅，非递归实现需要借助栈（DFS）或队列（BFS）。二叉树的遍历是二叉树所有算法的基础。

## 关键结论

- 前序遍历：先根后左右；中序遍历：先左后根后右；后序遍历：先左右后根
- 已知前序+中序、或后序+中序可以唯一确定一棵二叉树；仅有前序+后序不行
- 递归遍历时空复杂度均为 O(n)；非递归遍历需 O(n) 辅助栈空间
- 层序遍历（BFS）使用队列实现，常用于求二叉树的宽度、深度等
- 中序遍历二叉排序树得到递增有序序列（这是 BST 判定的依据）

## 易错点

1. "根-左-右"中的左右指子树而非孩子结点：遍历是递归的，当访问左孩子后要完整遍历左子树才能到右子树
2. 前序+后序无法唯一确定二叉树：例如前序 AB、后序 BA，有可能 A 是根 B 是左孩子，也可能 A 是根 B 是右孩子
3. 非递归中序遍历中栈顶出栈时机：只有左子树全部处理完（即一路入栈到最左），才出栈访问并转向右子树

## 例题

**例1：** 已知二叉树的前序序列为 ABDCEF，中序序列为 DBAECF，请画出该二叉树。

**解答：** 前序首字符 A 为根；中序中 DBAECF，A 左边 DB 为左子树，右边 ECF 为右子树；递归地对左右子树应用相同逻辑。左子树前序 BD，中序 DB，B 为根，D 为左孩子；右子树前序 CEF，中序 ECF，C 为根，E 为左孩子，F 为右孩子。

## 代码示例

```cpp
#include <iostream>
#include <stack>
#include <queue>
using namespace std;

template<typename T>
struct BiNode { T data; BiNode *l, *r; BiNode(T v) : data(v), l(nullptr), r(nullptr) {} };

// 递归遍历
template<typename T>
void preOrder(BiNode<T>* root) {
    if (!root) return;
    cout << root->data << " ";
    preOrder(root->l);
    preOrder(root->r);
}

// 非递归前序遍历
template<typename T>
void preOrderNR(BiNode<T>* root) {
    stack<BiNode<T>*> s;
    BiNode<T>* p = root;
    while (p || !s.empty()) {
        if (p) {
            cout << p->data << " ";
            s.push(p);
            p = p->l;
        } else {
            p = s.top(); s.pop();
            p = p->r;
        }
    }
}

// 非递归中序遍历
template<typename T>
void inOrderNR(BiNode<T>* root) {
    stack<BiNode<T>*> s;
    BiNode<T>* p = root;
    while (p || !s.empty()) {
        while (p) { s.push(p); p = p->l; }
        p = s.top(); s.pop();
        cout << p->data << " ";
        p = p->r;
    }
}

// 非递归后序遍历（双栈法）
template<typename T>
void postOrderNR(BiNode<T>* root) {
    if (!root) return;
    stack<BiNode<T>*> s1, s2;
    s1.push(root);
    while (!s1.empty()) {
        BiNode<T>* p = s1.top(); s1.pop();
        s2.push(p);
        if (p->l) s1.push(p->l);
        if (p->r) s1.push(p->r);
    }
    while (!s2.empty()) {
        cout << s2.top()->data << " "; s2.pop();
    }
}

// 层序遍历
template<typename T>
void levelOrder(BiNode<T>* root) {
    if (!root) return;
    queue<BiNode<T>*> q;
    q.push(root);
    while (!q.empty()) {
        BiNode<T>* p = q.front(); q.pop();
        cout << p->data << " ";
        if (p->l) q.push(p->l);
        if (p->r) q.push(p->r);
    }
}
```
