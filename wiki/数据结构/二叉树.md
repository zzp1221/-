## 核心定义

二叉树（Binary Tree）是 n（n >= 0）个结点的有限集合，它或者为空（n=0），或者由一个根结点和两棵互不相交的、分别称为左子树和右子树的二叉树组成。二叉树与度为 2 的有序树不同：二叉树的子树有严格的左右之分，即使只有一棵子树也要区分是左子树还是右子树。二叉树有五种基本形态：空树、只有根结点、只有左子树、只有右子树、左右子树均有。二叉树通常采用二叉链表存储：每个结点包含 data、lchild、rchild 三个域。n 个结点的二叉链表中含有 n+1 个空指针域。

## 关键结论

- 二叉树第 i 层最多有 2^(i-1) 个结点（i >= 1）
- 深度为 k 的二叉树最多有 2^k - 1 个结点
- 对任意二叉树，若叶子结点数为 n0，度为 2 的结点数为 n2，则 n0 = n2 + 1
- 具有 n 个结点的完全二叉树的深度为 floor(log2 n) + 1
- n 个结点的二叉链表中空指针域的数量为 n + 1 个

## 易错点

1. 二叉树不是度为 2 的树：度为 0 或 1 的二叉树也是合法的，二叉树严格区分左右。度为 2 的树中，若一个结点只有一个孩子则不分左右
2. 满二叉树与完全二叉树的区别：满二叉树是深度 k、有 2^k-1 个结点的二叉树（每层都满）；完全二叉树是前 k-1 层满、第 k 层从左到右连续排列的二叉树。满二叉树一定完全，反之不成立
3. 叶子数（n0）与度为 2 结点数（n2）的关系：n0 = n2 + 1，推导依据是总边数 = n - 1 = n1 + 2*n2，且总节点数 n = n0 + n1 + n2

## 例题

**例1：** 一棵二叉树有 10 个度为 2 的结点，5 个度为 1 的结点，则叶子结点数为多少？

**解答：** n0 = n2 + 1 = 10 + 1 = 11。与度为 1 的结点数无关。

## 代码示例

```cpp
#include <iostream>
using namespace std;

template<typename T>
struct BiNode {
    T data;
    BiNode *lchild, *rchild;
    BiNode(T val) : data(val), lchild(nullptr), rchild(nullptr) {}
};

template<typename T>
class BinaryTree {
    BiNode<T>* root;
public:
    BinaryTree() : root(nullptr) {}
    
    void setRoot(BiNode<T>* r) { root = r; }
    
    // 叶子结点计数
    int countLeaves(BiNode<T>* node) {
        if (!node) return 0;
        if (!node->lchild && !node->rchild) return 1;
        return countLeaves(node->lchild) + countLeaves(node->rchild);
    }
    
    // 深度计算
    int depth(BiNode<T>* node) {
        if (!node) return 0;
        int l = depth(node->lchild);
        int r = depth(node->rchild);
        return max(l, r) + 1;
    }
};
```

```java
public class BinaryTree<T> {
    class Node {
        T data;
        Node left, right;
        Node(T val) { data = val; }
    }
    Node root;
    
    public int countLeaves(Node node) {
        if (node == null) return 0;
        if (node.left == null && node.right == null) return 1;
        return countLeaves(node.left) + countLeaves(node.right);
    }
    
    public int depth(Node node) {
        if (node == null) return 0;
        return Math.max(depth(node.left), depth(node.right)) + 1;
    }
}
```
