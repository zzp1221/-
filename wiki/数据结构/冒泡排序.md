## 核心定义

冒泡排序（Bubble Sort）是最基础的交换排序算法。其思想是：从头到尾依次比较相邻的两个元素，若逆序（前一个大于后一个）则交换它们，这样每趟排序都会将当前未排序部分的最大值"冒泡"到最右端（正确位置）。长度为 n 的序列最多需要 n-1 趟排序。可以通过设置 flag 标记某趟是否发生了交换，若无交换说明序列已经有序，可以提前终止。冒泡排序是稳定排序，时间复杂度为 O(n^2)，最好情况（已有序）通过提前终止可达 O(n)，空间复杂度 O(1)。冒泡排序在教材中常见但实际应用中很少使用（效率太低），主要作为教学算法。

## 关键结论

- 稳定排序
- 时间复杂度：最好 O(n)（带优化），平均和最坏 O(n^2)
- 空间复杂度 O(1)，原地排序
- 每趟将未排部分的最大元素沉到末尾
- 带优化版本：若某趟无交换则提前终止

## 易错点

1. 每趟比较的边界：第 i 趟（i 从 0 开始）需要比较 n-1-i 对相邻元素，因为末尾 i 个元素已就位
2. 交换方式的稳定性：必须用"严格大于"才交换，即 `arr[j] > arr[j+1]`（不用 >=），否则会破坏稳定性
3. 优化版中 flag 的重置：每趟开始前 flag = false，发生交换则置 true；若无交换直接 break

## 例题

**例1：** 对序列 {5, 1, 4, 2, 8} 进行冒泡排序，写出每趟结果。

**解答：** 第1趟：{1,4,2,5,8}；第2趟：{1,2,4,5,8}；第3趟：无交换，提前结束。

## 代码示例

```cpp
void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        if (!swapped) break;
    }
}
```

```java
public void bubbleSort(int[] arr) {
    for (int i = 0; i < arr.length - 1; i++) {
        boolean swapped = false;
        for (int j = 0; j < arr.length - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int t = arr[j]; arr[j] = arr[j+1]; arr[j+1] = t;
                swapped = true;
            }
        }
        if (!swapped) break;
    }
}
```
