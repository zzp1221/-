## 核心定义

红黑树（Red-Black Tree, RBT）是一种自平衡二叉搜索树。它的每个结点要么是红色要么是黑色，通过颜色规则保证没有一条路径会比其他路径长出两倍，从而实现近似平衡。红黑树满足以下 5 条性质：(1) 每个结点是红色或黑色；(2) 根结点是黑色；(3) 每个叶子结点（NIL）是黑色；(4) 红色结点的两个子结点必须都是黑色（不允许连续红色）；(5) 从任意结点到其所有后代叶子结点的每条路径上包含相同数量的黑色结点（黑高相同）。红黑树的查找、插入、删除时间复杂度均为 O(log n)，且在实际应用中由于旋转更少，常比 AVL 树更高效。C++ STL 的 map、set，Java 的 TreeMap、TreeSet 均基于红黑树实现。

## 关键结论

- 红黑树是近似平衡而非严格高度平衡：最长路径不超过最短路径的两倍
- 5 条规则核心：根黑、无连续红、黑高相同、叶黑
- 查找 O(log n)，与 AVL 相同；插入和删除也是 O(log n)，但旋转次数比 AVL 少
- 插入时新结点默认设为红色，若违反规则（如父红）通过重新着色和旋转修复
- 插入修复主要处理"叔结点"的颜色情况：叔红则重新着色上推，叔黑则旋转解决

## 易错点

1. 红黑树的"叶子"是 NIL 结点（黑色），不是空指针：这是理解黑高定义的关键，所有 NIL 视为外部结点
2. 插入修复中的 case 判断：case1（叔红，重新着色上推）、case2（叔黑，LR/RL 型，先旋转变成 case3）、case3（叔黑，LL/RR 型，旋转并变色解决）。三种 case 的处理不可混淆
3. 红黑树与 AVL 的比较：红黑树查找略慢（树更高），但插入删除更快（旋转少）；AVL 适合"查找多"的场景，红黑树适合"频繁插入删除"的场景

## 例题

**例1：** 下列关于红黑树的说法错误的是？A. 根结点是黑色的 B. 红色结点的父结点一定是黑色 C. 从根到叶子的最长路径不超过最短路径的两倍 D. 红黑树是严格平衡二叉树

**解答：** D 错误。红黑树是近似平衡而非严格平衡（AVL 才是严格平衡），它通过颜色规则保证了最长路径不超过最短路径的两倍。

## 代码示例

```cpp
enum Color { RED, BLACK };

template<typename T>
struct RBNode {
    T key;
    Color color;
    RBNode *left, *right, *parent;
    RBNode(T k) : key(k), color(RED), left(nullptr), right(nullptr), parent(nullptr) {}
};

template<typename T>
class RedBlackTree {
    RBNode<T>* root;
    RBNode<T>* NIL;  // 哨兵：黑色叶子
    
    void leftRotate(RBNode<T>* x) {
        RBNode<T>* y = x->right;
        x->right = y->left;
        if (y->left != NIL) y->left->parent = x;
        y->parent = x->parent;
        if (x->parent == nullptr) root = y;
        else if (x == x->parent->left) x->parent->left = y;
        else x->parent->right = y;
        y->left = x;
        x->parent = y;
    }
    
    void rightRotate(RBNode<T>* y) {
        RBNode<T>* x = y->left;
        y->left = x->right;
        if (x->right != NIL) x->right->parent = y;
        x->parent = y->parent;
        if (y->parent == nullptr) root = x;
        else if (y == y->parent->left) y->parent->left = x;
        else y->parent->right = x;
        x->right = y;
        y->parent = x;
    }
    
    void insertFixup(RBNode<T>* z) {
        while (z->parent && z->parent->color == RED) {
            if (z->parent == z->parent->parent->left) {
                RBNode<T>* y = z->parent->parent->right;  // 叔结点
                if (y->color == RED) {
                    // Case 1: 叔红
                    z->parent->color = BLACK;
                    y->color = BLACK;
                    z->parent->parent->color = RED;
                    z = z->parent->parent;
                } else {
                    if (z == z->parent->right) {
                        // Case 2: LR -> RR
                        z = z->parent;
                        leftRotate(z);
                    }
                    // Case 3: LL
                    z->parent->color = BLACK;
                    z->parent->parent->color = RED;
                    rightRotate(z->parent->parent);
                }
            } else {
                // 对称情况（父为右孩子）
                RBNode<T>* y = z->parent->parent->left;
                if (y->color == RED) {
                    z->parent->color = BLACK;
                    y->color = BLACK;
                    z->parent->parent->color = RED;
                    z = z->parent->parent;
                } else {
                    if (z == z->parent->left) {
                        z = z->parent;
                        rightRotate(z);
                    }
                    z->parent->color = BLACK;
                    z->parent->parent->color = RED;
                    leftRotate(z->parent->parent);
                }
            }
        }
        root->color = BLACK;
    }
    
public:
    RedBlackTree() {
        NIL = new RBNode<T>(T());
        NIL->color = BLACK;
        root = NIL;
    }
    
    void insert(T key) {
        RBNode<T>* z = new RBNode<T>(key);
        z->left = z->right = NIL;
        RBNode<T>* y = nullptr;
        RBNode<T>* x = root;
        while (x != NIL) {
            y = x;
            if (key < x->key) x = x->left;
            else x = x->right;
        }
        z->parent = y;
        if (!y) root = z;
        else if (key < y->key) y->left = z;
        else y->right = z;
        insertFixup(z);
    }
};
```
