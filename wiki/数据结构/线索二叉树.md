## 核心定义

线索二叉树（Threaded Binary Tree）是对传统二叉链表的一种改进。在 n 个结点的二叉链表中，共有 2n 个指针域，但只有 n-1 个分支（非空指针），剩余 n+1 个空指针域被浪费。线索二叉树将这些空指针域利用起来：若某结点左指针为空，则让它指向遍历序列中的直接前驱；若右指针为空，则让它指向遍历序列中的直接后继。这些指向前驱和后继的指针称为"线索"。为区分指针是指向孩子还是线索，每个结点需增设两个标志域 ltag 和 rtag（0 表示孩子，1 表示线索）。线索化后，可以不用栈或递归地实现二叉树的遍历，显著提高遍历效率。

## 关键结论

- 线索二叉树利用 n+1 个空指针域存储遍历顺序下的前驱或后继信息
- ltag=0 表示 lchild 指向左孩子，ltag=1 表示 lchild 指向前驱；rtag 同理
- 中序线索化最常用，可不用栈/递归实现中序遍历，O(n) 时间 O(1) 空间（不计递归栈）
- 线索化过程就是对二叉树的一次遍历，在遍历过程中修改空指针
- 通常增设一个头结点，其 lchild 指向根结点，方便遍历的起止

## 易错点

1. ltag 和 rtag 的语义：0=孩子/1=线索，这是最常记反的。必须是新设的标志域，不能靠指针是否为空来判断（因为线索化后空指针被填上了）
2. 线索化时前驱的传递：线索化过程中需要一个全局/引用变量 pre 记录上次访问的结点；若 pre->rchild 为空则 pre->rtag=1 且 pre->rchild=当前；若当前->lchild 为空则当前->ltag=1 且当前->lchild=pre
3. 线索化后遍历时需要区分孩子和线索：对于中序线索二叉树，找后继的规则是：若 rtag==1 直接取 rchild 为后继；否则走右子树的"最左下"结点

## 例题

**例1：** 在一棵中序线索二叉树中，若某结点有左孩子但没有右孩子，问如何找到它的后继？

**解答：** 该结点 rtag 应为 1（因右指针为空），其右指针直接指向后继，故直接取 rchild 即可。但注意该结点若有右孩子，则后继是其右子树的最左下结点。

## 代码示例

```cpp
template<typename T>
struct ThrNode {
    T data;
    ThrNode *lchild, *rchild;
    int ltag, rtag;  // 0:孩子 1:线索
    ThrNode(T val) : data(val), lchild(nullptr), rchild(nullptr), ltag(0), rtag(0) {}
};

template<typename T>
class ThreadedTree {
    ThrNode<T>* pre;  // 全局前驱
    
    void inThread(ThrNode<T>* p) {
        if (!p) return;
        inThread(p->lchild);
        if (!p->lchild) { p->ltag = 1; p->lchild = pre; }
        if (pre && !pre->rchild) { pre->rtag = 1; pre->rchild = p; }
        pre = p;
        inThread(p->rchild);
    }
    
public:
    void createInThread(ThrNode<T>* root) {
        ThrNode<T>* head = new ThrNode<T>(T());
        head->ltag = 0; head->rtag = 1;
        head->rchild = head;
        if (!root) { head->lchild = head; return; }
        head->lchild = root;
        pre = head;
        inThread(root);
        pre->rchild = head; pre->rtag = 1;
        head->rchild = pre;
    }
    
    // 中序线索二叉树的非递归遍历
    void inOrderTraverse(ThrNode<T>* head) {
        ThrNode<T>* p = head->lchild;
        while (p != head) {
            while (p->ltag == 0) p = p->lchild;  // 找最左下
            visit(p);
            while (p->rtag == 1 && p->rchild != head) {
                p = p->rchild; visit(p);
            }
            p = p->rchild;
        }
    }
};
```
