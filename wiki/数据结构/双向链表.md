## 核心定义

双向链表（Doubly Linked List）是在单链表的每个结点中再增设一个指向前驱结点的指针域。每个结点包含三个域：prior（前驱指针）、data（数据）、next（后继指针）。双向链表解决了单链表中无法逆向遍历的问题，也使得删除给定结点 p 的操作可以在 O(1) 时间内完成（单链表需要找前驱，O(n)）。双向链表的缺点是需要更多存储空间来存储 prior 指针，存储密度更低。双向链表可以是循环的也可以是非循环的，通常实现为带头结点的双向循环链表，使得所有插入删除操作可以统一处理。

## 关键结论

- 双向链表可以在 O(1) 时间内访问给定结点的前驱和后继
- 对称性：`p->next->prior == p` 且 `p->prior->next == p`
- 插入操作需要修改 4 个指针（两个结点的 prior 和 next）
- 删除给定结点 p 的时间为 O(1)，因为可以直接通过 `p->prior` 访问其前驱
- 通常采用带头结点的双向循环链表以简化边界处理

## 易错点

1. 插入时指针修改顺序混乱：在 p 之后插入 s，标准四步为 `s->next = p->next; s->prior = p; p->next->prior = s; p->next = s;`，其中后两步顺序不能颠倒
2. 删除时忘记处理边界：普通删除时 `p->next->prior = p->prior; p->prior->next = p->next;` 两步即可；但删除的是表尾结点时 p->next 为头结点，仍能正确执行（循环链表）
3. 空表判断条件：带头结点的双向循环链表为空时，`head->next == head && head->prior == head`

## 例题

**例1：** 在非空双向循环链表中，在结点 p 之前插入新结点 s 的正确操作序列是什么？

**解答：** `s->prior = p->prior; s->next = p; p->prior->next = s; p->prior = s;` 先将 s 的两端连接好，再让原前驱指向 s，最后让 p 指向 s。

## 代码示例

```cpp
template<typename T>
struct DNode {
    T data;
    DNode *prior, *next;
    DNode(T val) : data(val), prior(nullptr), next(nullptr) {}
};

template<typename T>
class DoublyLinkedList {
private:
    DNode<T>* head;  // 头结点，循环
    
public:
    DoublyLinkedList() {
        head = new DNode<T>(T());
        head->next = head->prior = head;
    }
    
    // 在 p 之后插入新结点
    void insertAfter(DNode<T>* p, T val) {
        DNode<T>* s = new DNode<T>(val);
        s->next = p->next;
        s->prior = p;
        p->next->prior = s;
        p->next = s;
    }
    
    // 删除结点 p（p 不能是头结点）
    void remove(DNode<T>* p) {
        if (p == head) return;
        p->prior->next = p->next;
        p->next->prior = p->prior;
        delete p;
    }
    
    bool isEmpty() {
        return head->next == head;
    }
};
```

```java
public class DoublyLinkedList<T> {
    class DNode {
        T data;
        DNode prior, next;
        DNode(T val) { data = val; }
    }
    private DNode head;
    
    public DoublyLinkedList() {
        head = new DNode(null);
        head.next = head.prior = head;
    }
    
    public void insertAfter(DNode p, T val) {
        DNode s = new DNode(val);
        s.next = p.next;
        s.prior = p;
        p.next.prior = s;
        p.next = s;
    }
    
    public void remove(DNode p) {
        if (p == head) return;
        p.prior.next = p.next;
        p.next.prior = p.prior;
    }
}
```
