## 核心定义

插入排序（Insertion Sort）是最简单直观的排序算法之一，模仿整理扑克牌的过程。其基本思想是：将待排序序列分为已排序区和未排序区，每次取未排序区的第一个元素，在已排序区中从后向前扫描，找到合适位置插入，保持已排序区始终有序。对于长度为 n 的序列，需要进行 n-1 趟排序。插入排序是稳定的排序算法。最好情况（输入已有序）：每趟只需比较 1 次，时间复杂度 O(n)；最坏情况（输入逆序）：每趟需比较 i 次并移动 i 次，时间复杂度 O(n^2)；平均时间复杂度 O(n^2)。空间复杂度 O(1)（仅需一个临时变量）。当数据量较小或数据基本有序时，插入排序效率很高（甚至优于 O(n log n) 算法），因此常被用作高级排序算法（如快速排序）对小规模子问题的兜底算法。

## 关键结论

- 稳定排序
- 最好 O(n)，最坏和平均 O(n^2)
- 空间复杂度 O(1)，原地排序
- 对小数据集或近乎有序数据的效率极高
- 可以作为快速排序在子数组较小时的兜底优化

## 易错点

1. 从后向前比较，边比较边移动：`while (j >= 0 && data[j] > temp) { data[j+1] = data[j]; j--; }` 顺序不能颠倒（先移再减）
2. 稳定性的保持条件：必须是 `data[j] > temp`（严格大于）才后移，如果是 `>=` 则会破坏稳定性（相等的元素顺序会颠倒）
3. 边界条件：j 可能减到 -1（temp 比所有元素都小），此时 data[j+1] 即 data[0] 放入 temp

## 例题

**例1：** 对序列 {5, 2, 4, 6, 1, 3} 进行插入排序，写出每趟排序后的结果。

**解答：** 初始 [5|2,4,6,1,3]; 第1趟 [2,5|4,6,1,3]; 第2趟 [2,4,5|6,1,3]; 第3趟 [2,4,5,6|1,3]; 第4趟 [1,2,4,5,6|3]; 第5趟 [1,2,3,4,5,6]。

## 代码示例

```cpp
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int temp = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > temp) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = temp;
    }
}
```

public void insertionSort(int[] arr) {
    for (int i = 1; i < arr.length; i++) {
        int temp = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > temp) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = temp;
    }
}
```

def insertion_sort(arr):
    for i in range(1, len(arr)):
        temp = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > temp:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = temp
```
