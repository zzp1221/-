## 核心定义

选择排序（Selection Sort）是一种直观简单的排序算法。其基本思想是：将序列分为已排序区（前面）和未排序区（后面）。每趟从未排序区中选择一个最小的元素，与未排序区的第一个元素交换位置，将其纳入已排序区。经过 n-1 趟选择后序列有序。选择排序的时间复杂度为 O(n^2)（无论最好还是最坏），比较次数固定为 n(n-1)/2 次，交换次数最多 n-1 次。空间复杂度 O(1)。选择排序是不稳定排序，因为交换可能改变相等元素原有的相对顺序（例如将最小元素与前面的某相等元素交换）。选择排序的核心优势在于交换次数少（O(n) 级别），适合元素较大但关键字较小的情况。

## 关键结论

- 时间复杂度恒为 O(n^2)，不受输入顺序影响
- 比较次数 n(n-1)/2，交换次数最多 n-1
- 空间复杂度 O(1)，原地排序
- 不稳定排序
- 交换少，适合记录大但关键字小的场景

## 易错点

1. 找最小值时起始位置的设定：第 i 趟（i 从 0 开始）从 i 位置开始找（前 i 个已就位），而非总是从 0 开始
2. 稳定性失效的原因：如 {5, 5, 2}，第一趟选 2 和第一个 5 交换 -> {2, 5, 5}，第二个 5 跑到了第三个位置，与原先第一个 5 的相对顺序改变
3. 当最小值恰好在 i 位置时无需交换：但检查 minIdx == i 再跳过交换是一个优化，不检查也可（自己和自己交换无意义但正确）

## 例题

**例1：** 对序列 {64, 25, 12, 22, 11} 进行选择排序，写出每趟结果。

**解答：** 第1趟：[11, 25, 12, 22, 64]；第2趟：[11, 12, 25, 22, 64]；第3趟：[11, 12, 22, 25, 64]；第4趟：[11, 12, 22, 25, 64]（已有序但算法仍会扫描）。

## 代码示例

```cpp
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int minIdx = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIdx])
                minIdx = j;
        }
        if (minIdx != i)
            swap(arr[i], arr[minIdx]);
    }
}
```

```java
public void selectionSort(int[] arr) {
    for (int i = 0; i < arr.length - 1; i++) {
        int minIdx = i;
        for (int j = i + 1; j < arr.length; j++)
            if (arr[j] < arr[minIdx]) minIdx = j;
        if (minIdx != i) {
            int t = arr[i]; arr[i] = arr[minIdx]; arr[minIdx] = t;
        }
    }
}
```
