## 核心定义

并查集（Union-Find 或 Disjoint Set Union, DSU）是一种用于管理不相交集合的数据结构。它支持两种基本操作：(1) Find——确定某个元素属于哪个子集，通常返回该集合的代表元素；(2) Union——将两个子集合并为一个。并查集通常用森林结构实现：每个集合用一棵树表示，树根为该集合的代表元素。为实现高效操作，采用两种优化策略：(1) 路径压缩（Path Compression）——Find 操作时将路径上所有结点直接连接到根，摊还时间复杂度接近 O(1)；(2) 按秩合并（Union by Rank）——合并时总是将高度较小的树连接到高度较大的树根上，防止树退化为链。结合两种优化，并查集的操作平均时间复杂度为 O(alpha(n))，其中 alpha(n) 是反阿克曼函数，在实际规模下不超过 4。

## 关键结论

- 并查集支持近乎 O(1) 的 Find 和 Union 操作（摊还）
- 路径压缩 + 按秩合并可将复杂度降至 O(alpha(n))
- 典型应用：Kruskal 最小生成树、图的连通分量、等价类划分
- 实现简洁：使用 parent 数组和 rank 数组（或 size 数组）
- 初始状态下每个元素自成一个集合（parent[i] = i）

## 易错点

1. Find 的路径压缩写法：需用递归或迭代将路径上所有结点的 parent 直接指向根；忘记压缩或只压缩一个结点会导致性能退化
2. Union 中的按秩合并实现：合并时比较 rank（近似树高），而非 size；只有当两树 rank 相等时合并后的 rank 才加一
3. parent 数组的初始化：每个元素初始时 parent[i] = i（自己是集合代表）而非 parent[i] = 0

## 例题

**例1：** 初始有 5 个元素 {1,2,3,4,5}，执行 Union(1,2), Union(3,4), Union(2,4), Find(1) 后，画出并查集结构（使用按秩合并）。

**解答：** Union(1,2): rank[1]=1, parent[2]=1；Union(3,4): rank[3]=1, parent[4]=3；Union(2,4): Find(2)=1, Find(4)=3, rank[1]=rank[3]=1, 合并后 rank[1]=2, parent[3]=1。Find(1)=1（根）。

## 代码示例

```cpp
#include <vector>
using namespace std;

class UnionFind {
    vector<int> parent, rank;
public:
    UnionFind(int n) : parent(n), rank(n, 0) {
        for (int i = 0; i < n; i++)
            parent[i] = i;
    }
    
    // 路径压缩查找
    int find(int x) {
        if (parent[x] != x)
            parent[x] = find(parent[x]);  // 路径压缩
        return parent[x];
    }
    
    // 按秩合并
    void unite(int x, int y) {
        int rx = find(x), ry = find(y);
        if (rx == ry) return;
        if (rank[rx] < rank[ry])
            parent[rx] = ry;
        else if (rank[rx] > rank[ry])
            parent[ry] = rx;
        else {
            parent[ry] = rx;
            rank[rx]++;
        }
    }
    
    bool same(int x, int y) {
        return find(x) == find(y);
    }
};
```

public class UnionFind {
    private int[] parent, rank;
    
    public UnionFind(int n) {
        parent = new int[n];
        rank = new int[n];
        for (int i = 0; i < n; i++) parent[i] = i;
    }
    
    public int find(int x) {
        if (parent[x] != x)
            parent[x] = find(parent[x]);
        return parent[x];
    }
    
    public void union(int x, int y) {
        int rx = find(x), ry = find(y);
        if (rx == ry) return;
        if (rank[rx] < rank[ry]) parent[rx] = ry;
        else if (rank[rx] > rank[ry]) parent[ry] = rx;
        else { parent[ry] = rx; rank[rx]++; }
    }
}

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    
    def union(self, x, y):
        rx, ry = self.find(x), self.find(y)
        if rx == ry: return
        if self.rank[rx] < self.rank[ry]:
            self.parent[rx] = ry
        elif self.rank[rx] > self.rank[ry]:
            self.parent[ry] = rx
        else:
            self.parent[ry] = rx
            self.rank[rx] += 1
```
