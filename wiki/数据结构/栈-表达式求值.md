## 核心定义

表达式求值是栈的经典应用。计算机对算术表达式的求值通常采用"算符优先法"或"后缀表达式法"。中缀表达式（如 a+b*c）需要处理运算符优先级和括号，计算机通过栈来高效完成。后缀表达式（也叫逆波兰表达式，RPN）不需要括号，运算符出现在操作数之后（如 abc*+），非常适合计算机处理。算法核心思路：(1) 中缀转后缀——操作数直接输出，运算符入栈前比较优先级，遇到 '(' 无条件入栈，遇到 ')' 依次弹出直到 '('；(2) 后缀求值——操作数入栈，遇到运算符时弹出两个操作数计算后将结果入栈。整个过程使用了栈的后进先出的特性。

## 关键结论

- 中缀转后缀使用运算符栈，后缀求值使用操作数栈
- 运算符优先级需要明确定义：'*''/' > '+''-'，'('在栈外优先级最高，在栈内优先级最低
- 后缀表达式求值只需一次扫描，每遇到运算符就弹出栈顶两个操作数运算并入栈
- 时间复杂度为 O(n)，空间复杂度为 O(n)，n 为表达式长度
- 前缀表达式（波兰表达式）是中缀转后缀的对称形式，运算符在操作数之前

## 易错点

1. 中缀转后缀时遇到 '(' 的处理：'(' 在扫描到时应无条件入栈，但其内部栈优先级应设为最低（这样后面的运算符才能正常入栈而不是弹出 '('）；遇到 ')' 时要弹出到 '(' 并丢弃这对括号
2. 操作数可能是多位数：中缀转后缀时需要在操作数之间添加分隔符或在输出时做处理，否则 "12+3" 会混淆为 "1,2,3"
3. 右结合运算符（如幂运算 ^）的处理：当连续遇到同等优先级的右结合运算符时不应弹出栈内运算符，这与左结合不同

## 例题

**例1：** 将中缀表达式 `a+b*c+(d*e+f)*g` 转换为后缀表达式。

**解答：** `abc*+de*f+g*+`。逐步分析：a 输出，+ 入栈；b 输出，* 入栈；c 输出，遇到 +，* 出栈，+ 出栈，+ 入栈；( 入栈；d 输出，* 入栈；e 输出，遇到 +，* 出栈，+ 入栈；f 输出，遇到 )，弹出至 (；* 入栈；g 输出；最后清空栈。

## 代码示例

```cpp
#include <iostream>
#include <stack>
#include <string>
#include <unordered_map>
using namespace std;

// 得到运算符的优先级
int priority(char op) {
    if (op == '*' || op == '/') return 2;
    if (op == '+' || op == '-') return 1;
    return 0;
}

// 中缀表达式转后缀表达式
string infixToPostfix(string infix) {
    stack<char> ops;
    string postfix;
    for (char c : infix) {
        if (isdigit(c) || isalpha(c)) {
            postfix += c;
        } else if (c == '(') {
            ops.push(c);
        } else if (c == ')') {
            while (!ops.empty() && ops.top() != '(') {
                postfix += ops.top(); ops.pop();
            }
            if (!ops.empty()) ops.pop();  // 弹出 '('
        } else {
            // c 是运算符
            while (!ops.empty() && ops.top() != '(' &&
                   priority(ops.top()) >= priority(c)) {
                postfix += ops.top(); ops.pop();
            }
            ops.push(c);
        }
    }
    while (!ops.empty()) {
        postfix += ops.top(); ops.pop();
    }
    return postfix;
}

// 后缀表达式求值（操作数为一位数字）
int evalPostfix(string postfix) {
    stack<int> nums;
    for (char c : postfix) {
        if (isdigit(c)) {
            nums.push(c - '0');
        } else {
            int b = nums.top(); nums.pop();
            int a = nums.top(); nums.pop();
            switch (c) {
                case '+': nums.push(a + b); break;
                case '-': nums.push(a - b); break;
                case '*': nums.push(a * b); break;
                case '/': nums.push(a / b); break;
            }
        }
    }
    return nums.top();
}
```

```java
public class ExpressionEvaluator {
    // 中缀转后缀
    public static String infixToPostfix(String infix) {
        StringBuilder sb = new StringBuilder();
        Stack<Character> ops = new Stack<>();
        for (char c : infix.toCharArray()) {
            if (Character.isDigit(c) || Character.isLetter(c)) {
                sb.append(c);
            } else if (c == '(') {
                ops.push(c);
            } else if (c == ')') {
                while (ops.peek() != '(') sb.append(ops.pop());
                ops.pop();
            } else {
                while (!ops.isEmpty() && ops.peek() != '(' &&
                       priority(ops.peek()) >= priority(c))
                    sb.append(ops.pop());
                ops.push(c);
            }
        }
        while (!ops.isEmpty()) sb.append(ops.pop());
        return sb.toString();
    }
    
    private static int priority(char op) {
        if (op == '*' || op == '/') return 2;
        if (op == '+' || op == '-') return 1;
        return 0;
    }
}
```
