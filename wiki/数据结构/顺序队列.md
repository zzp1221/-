## 核心定义

顺序队列（Sequential Queue）是利用顺序存储结构（数组）实现的队列。队列（Queue）是一种操作受限的线性表，只允许在一端（队尾 rear）进行插入（入队 enqueue），在另一端（队头 front）进行删除（出队 dequeue），遵循先进先出（FIFO, First In First Out）的原则。顺序队列通常设置两个指针：front 指向队头元素的前一个位置（或队头元素本身），rear 指向队尾元素（或队尾的下一个位置）。入队时 rear 后移并将元素放入，出队时 front 后移并取出元素。普通的顺序队列存在"假溢出"问题——front 和 rear 不断后移，即使数组前方有空位也无法使用。

## 关键结论

- 普通顺序队列存在"假溢出"：front 前面的空间被废弃
- 入队和出队操作时间复杂度均为 O(1)
- 顺序队列需要判断队空和队满的条件，不同约定下条件不同
- 若 rear 指向实际队尾元素，则队满为 rear == MAXSIZE-1；若 rear 指向队尾下一个位置，判断更复杂
- 解决假溢出的方法是使用循环队列

## 易错点

1. 假溢出与真溢出混淆：假溢出是 front 前有空位但不能用（队列未满但无法入队）；真溢出是队列确实满了（所有位置都有元素）。这是普通顺序队列的根本缺陷
2. 初始状态设置：front 和 rear 的初始值约定不一致（同为 -1 或同为 0），需根据教材统一约定理解
3. 出队后数据并未被清除：出队只是移动 front 指针，原位置的数据仍保留在数组中，通常不需要专门清除

## 例题

**例1：** 在最大容量为 M 的普通顺序队列中（front 指向队头前一位，rear 指向队尾），队列中元素个数的计算公式是？

**解答：** 元素个数 = rear - front。队空条件为 front == rear。

## 代码示例

```cpp
#include <iostream>
#define MAXSIZE 100
using namespace std;

class SeqQueue {
private:
    int data[MAXSIZE];
    int front;  // 队头指针（指向队头元素的前一个位置）
    int rear;   // 队尾指针（指向队尾元素位置）
public:
    SeqQueue() { front = rear = -1; }
    
    bool isEmpty() { return front == rear; }
    
    bool isFull() { return rear >= MAXSIZE - 1; }
    
    bool enqueue(int e) {
        if (isFull()) return false;
        data[++rear] = e;
        return true;
    }
    
    bool dequeue(int& e) {
        if (isEmpty()) return false;
        e = data[++front];
        return true;
    }
};
```

```java
public class SeqQueue {
    private int[] data;
    private int front, rear;
    private static final int MAXSIZE = 100;
    
    public SeqQueue() {
        data = new int[MAXSIZE];
        front = rear = -1;
    }
    
    public boolean isEmpty() { return front == rear; }
    public boolean isFull() { return rear >= MAXSIZE - 1; }
    
    public boolean enqueue(int e) {
        if (isFull()) return false;
        data[++rear] = e;
        return true;
    }
    
    public Integer dequeue() {
        if (isEmpty()) return null;
        return data[++front];
    }
}
```
