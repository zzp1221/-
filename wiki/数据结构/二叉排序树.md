## 核心定义

二叉排序树（Binary Search Tree，BST）或者是一棵空树，或者是具有以下性质的二叉树：(1) 若左子树不空，则左子树上所有结点的值均小于根结点的值；(2) 若右子树不空，则右子树上所有结点的值均大于根结点的值；(3) 左、右子树也分别是二叉排序树。BST 的中序遍历序列是一个递增有序序列。BST 的查找过程类似二分查找：从根开始，若等于则找到；若小于则去左子树；若大于则去右子树。BST 的平均查找时间复杂度为 O(log n)，但最坏情况下（插入有序序列时退化成单链表）为 O(n)。BST 的插入操作总是在叶子位置进行。

## 关键结论

- BST 的中序遍历得到递增有序序列
- 查找/插入的平均时间复杂度为 O(log n)，最坏为 O(n)（退化链表）
- BST 删除分三种情况：叶子（直接删）、单孩子（子承父业）、双孩子（用中序后继替代并递归删除）
- 新插入的结点一定是叶子结点
- BST 的形态取决于插入顺序，相同数据不同插入顺序会得到不同形态的 BST

## 易错点

1. BST 中不能有等值结点：若插入值与现有结点相等，应视为失败（或根据需求处理）。BST 的定义明确"小于"和"大于"，不含"等于"
2. 删除双孩子结点时的后继选择：中序后继（右子树的最左下结点）替代后，还需要递归删除那个后继（它至多有一个右孩子）。不能只改数据不删结点
3. 插入操作需要一个"前驱指针"：遍历查找插入位置时，需要用 pre 记录当前结点的双亲，最终插入为 pre 的左孩子或右孩子

## 例题

**例1：** 依次插入关键字 {50, 30, 80, 20, 40, 70, 90, 35, 45} 构造 BST，写出其中序遍历序列。

**解答：** 中序遍历序列：20, 30, 35, 40, 45, 50, 70, 80, 90（递增有序）。树的结构为：50 为根，30 和 80 为左右孩子，20 和 40 为 30 的左右孩子，35 和 45 为 40 的左右孩子，70 和 90 为 80 的左右孩子。

## 代码示例

```cpp
template<typename T>
struct BSTNode {
    T key;
    BSTNode *left, *right;
    BSTNode(T k) : key(k), left(nullptr), right(nullptr) {}
};

template<typename T>
class BST {
    BSTNode<T>* root;
    
    BSTNode<T>* insert(BSTNode<T>* node, T key) {
        if (!node) return new BSTNode<T>(key);
        if (key < node->key)
            node->left = insert(node->left, key);
        else if (key > node->key)
            node->right = insert(node->right, key);
        return node;
    }
    
    BSTNode<T>* search(BSTNode<T>* node, T key) {
        if (!node || node->key == key) return node;
        if (key < node->key)
            return search(node->left, key);
        else
            return search(node->right, key);
    }
    
    BSTNode<T>* remove(BSTNode<T>* node, T key) {
        if (!node) return node;
        if (key < node->key)
            node->left = remove(node->left, key);
        else if (key > node->key)
            node->right = remove(node->right, key);
        else {
            // 找到要删除的结点
            if (!node->left) {
                BSTNode<T>* r = node->right;
                delete node;
                return r;
            } else if (!node->right) {
                BSTNode<T>* l = node->left;
                delete node;
                return l;
            }
            // 有两个孩子：找中序后继
            BSTNode<T>* succ = node->right;
            while (succ->left) succ = succ->left;
            node->key = succ->key;
            node->right = remove(node->right, succ->key);
        }
        return node;
    }
    
public:
    BST() : root(nullptr) {}
    void insert(T key) { root = insert(root, key); }
    BSTNode<T>* search(T key) { return search(root, key); }
    void remove(T key) { root = remove(root, key); }
};
```

```java
public class BST<T extends Comparable<T>> {
    class Node {
        T key; Node left, right;
        Node(T k) { key = k; }
    }
    Node root;
    
    public void insert(T key) { root = insert(root, key); }
    
    private Node insert(Node node, T key) {
        if (node == null) return new Node(key);
        int cmp = key.compareTo(node.key);
        if (cmp < 0) node.left = insert(node.left, key);
        else if (cmp > 0) node.right = insert(node.right, key);
        return node;
    }
    
    public Node search(T key) { return search(root, key); }
    
    private Node search(Node node, T key) {
        if (node == null || key.equals(node.key)) return node;
        return key.compareTo(node.key) < 0
            ? search(node.left, key) : search(node.right, key);
    }
}
```
