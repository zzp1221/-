## 核心定义

拓扑排序（Topological Sort）是将有向无环图（DAG）中所有顶点排成一个线性序列，使得对于图中任意一条有向边 (u, v)，u 在序列中都出现在 v 的前面。这种序列称为拓扑序列。AOV 网（Activity On Vertex）是一种用顶点表示活动、用有向边表示活动间的优先关系的 DAG。拓扑排序的典型算法有两种：(1) Kahn 算法（BFS 思想）——计算每个顶点的入度，将所有入度为 0 的顶点入队，反复弹出队头并将其后继的入度减 1（若减后入度为 0 则入队），直到队列为空；若输出的顶点数小于 n 则说明图中有环。(2) DFS 算法——对 DAG 做后序遍历，将顶点按完成时间逆序排列即得拓扑序。时间复杂度均为 O(n+e)。

## 关键结论

- 拓扑序列不唯一（当有多个入度为 0 的顶点时）
- AOV 网中不能有环，有环则无法拓扑排序
- Kahn 算法通过不断删除入度为 0 的顶点来实现
- DFS 后序逆序也可得到拓扑排序
- 时间复杂度为 O(n+e)，适用于检测死锁依赖、编译依赖分析等

## 易错点

1. 拓扑排序的前提是有向无环图（DAG）：如果图中存在环，拓扑排序失败。Kahn 算法结束后小于 n 个顶点输出即有环
2. Kahn 算法中每次取入度为 0 的顶点：若使用栈则为 DFS 式输出，使用队列为 BFS 式输出；不同容器会影响序列但都是合法拓扑序
3. 拓扑序列不是唯一的：当多个顶点入度同时为 0 时选择哪个都可以

## 例题

**例1：** 对以下课程依赖图做拓扑排序：C1 无先修，C2 依赖 C1，C3 依赖 C1，C4 依赖 C2 和 C3，C5 依赖 C4。写出一个合法的选修顺序。

**解答：** 一种可能顺序：C1 -> C2 -> C3 -> C4 -> C5（也可是 C1 -> C3 -> C2 -> C4 -> C5）。

## 代码示例

```cpp
#include <vector>
#include <queue>
using namespace std;

// Kahn 算法（BFS 法）
vector<int> topologicalSort(int n, vector<vector<int>>& adj) {
    vector<int> indegree(n, 0);
    for (int u = 0; u < n; u++)
        for (int v : adj[u])
            indegree[v]++;
    
    queue<int> q;
    for (int i = 0; i < n; i++)
        if (indegree[i] == 0) q.push(i);
    
    vector<int> order;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        order.push_back(u);
        for (int v : adj[u]) {
            if (--indegree[v] == 0)
                q.push(v);
        }
    }
    // 若 order.size() < n，说明有环
    return order.size() == n ? order : vector<int>();
}

// DFS 法
vector<int> topoSortDFS(int n, vector<vector<int>>& adj) {
    vector<int> visited(n, 0);  // 0=未访, 1=访问中, 2=已处理
    vector<int> order;
    
    function<bool(int)> dfs = [&](int u) -> bool {
        visited[u] = 1;
        for (int v : adj[u]) {
            if (visited[v] == 1) return false;  // 发现环
            if (visited[v] == 0 && !dfs(v)) return false;
        }
        visited[u] = 2;
        order.push_back(u);
        return true;
    };
    
    for (int i = 0; i < n; i++)
        if (visited[i] == 0 && !dfs(i))
            return {};
    
    reverse(order.begin(), order.end());
    return order;
}
```
