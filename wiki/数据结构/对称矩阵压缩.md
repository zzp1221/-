## 核心定义

对称矩阵是指满足 a[i][j] == a[j][i] 的 n 阶方阵。由于对称性，可以只存储上三角（或下三角）部分的元素（包括主对角线），从而将原来需要 n^2 个存储单元压缩到 n(n+1)/2 个，几乎节省了一半空间。通常将下三角（含对角线）按行优先顺序存储在一维数组中：对于 a[i][j]（i >= j），其下标 k = i(i-1)/2 + (j-1)（下标从 1 开始，转换到 0 起始为 k = i(i+1)/2 + j）。若访问上三角元素 a[i][j]（i < j），则利用对称性转换为 a[j][i] 来访问。同类概念还有三角矩阵（上三角或下三角为同一常数的矩阵）和对角矩阵（如三对角矩阵）。

## 关键结论

- 对称矩阵可压缩到 n(n+1)/2 个元素，节省约一半空间
- 访问下三角元素 a[i][j]（i >= j，下标从 0 起）：k = i*(i+1)/2 + j
- 访问上三角元素 a[i][j]（i < j）：利用对称性，访问 a[j][i]，k = j*(j+1)/2 + i
- 三对角矩阵（只有主对角线和两条次对角线非零）可压缩到 3n-2 个元素
- 三角矩阵存储时通常额外用一个单元存放常数

## 易错点

1. 下标转换公式中的起始值：公式 k = i(i-1)/2 + (j-1) 是基于下标从 1 开始的；若数组从 0 开始，公式应为 k = i(i+1)/2 + j
2. 混淆上三角和下三角的访问方式：对下三角存储的矩阵，访问 a[i][j]（i < j，即上三角）时必须转换为访问 a[j][i]（下三角位置），不可直接带入公式
3. 三角矩阵的常数域：上三角矩阵所有上三角元素相同，存储时最后一个单元保存该常数；访问时需判断所在的三角区域来选择不同的寻址公式

## 例题

**例1：** 设 5 阶对称矩阵 A 按行优先顺序将其下三角（含对角线）元素存入一维数组 B 中。若 a[3][2]（下标从 1 开始）在 B 中的位置是 5，求 a[5][3] 的位置。

**解答：** 按公式 k = i(i-1)/2 + (j-1)，a[3][2] 即 i=3, j=2，k = 3*2/2 + 1 = 4（下标从 0 开始）。题中下标从 1 开始所以位置为 5。对于 a[5][3]：i=5, j=3，由于 i > j 直接可访问，k = 5*4/2 + 2 = 12（下标从 0），位置为 13。

## 代码示例

```cpp
#include <iostream>
#include <vector>
using namespace std;

class SymmetricMatrix {
private:
    vector<int> data;
    int n;  // 矩阵阶数
public:
    SymmetricMatrix(int size) : n(size), data(size * (size + 1) / 2) {}
    
    // k = i*(i+1)/2 + j (i >= j, 下三角)
    // k = j*(j+1)/2 + i (i < j, 利用对称性)
    int index(int i, int j) const {
        if (i >= j)
            return i * (i + 1) / 2 + j;
        else
            return j * (j + 1) / 2 + i;
    }
    
    void set(int i, int j, int val) {
        data[index(i, j)] = val;
    }
    
    int get(int i, int j) const {
        return data[index(i, j)];
    }
};
```

```java
public class SymmetricMatrix {
    private int[] data;
    private int n;
    
    public SymmetricMatrix(int size) {
        n = size;
        data = new int[n * (n + 1) / 2];
    }
    
    private int index(int i, int j) {
        return i >= j ? i * (i + 1) / 2 + j
                      : j * (j + 1) / 2 + i;
    }
    
    public void set(int i, int j, int val) {
        data[index(i, j)] = val;
    }
    
    public int get(int i, int j) {
        return data[index(i, j)];
    }
}
```
