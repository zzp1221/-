## 核心定义

关键路径（Critical Path）是 AOE 网（Activity On Edge）中的核心概念。AOE 网是一种带权的有向无环图，其中顶点表示事件（Event），有向边表示活动（Activity），边上的权值表示活动持续的时间。AOE 网通常用来估算工程的完成时间。关键路径是指从工程开始顶点（源点）到结束顶点（汇点）的所有路径中长度最长的那一条。关键路径上的活动（关键活动）一旦延误，整个工程的完成时间就会延误。计算关键路径需要四个基本量：(1) ve(i)——事件 i 的最早发生时间（从源点正向拓扑计算）；(2) vl(i)——事件 i 的最晚发生时间（从汇点反向拓扑计算）；(3) e(k)——活动 k 的最早开始时间 = ve(起点事件)；(4) l(k)——活动 k 的最晚开始时间 = vl(终点事件) - 活动时长。当 e(k) == l(k) 时，活动 k 为关键活动。

## 关键结论

- 关键路径是 AOE 网中从源点到汇点的最长路径
- 求 ve 按拓扑序正向推导（选最大）；求 vl 按逆拓扑序反向推导（选最小）
- e(i) == l(i) 的活动为关键活动，关键活动组成关键路径
- 关键路径可能有多条（多条最长路径）
- 缩短非关键活动不能缩短总工期；缩短关键活动才有效
- 关键路径算法的时间复杂度为 O(n+e)

## 易错点

1. ve 和 vl 的计算方向：ve 正向（从源到汇，取 max），vl 反向（从汇到源，取 min）。方向搞反会全错
2. 初始化 vl(n-1) = ve(n-1)：即汇点的最晚发生时间等于最早发生时间，这是反向计算的起点
3. 多个前驱/后继时的处理：ve(j) = max{ve(i) + w(i,j) | (i,j) 是边}，取最大值；vl(i) = min{vl(j) - w(i,j) | (i,j) 是边}，取最小值

## 例题

**例1：** AOE 网中有 6 个事件（V1-V6），活动及持续时间为：(V1,V2)=3, (V1,V3)=2, (V2,V4)=2, (V2,V5)=3, (V3,V4)=4, (V3,V6)=3, (V4,V6)=2, (V5,V6)=1。求关键路径和工程最短完成时间。

**解答：** 先拓扑排序，计算 ve：ve(V1)=0; ve(V2)=3; ve(V3)=2; ve(V4)=max(3+2, 2+4)=6; ve(V5)=3+3=6; ve(V6)=max(6+2, 6+1, 2+3)=8。再逆拓扑计算 vl：vl(V6)=8; vl(V5)=8-1=7; vl(V4)=8-2=6; vl(V3)=min(6-4, 8-3)=2; vl(V2)=min(6-2, 7-3)=4; vl(V1)=min(4-3, 2-2)=0。e(活动)=ve(起点), l(活动)=vl(终点)-时长。e==l 的关键活动：(V1,V3), (V3,V4), (V4,V6)。关键路径：V1->V3->V4->V6，总工期 8。

## 代码示例

```cpp
#include <vector>
#include <queue>
#include <stack>
using namespace std;

struct Edge { int to, weight; };

void criticalPath(int n, vector<vector<Edge>>& adj,
                  vector<int>& ve, vector<int>& vl, vector<bool>& isCritical) {
    // 1. 计算 indegree，拓扑排序求 ve
    vector<int> indegree(n, 0);
    for (int u = 0; u < n; u++)
        for (auto& e : adj[u])
            indegree[e.to]++;
    
    stack<int> topo;  // 拓扑序列（用于后续逆序计算 vl）
    queue<int> q;
    for (int i = 0; i < n; i++)
        if (indegree[i] == 0) q.push(i);
    
    ve.assign(n, 0);
    while (!q.empty()) {
        int u = q.front(); q.pop();
        topo.push(u);
        for (auto& e : adj[u]) {
            ve[e.to] = max(ve[e.to], ve[u] + e.weight);
            if (--indegree[e.to] == 0) q.push(e.to);
        }
    }
    
    // 2. 初始化 vl
    vl.assign(n, ve[n-1]);  // 汇点 vl = ve
    
    // 3. 逆拓扑序计算 vl
    while (!topo.empty()) {
        int u = topo.top(); topo.pop();
        for (auto& e : adj[u])
            vl[u] = min(vl[u], vl[e.to] - e.weight);
    }
    
    // 4. 判断关键活动
    for (int u = 0; u < n; u++) {
        for (auto& e : adj[u]) {
            int earliest = ve[u];
            int latest = vl[e.to] - e.weight;
            if (earliest == latest)
                isCritical[u] = true;  // 简化：标记关键事件
        }
    }
}
```
