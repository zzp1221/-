## 核心定义

循环队列（Circular Queue）是顺序队列的一种改进形式，用于解决普通顺序队列的"假溢出"问题。它把顺序存储的数组在逻辑上看成一个首尾相连的环，当 rear 或 front 到达数组末尾（MAXSIZE-1）后，通过取模运算 `(index + 1) % MAXSIZE` 使它们回到数组开头。这样，只要队列没有真正满，新元素就可以入队，充分利用数组空间。循环队列的入队和出队操作时间复杂度均为 O(1)。由于队空和队满时 front == rear 均可能成立，通常采用以下三种方式之一来区分：(1) 牺牲一个存储单元，即当 (rear+1)%MAXSIZE == front 时认为队满；(2) 增设一个表示队列大小的变量 size；(3) 增设一个 tag 标记最后一次操作是入队还是出队。

## 关键结论

- 循环队列通过取模运算实现 rear 和 front 的循环移动，消除假溢出
- 队满条件：`(rear + 1) % MAXSIZE == front`（牺牲一个单元时）
- 队空条件：`front == rear`
- 队列长度：`(rear - front + MAXSIZE) % MAXSIZE`
- 入队：`rear = (rear + 1) % MAXSIZE; data[rear] = e;`（rear 指向队尾元素时）
- 出队：`front = (front + 1) % MAXSIZE; e = data[front];`
- 实际可用空间为 MAXSIZE - 1（牺牲一个单元时）

## 易错点

1. 队空和队满的条件混淆：二者都是 front == rear 时，但一个空闲一个满载。若使用"牺牲一个单元"方式，队满实际上是 front == (rear+1)%MAXSIZE，队空仍然是 front == rear
2. 取模时忘记加 MAXSIZE：求队列长度时，公式为 (rear - front + MAXSIZE) % MAXSIZE，若不加 MAXSIZE 直接取模，rear < front 时会得到负数
3. rear 指针的语义约定：若 rear 指向队尾元素，入队时先移 rear 再放入；若 rear 指向队尾下一个空位，入队时先放再移 rear；要注意 front 与 rear 初始值须对应

## 例题

**例1：** 设循环队列的最大容量为 6（数组下标 0-5），牺牲一个单元区分空满。当前 front=0, rear=4，问最多还能入队几个元素？

**解答：** 队列长度为 (4-0+6)%6 = 4。最大容量为 5（牺牲一个单元），所以还能入 5-4=1 个元素。

## 代码示例

```cpp
#include <iostream>
#define MAXSIZE 6
using namespace std;

class CircularQueue {
private:
    int data[MAXSIZE];
    int front, rear;  // front 指向队头前一位，rear 指向队尾
public:
    CircularQueue() { front = rear = 0; }
    
    bool isEmpty() { return front == rear; }
    
    bool isFull() {
        return (rear + 1) % MAXSIZE == front;
    }
    
    bool enqueue(int e) {
        if (isFull()) return false;
        rear = (rear + 1) % MAXSIZE;
        data[rear] = e;
        return true;
    }
    
    bool dequeue(int& e) {
        if (isEmpty()) return false;
        front = (front + 1) % MAXSIZE;
        e = data[front];
        return true;
    }
    
    int size() {
        return (rear - front + MAXSIZE) % MAXSIZE;
    }
};
```

```java
public class CircularQueue {
    private int[] data;
    private int front, rear;
    private final int MAXSIZE = 100;
    
    public CircularQueue() {
        data = new int[MAXSIZE];
        front = rear = 0;
    }
    
    public boolean isEmpty() { return front == rear; }
    
    public boolean isFull() {
        return (rear + 1) % MAXSIZE == front;
    }
    
    public boolean enqueue(int e) {
        if (isFull()) return false;
        rear = (rear + 1) % MAXSIZE;
        data[rear] = e;
        return true;
    }
    
    public Integer dequeue() {
        if (isEmpty()) return null;
        front = (front + 1) % MAXSIZE;
        return data[front];
    }
}
```
