## 核心定义

括号匹配是栈的最经典和最简单的应用之一。在编译原理中，编译器需要检查源程序中的括号（包括圆括号 ()、方括号 []、花括号 {} ）是否正确匹配。算法使用一个栈，从左到右扫描输入字符串：遇到左括号则将其压入栈中；遇到右括号时，检查栈顶元素是否为对应的左括号，若是则弹出栈顶，若否则说明括号不匹配。扫描结束后，若栈为空则括号完全匹配，否则说明有未闭合的左括号。算法的时间复杂度为 O(n)，空间复杂度为 O(n)，其中 n 为输入字符串的长度。

## 关键结论

- 括号匹配使用栈的后进先出特性，最内层的左括号最先被匹配
- 算法的三部曲：左括号入栈、右括号匹配栈顶、最终检查栈是否为空
- 时间复杂度 O(n)，只需一次线性扫描
- 可以扩展用于 XML/HTML 标签匹配、代码块界定符匹配等
- 三种括号可以混合处理，只需在匹配时判断类型是否对应即可

## 易错点

1. 只判断左右括号数量相等：这是不充分的！例如 ")(" 左右数量相等但显然不匹配；必须同时满足"右括号与最近未匹配左括号类型一致"
2. 遇到右括号时忘记检查栈是否为空：若栈已空但仍有右括号，说明右括号多余，应立即返回不匹配，否则访问栈顶会出错
3. 匹配判断时采用多个 if-else 而非映射表：当括号种类较多时，建议用哈希表或映射数组（如 char[128]）来存放配对关系，代码更简洁不易出错

## 例题

**例1：** 判断字符串 "{[()]}" 的括号是否匹配。

**解答：** 扫描过程：{ 入栈，[ 入栈，( 入栈；遇到 )，栈顶是 (，匹配，弹出；遇到 ]，栈顶是 [，匹配，弹出；遇到 }，栈顶是 {，匹配，弹出。栈为空，完全匹配。

## 代码示例

```cpp
#include <iostream>
#include <stack>
#include <string>
using namespace std;

bool isBracketMatched(const string& s) {
    stack<char> st;
    for (char c : s) {
        if (c == '(' || c == '[' || c == '{') {
            st.push(c);
        } else if (c == ')' || c == ']' || c == '}') {
            if (st.empty()) return false;
            char top = st.top();
            // 检查是否匹配
            if ((c == ')' && top != '(') ||
                (c == ']' && top != '[') ||
                (c == '}' && top != '{'))
                return false;
            st.pop();
        }
    }
    return st.empty();
}
```

```java
public class BracketMatcher {
    public static boolean isMatched(String s) {
        Stack<Character> stack = new Stack<>();
        for (char c : s.toCharArray()) {
            if (c == '(' || c == '[' || c == '{') {
                stack.push(c);
            } else if (c == ')' || c == ']' || c == '}') {
                if (stack.isEmpty()) return false;
                char top = stack.pop();
                if ((c == ')' && top != '(') ||
                    (c == ']' && top != '[') ||
                    (c == '}' && top != '{'))
                    return false;
            }
        }
        return stack.isEmpty();
    }
}
```

```python
def is_bracket_matched(s):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}
    for c in s:
        if c in '([{':
            stack.append(c)
        elif c in ')]}':
            if not stack or stack[-1] != pairs[c]:
                return False
            stack.pop()
    return len(stack) == 0
```
