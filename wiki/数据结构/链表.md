## 核心定义

链表（Linked List）是线性表的链式存储结构。它通过一组任意的存储单元（可以不连续）来存储线性表中的数据元素，每个结点包含两个部分：数据域（存储元素本身）和指针域（存储指向后继结点的指针）。单链表是最基本的链表形式，每个结点只有一个指向后继的指针。通常设置一个头结点来简化操作（插入和删除时无需特殊处理表头情况）。链表通过指针来维护元素之间的逻辑关系，因此插入和删除操作只需修改指针即可，时间复杂度为 O(1)（若已知插入位置）。但访问元素需要从头遍历，时间复杂度为 O(n)。链表是一种动态数据结构，不需要预分配空间，链表的长度理论上仅受限于内存大小。

## 关键结论

- 链表不支持随机存取，按位置访问需从头遍历，时间复杂度 O(n)
- 在已知位置的前提下，插入和删除的时间复杂度为 O(1)（只需修改指针）
- 链表存储密度小于 1，因为每个结点需要额外空间存储指针
- 链表天然具有动态扩展能力，不会出现顺序表的空间浪费问题
- 带头结点的链表可以使代码统一，避免对空表和首元素操作的特殊处理

## 易错点

1. 插入操作的指针顺序错误：在结点 p 之后插入结点 s，必须先将 s->next = p->next，再 p->next = s；顺序颠倒会导致 p 后面的链表丢失
2. 删除时忘记释放内存：删除结点时，需先用临时指针保存目标结点，修改链接后再 free/delete 释放内存，否则会导致内存泄漏
3. 头指针与头结点混淆：头指针指向链表的第一个结点（或头结点）；头结点是人为添加的非数据结点，其指针域指向首元结点；二者必须区分清楚

## 例题

**例1：** 在单链表中，删除指针 p 所指结点的后继结点，写出核心操作语句。

**解答：** `q = p->next; p->next = q->next; free(q);` 先用 q 保存待删结点，修改指针绕过它，再释放内存。

**例2：** 带头结点的单链表为空的条件是什么？

**解答：** `head->next == NULL`，即头结点的指针域为空。

## 代码示例

```cpp
#include <iostream>
using namespace std;

template<typename T>
struct Node {
    T data;
    Node* next;
    Node(T val, Node* n = nullptr) : data(val), next(n) {}
};

template<typename T>
class LinkedList {
private:
    Node<T>* head;  // 头结点
public:
    LinkedList() { head = new Node<T>(T()); }
    
    ~LinkedList() {
        while (head) {
            Node<T>* p = head;
            head = head->next;
            delete p;
        }
    }
    
    // 头插法
    void insertHead(T val) {
        Node<T>* s = new Node<T>(val, head->next);
        head->next = s;
    }
    
    // 尾插法
    void insertTail(T val) {
        Node<T>* p = head;
        while (p->next) p = p->next;
        p->next = new Node<T>(val);
    }
    
    // 删除第一个值为 val 的结点
    bool remove(T val) {
        Node<T>* p = head;
        while (p->next && p->next->data != val)
            p = p->next;
        if (!p->next) return false;
        Node<T>* q = p->next;
        p->next = q->next;
        delete q;
        return true;
    }
};
```

```java
public class LinkedList<T> {
    class Node {
        T data;
        Node next;
        Node(T data, Node next) {
            this.data = data; this.next = next;
        }
    }
    private Node head;
    
    public LinkedList() {
        head = new Node(null, null);
    }
    
    public void insertHead(T val) {
        head.next = new Node(val, head.next);
    }
    
    public void insertTail(T val) {
        Node p = head;
        while (p.next != null) p = p.next;
        p.next = new Node(val, null);
    }
    
    public boolean remove(T val) {
        Node p = head;
        while (p.next != null && !p.next.data.equals(val))
            p = p.next;
        if (p.next == null) return false;
        p.next = p.next.next;
        return true;
    }
}
```
