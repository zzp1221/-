## 核心定义

稀疏矩阵（Sparse Matrix）是指非零元素个数远少于零元素个数（通常非零元素占比小于5%）的矩阵。由于非零元素分布没有规律，稀疏矩阵的压缩存储只存储非零元素的信息。最常用的方法有：(1) 三元组顺序表——每个非零元素用 (row, col, value) 三元组表示，按行优先（或列优先）顺序存储在一维数组中；(2) 十字链表——行和列各组织成一条带头结点的循环链表，每个非零元素结点同时位于所在行链表和所在列链表上。存储稀疏矩阵可以节省大量内存，但矩阵的转置、加法、乘法等运算需要重写。

## 关键结论

- 三元组表示从 (row, col, value) 三元组序列，通常按行优先顺序存储
- 普通三元组转置时间复杂度 O(n*t)（n 为列数，t 为非零元个数），快速转置可优化到 O(n+t)
- 快速转置思想：先统计每列非零元素个数得到位置数组，然后一次性定位
- 十字链表适合需要频繁插入删除或动态修改的稀疏矩阵运算
- 稀疏矩阵的乘法运算比较复杂，需处理累加值可能为零的情况

## 易错点

1. 三元组的下标起始：有些教材三元组从 0 开始索引，有些从 1 开始（如 C 语言教材），写代码时要注意一致性
2. 快速转置中两个数组的用途：num[col] 统计每列非零元个数，position[col] 定位每列第一个非零元在转置后的位置；"position[col] = position[col-1] + num[col-1]" 是关键递推公式
3. 十字链表插入时行列链表都要修改：每个非零元素结点需要同时挂到行链表和列链表上，插入时要维护两个方向

## 例题

**例1：** 已知稀疏矩阵 A 的三元组为 [(1,2,5), (2,1,3), (2,3,7), (3,1,2)]，写出 A 的快速转置过程。

**解答：** 列数 n=3，统计各列非零元个数 num[1]=2, num[2]=1, num[3]=1；计算位置 position[1]=1, position[2]=3, position[3]=4；转置得到三元组 [(1,2,3), (1,3,2), (2,1,5), (3,2,7)]。

## 代码示例

```cpp
#include <iostream>
#include <vector>
using namespace std;

struct Triple {
    int row, col, value;
    Triple(int r, int c, int v) : row(r), col(c), value(v) {}
};

class SparseMatrix {
    vector<Triple> data;
    int rows, cols;
public:
    SparseMatrix(int r, int c) : rows(r), cols(c) {}
    
    void add(int r, int c, int v) {
        if (v != 0) data.emplace_back(r, c, v);
    }
    
    // 快速转置
    SparseMatrix fastTranspose() {
        SparseMatrix result(cols, rows);
        vector<int> num(cols + 1, 0);
        vector<int> pos(cols + 1, 1);
        // 统计每列非零元个数
        for (auto& t : data)
            num[t.col]++;
        // 计算每列第一个非零元的位置
        for (int i = 2; i <= cols; i++)
            pos[i] = pos[i - 1] + num[i - 1];
        // 转置
        result.data.resize(data.size());
        for (auto& t : data) {
            int idx = pos[t.col]++;
            result.data[idx - 1] = Triple(t.col, t.row, t.value);
        }
        return result;
    }
};
```

```java
public class SparseMatrix {
    static class Triple {
        int row, col, value;
        Triple(int r, int c, int v) { row = r; col = c; value = v; }
    }
    List<Triple> data = new ArrayList<>();
    int rows, cols;
    
    public SparseMatrix fastTranspose() {
        SparseMatrix res = new SparseMatrix(cols, rows);
        int[] num = new int[cols + 1];
        int[] pos = new int[cols + 1];
        Arrays.fill(pos, 1);
        for (Triple t : data) num[t.col]++;
        for (int i = 2; i <= cols; i++)
            pos[i] = pos[i - 1] + num[i - 1];
        Triple[] arr = new Triple[data.size()];
        for (Triple t : data)
            arr[pos[t.col]++ - 1] = new Triple(t.col, t.row, t.value);
        Collections.addAll(res.data, arr);
        return res;
    }
}
```
