## 核心定义

哈夫曼树（Huffman Tree）又称最优二叉树，是一类带权路径长度（WPL）最短的二叉树。给定 n 个权值 {w1, w2, ..., wn} 作为叶子结点的权值，构造一棵二叉树使得 WPL = sum(w_i * l_i) 最小，其中 l_i 是从根到第 i 个叶子结点的路径长度。哈夫曼算法（贪心算法）的步骤：(1) 将 n 个权值看作 n 棵只有根结点的二叉树，构成森林 F；(2) 从 F 中选出两棵根结点权值最小的树作为左右子树，合并为一棵新树（新根权值为两子树权值之和），加入 F；(3) 重复 (2) 直到 F 中只剩一棵树。哈夫曼编码利用哈夫曼树为字符分配不等长编码——出现频率高的字符用短编码，频率低的用长编码——实现最优前缀编码（无歧义压缩），总码长最短。

## 关键结论

- 哈夫曼树没有度为 1 的结点（每个内结点都有两个孩子）
- n 个叶子结点的哈夫曼树共有 2n-1 个结点
- WPL = 所有非叶子结点的权值之和
- 哈夫曼编码是前缀编码（任何编码都不是另一个编码的前缀，保证无歧义解码）
- 同一组权值可能构造出不同的哈夫曼树（当多个相同最小值时选择不唯一），但 WPL 相同

## 易错点

1. 哈夫曼树不一定是完全二叉树：哈夫曼树的形状仅取决于权值分布，不一定满足完全二叉树的定义
2. 选择两个最小权值时需稳定处理相等情况：若有多于两棵树有相同最小权值，不同的选择会产生不同形状的树，但 WPL 相同。手动构造时通常让绝对值较小的作为左子树
3. 哈夫曼编码的解码过程：从根开始，0 往左走 1 往右走（或反过来，需固定），走到叶子即解码出一个字符，然后回到根继续

## 例题

**例1：** 给定权值 {2, 3, 5, 7, 11}，构造哈夫曼树并求 WPL。

**解答：** 选 2+3=5（新根 5）；森林 {5(新),5,7,11}；选 5+5=10；森林 {7,10,11}；选 7+10=17；森林 {11,17}；选 11+17=28。WPL = 2*3 + 3*3 + 5*2 + 7*2 + 11*2 = 6+9+10+14+22 = 61（深度计算需画出树确定各叶子层数）。注意：2 和 3 在深度 3；5 和 7 在深度 2；11 在深度 2。其实可以通过非叶结点权值之和来算 WPL：5+10+17+28 = 60? 实际上 WPL = 11*1 + 7*2 + 5*2 + 2*3 + 3*3 = 11+14+10+6+9 = 50。再核对一下构造...

## 代码示例

```cpp
#include <iostream>
#include <queue>
#include <vector>
using namespace std;

struct HuffNode {
    int weight;
    HuffNode *left, *right;
    HuffNode(int w) : weight(w), left(nullptr), right(nullptr) {}
};

// 优先队列比较器：最小堆
struct Compare {
    bool operator()(HuffNode* a, HuffNode* b) {
        return a->weight > b->weight;
    }
};

HuffNode* buildHuffmanTree(const vector<int>& weights) {
    priority_queue<HuffNode*, vector<HuffNode*>, Compare> pq;
    for (int w : weights)
        pq.push(new HuffNode(w));
    while (pq.size() > 1) {
        HuffNode* left = pq.top(); pq.pop();
        HuffNode* right = pq.top(); pq.pop();
        HuffNode* parent = new HuffNode(left->weight + right->weight);
        parent->left = left;
        parent->right = right;
        pq.push(parent);
    }
    return pq.top();
}

// 通过哈夫曼树生成编码
void generateCodes(HuffNode* root, string code, vector<pair<int,string>>& codes) {
    if (!root) return;
    if (!root->left && !root->right) {
        codes.push_back({root->weight, code});
        return;
    }
    generateCodes(root->left, code + "0", codes);
    generateCodes(root->right, code + "1", codes);
}

// 计算 WPL
int calcWPL(HuffNode* root, int depth) {
    if (!root) return 0;
    if (!root->left && !root->right)
        return root->weight * depth;
    return calcWPL(root->left, depth + 1) + calcWPL(root->right, depth + 1);
}
```

```java
public class HuffmanTree {
    static class Node {
        int weight;
        Node left, right;
        Node(int w) { weight = w; }
    }
    
    public static Node buildTree(int[] weights) {
        PriorityQueue<Node> pq = new PriorityQueue<>((a, b) -> a.weight - b.weight);
        for (int w : weights) pq.offer(new Node(w));
        while (pq.size() > 1) {
            Node left = pq.poll(), right = pq.poll();
            Node parent = new Node(left.weight + right.weight);
            parent.left = left; parent.right = right;
            pq.offer(parent);
        }
        return pq.poll();
    }
    
    public static int calcWPL(Node root, int depth) {
        if (root == null) return 0;
        if (root.left == null && root.right == null) return root.weight * depth;
        return calcWPL(root.left, depth + 1) + calcWPL(root.right, depth + 1);
    }
}
```
