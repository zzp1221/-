## 核心定义

顺序表（Sequential List）是线性表的顺序存储结构。它用一组地址连续的存储单元依次存储线性表中的数据元素，从而使得逻辑上相邻的两个元素在物理位置上也相邻。顺序表的最常见实现方式是数组。第 i（1 <= i <= n）个元素的存储地址满足：LOC(ai) = LOC(a1) + (i-1) * sizeof(ElemType)。顺序表支持随机存取，即可以在 O(1) 时间内访问任意位置的元素。顺序表的三个核心要素是：存储空间的起始位置（基地址）、顺序表的最大存储容量（MaxSize）、顺序表的当前长度（length）。由于存储空间是预分配的，顺序表在插入和删除操作时需要大量移动元素，平均时间复杂度为 O(n)。

## 关键结论

- 顺序表支持 O(1) 的随机存取（按位置访问），这是其最大优势
- 插入操作平均需要移动 n/2 个元素，时间复杂度 O(n)；删除操作同理
- 顺序表的存储密度为 1（不需要额外指针，所有空间都用来存数据）
- 需要预先分配固定大小的存储空间，存在空间浪费或扩容开销的问题
- 查找元素按值搜索时需遍历比较，平均时间复杂度 O(n)，除非元素有序则可用二分查找优化到 O(log n)

## 易错点

1. 下标与位序混淆：顺序表中逻辑位序从 1 开始，但数组下标从 0 开始，编程时需要注意转换关系（第 i 个元素对应下标 i-1）
2. 插入时忘记检查空间：插入前必须先判断 length >= MaxSize 是否成立，否则会发生数组越界错误；同样删除前需判断表是否为空
3. 数据移动方向错误：插入时要从最后一个元素开始向后移动（从后往前），删除时要从被删元素的后一个开始向前移动（从前往后），顺序搞反会覆盖数据

## 例题

**例1：** 在长度为 n 的顺序表第 i 个位置插入元素 x 的合法 i 取值范围是？

**解答：** i 的取值范围是 1 <= i <= n+1。因为可以在表头（i=1）、表尾（i=n+1）或中间任意位置插入。i < 1 或 i > n+1 均不合法。

**例2：** 顺序表中删除第 i 个元素，需要移动多少个元素？

**解答：** 需要移动 n - i 个元素（将第 i+1 到第 n 个元素依次前移一位）。

## 代码示例

```cpp
#include <iostream>
using namespace std;
#define MAXSIZE 100

class SeqList {
private:
    int data[MAXSIZE];
    int length;
public:
    SeqList() { length = 0; }
    
    // 按位插入：在第 i 个位置插入元素 e
    bool insert(int i, int e) {
        if (i < 1 || i > length + 1 || length >= MAXSIZE)
            return false;
        for (int j = length; j >= i; j--)
            data[j] = data[j - 1];  // 元素后移
        data[i - 1] = e;
        length++;
        return true;
    }
    
    // 按位删除：删除第 i 个位置的元素
    bool remove(int i) {
        if (i < 1 || i > length)
            return false;
        for (int j = i; j < length; j++)
            data[j - 1] = data[j];  // 元素前移
        length--;
        return true;
    }
    
    // 按值查找：返回第一个等于 e 的元素位序
    int locate(int e) {
        for (int i = 0; i < length; i++)
            if (data[i] == e)
                return i + 1;
        return 0;
    }
};
```

```java
public class SeqList {
    private int[] data;
    private int length;
    private static final int MAXSIZE = 100;
    
    public SeqList() {
        data = new int[MAXSIZE];
        length = 0;
    }
    
    public boolean insert(int i, int e) {
        if (i < 1 || i > length + 1 || length >= MAXSIZE)
            return false;
        for (int j = length; j >= i; j--)
            data[j] = data[j - 1];
        data[i - 1] = e;
        length++;
        return true;
    }
    
    public boolean remove(int i) {
        if (i < 1 || i > length) return false;
        for (int j = i; j < length; j++)
            data[j - 1] = data[j];
        length--;
        return true;
    }
}
```
