## 核心定义

哈希表（Hash Table）又称散列表，是一种根据关键字直接访问数据的存储结构。开放定址法（Open Addressing）是哈希冲突解决的一种主流方法。其核心思想是：当发生冲突（两个不同的 key 映射到同一哈希地址）时，按照某种探测序列在哈希表中寻找下一个空闲位置存储。探测序列的一般形式为：h_i = (h(key) + d_i) mod m，其中 d_i 为探测序列。根据 d_i 的不同，分为三种探测方法：(1) 线性探测 d_i = i ——最简单但容易产生"堆积"（clustering）；(2) 二次探测 d_i = +/- i^2 ——减少堆积但可能无法覆盖所有槽；(3) 伪随机探测 d_i = random(i) ——性能介于前两者之间。开放定址法不需要额外的链表结点，平均查找时间取决于负载因子 alpha（= n / m）。

## 关键结论

- 开放定址法所有数据存储在哈希表数组中，无需指针
- 负载因子 alpha = n/m 不能太大（通常 < 0.7），否则效率急剧下降
- 线性探测实现简单但"一次聚集"现象严重
- 二次探测可减轻聚集，但要求表长为素数且负载因子 < 0.5
- 查找必须遵循相同的探测序列，直到找到目标或遇到空槽（查找失败）
- 删除不能简单置空（会截断探测链），需用懒惰删除标记（tombstone）

## 易错点

1. 删除操作的实现：不能把被删元素的位置直接设为 NULL，否则查找链会被中断。需要设置特殊标记（如 DELETED）表示该位置曾被占用但现已删除，查找时遇到 DELETED 继续探测
2. 二次探测的探测序列：d_i = 1^2, -1^2, 2^2, -2^2, ... 而非 1, 4, 9, ...（因为只往正方向探测更易产生聚集）
3. 负载因子与 rehash：当 alpha 超过阈值（如 0.7）需要 rehash——申请更大的表并重新插入所有元素

## 例题

**例1：** 哈希表长为 m=11，哈希函数 h(k)=k%11。依次插入 {47, 7, 29, 11, 16, 92, 22, 8, 3}，用线性探测法处理冲突，求平均查找长度。

**解答：** 47->3, 7->7, 29->7(冲突)->8, 11->0, 16->5, 92->4, 22->0(冲突)->1, 8->8(冲突)->9, 3->3(冲突)->4(冲突)->5(冲突)->6。各元素比较次数：1,1,2,1,1,1,2,2,4。ASL = (1+1+2+1+1+1+2+2+4)/9 = 15/9 ≈ 1.67。

## 代码示例

```cpp
#include <vector>
using namespace std;

enum Status { EMPTY, OCCUPIED, DELETED };

class OpenAddrHash {
    struct Slot { int key; Status status; Slot() : key(0), status(EMPTY) {} };
    vector<Slot> table;
    int m;  // 表长
    int hash(int key) { return key % m; }
    
public:
    OpenAddrHash(int size) : m(size), table(size) {}
    
    int linearProbe(int key, int i) {
        return (hash(key) + i) % m;
    }
    
    int quadraticProbe(int key, int i) {
        return (hash(key) + i * i) % m;
    }
    
    bool insert(int key) {
        for (int i = 0; i < m; i++) {
            int idx = linearProbe(key, i);
            if (table[idx].status != OCCUPIED) {
                table[idx].key = key;
                table[idx].status = OCCUPIED;
                return true;
            }
        }
        return false;  // 表满
    }
    
    int search(int key) {
        for (int i = 0; i < m; i++) {
            int idx = linearProbe(key, i);
            if (table[idx].status == EMPTY) return -1;
            if (table[idx].status == OCCUPIED && table[idx].key == key)
                return idx;
        }
        return -1;
    }
    
    bool remove(int key) {
        int idx = search(key);
        if (idx == -1) return false;
        table[idx].status = DELETED;
        return true;
    }
};
```

```java
public class OpenAddrHash {
    enum Status { EMPTY, OCCUPIED, DELETED }
    static class Slot { int key; Status status; }
    Slot[] table;
    int m;
    
    public OpenAddrHash(int size) {
        m = size; table = new Slot[m];
        for (int i = 0; i < m; i++) { table[i] = new Slot(); table[i].status = Status.EMPTY; }
    }
    
    int hash(int key) { return key % m; }
    
    public boolean insert(int key) {
        for (int i = 0; i < m; i++) {
            int idx = (hash(key) + i) % m;
            if (table[idx].status != Status.OCCUPIED) {
                table[idx].key = key;
                table[idx].status = Status.OCCUPIED;
                return true;
            }
        }
        return false;
    }
}
```
